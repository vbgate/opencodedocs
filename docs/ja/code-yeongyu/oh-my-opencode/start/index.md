# クイックスタート

このセクションでは、oh-my-opencode のインストールから使用開始までを素早く進めることができます。基本設定からメインオーケストレーター、そしてワンクリックですべての機能を有効にする Ultrawork モードまでをカバーします。

## 学習パス

以下の順序で学習することで、oh-my-opencode の基本機能を段階的に習得できます：

### 1. [クイックインストールと設定](./installation/)

oh-my-opencode のインストール、Provider 設定、および基本検証を完了します。
- oh-my-opencode のインストール
- AI Provider の設定（Claude、OpenAI、Gemini など）
- 設定が正しいか検証
- 設定ファイル構造の理解

### 2. [Sisyphus の紹介：メインオーケストレーター](./sisyphus-orchestrator/)

エージェントオーケストレーションの核心概念を理解し、Sisyphus エージェントの基本的な使い方を学びます。
- エージェントオーケストレーションシステムとは
- Sisyphus エージェントの核心機能
- 基本的な使い方とベストプラクティス
- マルチモデル協調のワークフローの理解

### 3. [Ultrawork モード：ワンクリックですべての機能を有効化](./ultrawork-mode/)

ultrawork キーワードを使用して、すべての専門エージェントと並列タスク機能を素早く有効にする方法を学びます。
- Ultrawork モードとは
- ultrawork キーワードのトリガー方法
- 10 人の専門エージェントの協調作業
- 並列バックグラウンドタスクの管理

## 前提条件

このセクションを始める前に、以下を確認してください：
- Node.js 18+ がインストールされていること（バージョン 20 以上を推奨）
- 基本的なコマンドライン操作に慣れていること
- 少なくとも 1 つの AI サービスの API キーを持っていること（Claude、OpenAI、または Gemini）

::: tip キーを持っていませんか？
まだ AI サービスの API キーを持っていない場合は、Claude から申請を始めることができます。現在、最も推奨される選択肢です。
:::
