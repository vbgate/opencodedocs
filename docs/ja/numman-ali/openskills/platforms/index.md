# プラットフォーム機能

このセクションでは、OpenSkills CLIのコアコマンドとスキル管理機能を深く解説し、「使える」から「熟達する」へのステップアップをサポートします。

## 前提条件

::: warning 始める前に確認してください
このセクションを学習する前に、[クイックスタート](../start/)セクションを完了していることを確認してください。特に：
- [OpenSkillsのインストール](../start/installation/)：OpenSkills CLIのインストールが完了している
- [最初のスキルをインストールする](../start/first-skill/)：基本的なスキルインストールプロセスを理解している
:::

## 本章の内容

このセクションには6つのトピックがあり、OpenSkillsの全コア機能を網羅しています：

### [コマンド詳解](./cli-commands/)

全7つのコマンド、パラメータ、フラグを完全に紹介し、コマンドクイックリファレンスを提供します。コマンドの使い方をすばやく確認したいユーザーのための適切なガイドです。

### [インストールソース詳解](./install-sources/)

3つのインストールソースを詳しく説明：GitHubリポジトリ、ローカルパス、プライベートGitリポジトリ。それぞれの用途、コマンドフォーマット、注意事項を解説します。

### [グローバルインストール vs プロジェクトローカルインストール](./global-vs-project/)

`--global`とデフォルト（プロジェクトローカル）インストールの違いを説明し、適切なインストール場所の選び方、スキル検索の優先順位ルールを理解するのに役立ちます。

### [インストール済みスキルのリスト表示](./list-skills/)

`list`コマンドを使用してインストール済みスキルを表示する方法を学び、`(project)`と`(global)`位置タグの意味を理解しましょう。

### [スキルの更新](./update-skills/)

`update`コマンドを使用してインストール済みスキルを更新する方法をガイドします。すべてを更新するか、特定のスキルを更新するかを選択でき、スキルとソースリポジトリを同期させます。

### [スキルの削除](./remove-skills/)

2つの削除方法を紹介：インタラクティブな`manage`コマンドとスクリプト化された`remove`コマンド。スキルライブラリを効率的に管理するのに役立ちます。
