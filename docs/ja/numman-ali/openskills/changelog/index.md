# 変更履歴

この章ではOpenSkillsのバージョン変更履歴を記録し、各バージョンの新機能、改善、バグ修正を理解できるようにします。

## 本章の内容

| ページ | 説明 |
|--- | ---|
| [変更履歴](./changelog/) | 新機能、改善、バグ修正、セキュリティ更新を含む完全なバージョン変更記録 |

## 変更履歴の使い方

変更履歴はバージョン番号の逆順（最新バージョンが先頭）で並んでおり、各バージョンには以下の分類が含まれます：

- **新機能** - 新しく追加された機能とコマンド
- **改善** - 既存機能の最適化と強化
- **バグ修正** - 修正されたバグ
- **セキュリティ** - セキュリティ関連の改善

::: tip アップグレードの推奨
OpenSkillsをアップグレードする前に、変更履歴を確認してバージョン変化を理解することをお勧めします。特に以下の点に注目してください：
- **Breaking Changes**（破壊的変更）
- 新機能の使用方法
- 既知問題の修正状況
:::
