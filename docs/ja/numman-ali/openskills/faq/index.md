# よくある質問

このセクションでは、OpenSkillsの使用中によくある質問、トラブルシューティング方法、設計理念の解説をまとめています。問題の特定と解決に役立ててください。

## このセクションの内容

<div class="grid-cards">

### [よくある質問](./faq/)

インストール、設定、使用方法に関するよくある疑問に回答します。OpenSkillsとClaude Codeの違い、スキルのインストール場所、AGENTS.mdの同期問題、複数プロジェクトでのスキル共有などを解説します。

### [トラブルシューティング](./troubleshooting/)

よくあるエラーの診断と修正方法を提供します。Git cloneの失敗、SKILL.mdが見つからない、スキルが見つからない、権限エラー、更新スキップなどの問題について、詳細な解決手順を説明します。

### [なぜCLIなのか？](./why-cli-not-mcp/)

OpenSkillsがMCPではなくCLIを選択した設計理由を解説します。両者の位置づけの違いを深く比較し、スキルシステムが静的ファイルモードに適している理由を理解します。

</div>

## 学習パス

```
問題が発生した？
    │
    ├─ 使い方がわからない ────────→ よくある質問
    │                               （概念、設定、使用テクニック）
    │
    ├─ エラーが出た ──────────────→ トラブルシューティング
    │                               （エラー診断、修正手順）
    │
    └─ 設計理念を知りたい ────────→ なぜCLIなのか？
                                    （技術選定、アーキテクチャ決定）
```

**おすすめの順序**：

1. **まずよくある質問を見る** → 核心的な概念と使用テクニックを素早く理解
2. **エラーが出たらトラブルシューティングを見る** → エラーメッセージから解決策を検索
3. **深く理解したい場合は設計理念を見る** → CLI vs MCPの技術的考慮を理解
