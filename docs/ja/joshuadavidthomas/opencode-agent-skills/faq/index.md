## 学習パス

以下の順序で学習することをお勧めします：

```
よくある問題のトラブルシューティング → セキュリティについて
```

1. **まずトラブルシューティングを学ぶ**：問題が発生したら、まず「よくある問題のトラブルシューティング」で素早く原因を特定し解決する
2. **次にセキュリティを学ぶ**：プラグインのセキュリティ機構を理解し、安全な使用習慣を身につける

::: tip 問題が発生したら
プラグイン使用中に具体的な問題が発生した場合は、「よくある問題のトラブルシューティング」で該当する症状を直接検索してください。プラグインのセキュリティ設計を体系的に理解したい場合は、「セキュリティについて」をお読みください。
:::
