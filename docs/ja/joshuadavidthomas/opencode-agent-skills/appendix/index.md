# 付録

本セクションでは、OpenCode Agent Skillsの参考資料と開発規範を提供し、高品質なスキルを作成し、APIインターフェースを迅速に参照できるようにします。

## このセクションの内容

| ページ | 説明 |
|--- | ---|
| [スキル開発ベストプラクティス](./best-practices/) | 命名規範、説明の作成、ディレクトリ構成、スクリプトの使用など、高品質なスキルを作成するための技術と規範を習得します |
| [APIツールリファレンス](./api-reference/) | `get_available_skills`、`read_skill_file`、`run_skill_script`、`use_skill` を含む、利用可能なすべてのツールの詳細なパラメータと戻り値を確認できます |

## 学習パスの推奨

::: tip 推奨順序
1. **まずベストプラクティスを学ぶ** → スキル開発の規範と技術を理解し、一般的なエラーを回避します
2. **次にAPIリファレンスを参照** → 実際の開発で問題が発生した場合、具体的なパラメータと戻り値を確認します
:::

これら2つのコンテンツの位置づけは異なります：

- **ベストプラクティス**：スキルの作成を開始する前に一度通読し、正しい開発習慣を確立するのに適しています
- **APIリファレンス**：リファレンスとして必要なときに具体的な詳細を確認するのに適しています

## 前提条件

:::
