# 更新履歴

## 学習パス

### 1. [バージョン履歴](./version-history/)

プラグインの初版から最新版までの完全な変更記録を確認します。

- 各バージョンの新機能と改善
- 重要な破壊的変更
- バグ修正とパフォーマンス最適化
