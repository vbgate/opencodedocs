# バージョン履歴

この文書では、opencode-mystatusのすべてのバージョンと主要な変更を列挙します。

## 最新バージョン

### [v1.2.0 - v1.2.4](./v120-v124/)

- Google Cloudサポート
- 複数アカウント管理
- 4つのモデルクォータ表示（G3 Pro、G3 Image、G3 Flash、Claude）
- 複数のプラットフォーム認証問題を修正
