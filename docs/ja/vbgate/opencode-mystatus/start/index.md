# クイックスタート

この章では、opencode-mystatusプラグインを5分以内にセットアップする方法を案内します。

## 学習パス

以下の順序で学習することをお勧めします：

### 1. [クイックスタート](./quick-start/)

5分以内にプラグインのインストールと設定を完了し、最初のAIプラットフォームのクォータをクエリします。

- 3つのインストール方法：AIにインストールさせる、手動インストール、ローカルファイルからインストール
- スラッシュコマンド `/mystatus` の設定
- インストールの検証

### 2. [mystatusの使用](./using-mystatus/)

スラッシュコマンドと自然言語の2つの使用方法を学びます。

- スラッシュコマンド `/mystatus` の使い方
- 自然言語での質問でmystatusツールをトリガーする方法
- 2つの方法の違いと適用シナリオ

### 3. [出力の解釈](./understanding-output/)

プログレスバー、リセット時間、複数アカウントの出力形式を理解します。

- プログレスバーの意味：塗りつぶしブロックと空ブロックの違い
- 異なるプラットフォームのクォータサイクル（3時間、5時間、月次）
- 複数アカウントシナリオでの出力形式

## 前提条件

学習を始める前に、以下を確認してください：

- [OpenCode](https://opencode.ai) がインストールされていること
- 少なくとも1つのAIプラットフォームの認証情報が設定されていること（OpenAI、Zhipu AI、Z.ai、GitHub Copilot、またはGoogle Cloud）

## 次のステップ

この章を完了したら、[プラットフォーム機能](../platforms/)に進んで、各プラットフォームのクォータクエリの詳細を学べます。
