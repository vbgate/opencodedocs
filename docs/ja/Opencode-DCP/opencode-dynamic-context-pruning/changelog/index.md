# 更新履歴

本セクションでは、DCP プラグインのバージョン進化の歴史を記録しています。各バージョンの新機能、改善点、修正点を理解し、アップグレードの意思決定に役立ててください。

## 本章の内容

| ページ | 説明 |
|--- | ---|
| | [バージョン履歴](./version-history/) | v1.0.1 から最新バージョンまでの完全な更新記録を確認できます。新機能、改善、バグ修正を含みます |

## 読み方

更新履歴は主に以下のシーンで活用できます：

1. **アップグレード前に確認** - 新バージョンでどのような変更があったかを理解し、アップグレードの必要性を評価
2. **トラブルシューティング** - 特定の機能がどのバージョンで導入されたか、特定のバグがどのバージョンで修正されたかを確認
3. **開発方向を理解** - バージョン履歴を通じて、DCP の機能進化と最適化の重点を把握

::: tip 最新版を維持
OpenCode 設定で `@latest` タグを使用することを推奨します。これにより、最新バージョンの機能と修正を自動的に取得できます。
:::
