# よくある質問

本章では、DCP 使用時のよくある質問への回答とベストプラクティスの提案をまとめています。問題を迅速に特定し、使用体験を最適化するのに役立ちます。

## 本章の内容

<div class="grid-cards">

### [トラブルシューティング](./troubleshooting/)

DCP 使用時のよくある問題を解決します。設定エラーの修正、デバッグ方法の有効化、Token が減少しない原因の分析など、トラブルシューティングのテクニックを紹介します。

### [ベストプラクティス](./best-practices/)

DCP の最適な設定戦略を学びます。Prompt Caching と Token 節約のトレードオフ、保護メカニズムの設定、自動戦略の最適化などの高度なテクニックを習得します。

</div>

## 学習パス

```
トラブルシューティング → ベストプラクティス
```

1. **まずトラブルシューティングガイドを見る**：問題が発生したときに迅速に解決策を特定
2. **次にベストプラクティスを学ぶ**：設定を最適化し、Token 節約効果を最大化

## 前提条件

::: warning 事前に完了することを推奨

- [設定ガイド](../../start/configuration/)：設定ファイルの基本構造を理解
:::
