# 主要機能

この章では、DCPの3つの核心能力について詳しく説明します：自動トリミング戦略、LLM駆動ツール、Slashコマンド。これらの機能を習得すれば、DCPのToken最適化可能性を最大限に発揮できます。

## 本章の内容

<div class="grid-cards">

### [自動トリミング戦略](./auto-pruning/)

DCPの3つの自動戦略の仕組みを深く理解します：重複除去戦略、上書き戦略、エラー消去戦略。各戦略のトリガー条件と適用シナリオを学びます。

### [LLM駆動トリミングツール](./llm-tools/)

AIがどのように `discard` と `extract` ツールを自律的に呼び出してコンテキストを最適化するかを理解します。これはDCPの最もインテリジェントな機能で、AIが能動的にコンテキスト管理に参加します。

### [Slashコマンドの使用](./commands/)

すべてのDCPコマンドの使い方を習得します：`/dcp context` でコンテキスト状態を確認、`/dcp stats` で統計データを確認、`/dcp sweep` で手動トリミングを実行。

</div>

## 学習パス

本章の内容は以下の順序で学習することをお勧めします：

```
自動トリミング戦略 → LLM駆動ツール → Slashコマンド
     ↓              ↓            ↓
   仕組みを理解    インテリジェントトリミングを習得  監視とデバッグを学習
```

1. **まず自動トリミング戦略から**：これはDCPの基礎です。3つの戦略の仕組みを理解します
2. **次にLLM駆動ツール**：自動戦略を理解した後、より高度なAI能動トリミング能力を学習します
3. **最後にSlashコマンド**：監視と手動制御の方法を習得し、デバッグと最適化に便利です

::: tip 前提条件
本章を学習する前に、以下を完了していることを確認してください：
- [インストールとクイックスタート](../start/getting-started/) - DCPプラグインのインストールを完了
- [設定完全解説](../start/configuration/) - 設定システムの基本概念を理解
:::
