# 付録

本章では、DCPの技術参考資料を提供します。内部アーキテクチャ設計、Token計算原理、完全なAPIドキュメントが含まれています。これらの内容は、DCPの仕組みを深く理解したいユーザーや、二次開発を行いたいユーザー向けです。

## 本章の内容

| ドキュメント | 説明 | 対象者 |
| --- | --- | --- |
| [アーキテクチャ概要](./architecture/) | DCPの内部アーキテクチャ、モジュールの依存関係、呼び出しチェーンを理解する | DCPの仕組みを理解したいユーザー |
| [Token計算原理](./token-calculation/) | DCPがToken使用量と節約統計をどのように計算するかを理解する | 節約効果を正確に評価したいユーザー |
| [APIリファレンス](./api-reference/) | 完全なAPIドキュメント。設定インターフェース、ツール仕様、状態管理が含まれます | プラグイン開発者 |

## 学習パス

```
アーキテクチャ概要 → Token計算原理 → APIリファレンス
      ↓              ↓              ↓
   設計を理解      統計を理解      拡張を開発
```

**推奨順序**：

1. **アーキテクチャ概要**：まず全体像を把握し、DCPのモジュール分割と呼び出しチェーンを理解します
2. **Token計算原理**：`/dcp context`が出力する計算ロジックを理解し、Tokenの分布を分析できるようになります
3. **APIリファレンス**：プラグインの開発や二次開発を行う場合は、完全なインターフェースドキュメントを参照します

::: tip 必要に応じて読む
DCPを単に使いたいだけの場合は、本章をスキップできます。これらの内容は主に、原理を深く理解したい方や開発を行いたいの方向けです。
:::
