この章では、OpenCode Markdown Table Formatterの入門内容を扱います。プラグインを素早くインストールし、そのコア機能を理解するのに役立ちます。

この章では、プラグインのインストールと設定を完了させ、自動フォーマットの8つのコア機能を包括的に理解します。数分でテーブルフォーマットの魔法のような効果を目にすることができます。
