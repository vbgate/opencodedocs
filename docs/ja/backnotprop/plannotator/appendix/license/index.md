# Plannotator ライセンス説明

## 学習後にできること

- Plannotator が使用するライセンスの種類とコア条項を理解する
- どの使用シナリオが許可され、どれが制限されるかを明確にする
- ライセンスの将来の変更タイムラインを把握する
- 商用ライセンスの取得方法やライセンサーへの連絡方法を知る

## 今知っておくべきこと

Plannotator は **Business Source License 1.1 (BSL)** をオープンソースライセンスとして採用しています。これは、商業的価値を保護しながら、最終的にプロジェクトを完全にオープンソース化することを目的とした特殊なライセンスです。

## コア条項

### ライセンスの種類

| 項目 | 内容 |
| --- | --- |
| **ライセンス名** | Business Source License 1.1 (BSL) |
| **著作権者** | backnotprop (2025) |
| **対象プロジェクト** | plannotator |
| **変更日** | 2030-01-01 |
| **変更後のライセンス** | Apache License, Version 2.0 |

::: info BSL とは？
Business Source License は MariaDB が開発した特殊なライセンスです。短期的には特定の商用利用を制限しますが、特定の日付以降にソフトウェアを完全なオープンソースライセンス（Apache 2.0 など）に変換することを約束しています。このモデルにより、オープンソースプロジェクトは商業化の過程で持続可能な収益を得ながら、長期的にはコミュニティに開放されることが保証されます。
:::

### 許可される使用方法

BSL 1.1 の条項に基づき、以下のことが可能です：

- ✅ **複製、修正、派生作品の作成**
  - Plannotator をベースに二次開発が可能
  - ニーズに合わせてソースコードを修正可能

- ✅ **再配布**
  - オリジナルまたは修正版の配布が可能
  - 元のライセンス表示を保持する必要あり

- ✅ **非本番環境での使用**
  - 社内開発、テスト、評価
  - 個人学習と研究

::: tip 個人および企業内部での幅広い使用
BSL の「非本番環境での使用」の定義は広範で、個人および企業内部のさまざまなシナリオを含みます。あなたとチームメンバー（独立請負業者を含む）は、開発、テスト、デプロイのために Plannotator を自由に使用できます。
:::

### 使用制限（重要！）

**コア制限**：Plannotator を **計画、コンテキスト、または承認サービス（Planning, Context, or Approval Service）** に使用することはできません。

#### 「計画、コンテキスト、または承認サービス」とは？

**第三者**（従業員および独立請負業者を除く）が直接的または間接的な操作を通じて Plannotator の機能にアクセスできる商用製品は、制限対象のサービスに該当します。具体的には以下を含みますが、これらに限定されません：

- **クラウドサービスプロバイダー**
  - SaaS プラットフォームがサービスの一部として Plannotator 機能を提供
  - ホスティングサービスプロバイダーが顧客やサブスクライバーに Plannotator を提供

- **インフラストラクチャサービス**
  - データセンターサービスプロバイダー
  - クラウドコンピューティングプラットフォームプロバイダー

- **類似サービスの第三者**
  - 上記の事業体およびその関連会社が提供するあらゆるサービスを含む

::: warning 商用利用に関する警告
Plannotator を商用サービスの一部として第三者に提供する予定がある場合は、事前に商用ライセンスを取得する必要があります。この条項に違反すると、現在およびすべてのバージョンの Plannotator の使用権が自動的に終了します。
:::

## ライセンス変更タイムライン

| 時期 | ライセンスの種類 | 説明 |
| --- | --- | --- |
| **現在 - 2030-01-01** | BSL 1.1 | 制限付きオープンソースライセンス |
| **2030-01-01 以降** | Apache 2.0 | 完全オープンソース、商用制限なし |

### 変更後に何が起こるか？

2030年1月1日から、Plannotator は自動的に **Apache License 2.0** に変換されます。これは以下を意味します：

- ✅ すべての商用制限が解除される
- ✅ あらゆる商用目的に自由に使用可能
- ✅ 任意の SaaS サービスに統合可能
- ✅ 寛容なオープンソースライセンス基準に準拠

## 商用ライセンス

BSL の制限に従わずに Plannotator を使用する必要がある場合（例：計画、コンテキスト、または承認サービス用）、以下の方法があります：

1. **商用ライセンスの購入**
   - backnotprop またはその認定販売代理店から購入
   - BSL 制限のない使用権を取得

2. **ライセンサーへの連絡**
   - メール送信先：backnotprop
   - 使用シナリオと要件を説明

## よくある質問

### 社内で Plannotator を使用できますか？

**はい。** BSL は、あなたと従業員、独立請負業者が社内開発、テスト、デプロイのために Plannotator を使用することを明示的に許可しています。制限は、**第三者**（従業員および請負業者以外）にサービスを提供するシナリオにのみ適用されます。

### Plannotator をベースに独自の製品を開発できますか？

**はい、ただし条件があります。** 製品が以下の場合：
- 社内ツール：✅ 許可
- 個人プロジェクト：✅ 許可
- オープンソースプロジェクト：✅ 許可（BSL ライセンスを保持）
- 商用製品：❓ 「計画、コンテキスト、または承認サービス」に該当するかどうかの評価が必要

### 当社は SaaS プロバイダーですが、使用できますか？

**商用ライセンスが必要です。** SaaS 製品が Plannotator の機能を顧客に提供する場合（機能の一部としてであっても）、これは制限対象の商用サービスに該当し、商用ライセンスを購入する必要があります。

### 2030年以降、Plannotator をあらゆる目的に使用できますか？

**はい。** 変更日以降、Plannotator は Apache 2.0 ライセンスを採用し、すべての商用制限が解除され、あらゆる目的に自由に使用できます。
