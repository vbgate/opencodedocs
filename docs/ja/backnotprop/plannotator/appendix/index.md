# 付録

このセクションでは、Plannotatorの技術リファレンスを提供します。APIドキュメント、データモデル定義、ライセンス情報が含まれています。これらの内容は、システムの内部メカニズムを深く理解したい方や、二次開発を行う方を対象としています。

## このセクションの内容

| ページ | 説明 |
| --- | --- |
| [APIリファレンス](./api-reference/) | Plannotatorが提供するすべてのAPIエンドポイントとリクエスト/レスポンス形式を確認 |
| [データモデル](./data-models/) | Plannotatorで使用されるデータモデル（Annotation、Block、CodeAnnotationなど）を理解 |
| [ライセンス](./license/) | Plannotatorのライセンス（BSL-1.1）と使用制限を確認 |

## 推奨学習パス

付録の内容は比較的独立しているため、必要に応じて選択的に読むことができます：

1. **APIインターフェースを知りたい** → [APIリファレンス](./api-reference/)を直接参照
2. **データ構造を理解したい** → [データモデル](./data-models/)を参照
3. **使用制限を知りたい** → [ライセンス](./license/)を参照

::: tip ヒント
付録の内容は主に参照用であり、順番に学ぶ必要はありません。具体的な問題に遭遇した際に、関連するセクションを参照することをお勧めします。
:::
