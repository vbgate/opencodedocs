# クイックスタート

この章では、Plannotatorをゼロから使い始める方法を案内します。Plannotatorとは何か、どのような問題を解決できるかを理解し、お使いのAI Coding Agent（Claude CodeまたはOpenCode）でインストールと設定を完了します。

## この章の内容

<div class="grid-cards">

### [クイックスタート](./getting-started/)

Plannotatorのコア機能と使用シナリオを理解し、5分以内に初回体験を完了します。すべての新規ユーザーの入門第一課として最適です。

### [Claude Codeプラグインのインストール](./installation-claude-code/)

Claude CodeでPlannotatorプラグインをインストールして設定します。プラグインシステムと手動Hookの2つの方法をサポートし、ローカルとリモート環境の設定をカバーします。

### [OpenCodeプラグインのインストール](./installation-opencode/)

OpenCodeでPlannotatorプラグインをインストールして設定し、`submit_plan`ツールと`/plannotator-review`コマンドを有効にします。

</div>

## 学習パス

::: tip 推奨順序
1. **まず[クイックスタート](./getting-started/)を読む**：Plannotatorとは何か、何ができるかを理解し、全体像を把握します
2. **次にインストールチュートリアルを選択**：お使いのAI Coding Agentに応じたインストールガイドを選びます
   - Claude Codeを使用 → [Claude Codeプラグインのインストール](./installation-claude-code/)
   - OpenCodeを使用 → [OpenCodeプラグインのインストール](./installation-opencode/)
:::

インストールが完了したら、Plannotatorを使用してプランレビューとコードレビューを開始できます。

## 次のステップ

この章を完了したら、**[プラットフォーム機能](../platforms/)**に進んで、プランレビューとコードレビューの完全な使い方を習得します：

- プランレビューの基本：承認、却下、コメントの追加
- 画像アノテーション：プランにスクリーンショットを添付してアノテーションを追加
- コードレビュー：Git diffの可視化、行レベルのコメント追加
