# よくある問題

この章では Antigravity Tools 使用中に最もよく遭遇するエラーコードと異常シナリオを収集し、問題の根本原因を素早く特定し、解決策を見つけるのを助けます。

## 本章に含まれるもの

| 問題タイプ | ページ | 説明 |
|--- | --- | ---|
| アカウント無効 | [invalid_grant とアカウント自動無効化](./invalid-grant/) | アカウントが突然使えなくなった？OAuth Token 失効の原因と復旧フローを理解 |
| 認証失敗 | [401/認証失敗](./auth-401/) | リクエストが拒否された？auth_mode 設定と Header 形式を確認 |
| レート制限エラー | [429/容量エラー](./429-rotation/) | 頻繁に 429？クォータ不足とアップストリームレート制限を区別し、ローテーション戦略で影響を軽減 |
| パスエラー | [404/パス互換性](./404-base-url/) | インターフェース 404？Base URL と /v1 プレフィックスの連結問題を解決 |
| ストリーム異常 | [ストリーム中断/0 Token/署名無効](./streaming-0token/) | レスポンスが中断または空？プロキシの自己修復メカニズムとトラブルシューティングパスを理解 |

## 学習パスの提案

**エラーコードでトラブルシューティング**：特定のエラーに遭遇したとき、直接対応するページにジャンプ

**体系的に学習**：遭遇する可能性のある問題を全面的に理解したい場合、以下の順序で読むことを推奨：

1. **[404/パス互換性](./404-base-url/)** — 最も一般的な接続問題、通常最初の罠
2. **[401/認証失敗](./auth-401/)** — パスは合っているがリクエストが拒否された、認証設定を確認
3. **[invalid_grant とアカウント自動無効化](./invalid-grant/)** — アカウントレベルの問題
4. **[429/容量エラー](./429-rotation/)** — リクエスト成功だがレート制限された
5. **[ストリーム中断/0 Token/署名無効](./streaming-0token/)** — 上級問題、ストリームレスポンスと署名メカニズムに関連

## 前提条件

- [アカウントを追加](../start/add-account/) — アカウント追加の正しい方法を理解
