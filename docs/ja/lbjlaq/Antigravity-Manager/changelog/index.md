# 更新ログ

この章では Antigravity Tools のバージョン進化とリリースノートを記録します。アップグレード前に Changelog を確認すると、互換性の変更や回帰問題に遭遇する可能性を減らすことができます。

## 本章の内容

| ページ | 説明 |
| --- | --- |
| [バージョン進化](./release-notes/) | README 内蔵の Changelog を読む方法を学習し、アップグレードの影響を素早く判断し、アップグレード後の検証を完了する |

## 学習パス

1. **アップグレード前**：まず Settings → About で現在のバージョンを確認し、README の Changelog であなたがスキップしたバージョンの段落のみを確認する
2. **アップグレード後**：`/healthz` の最小検証を一度実行し、プロキシが正常に動作していることを確認する

## 次のステップ

- まだインストールしていない場合、またはインストール方法を知りたい場合は、まず [インストールとアップグレード](../start/installation/) を確認してください
- プロキシの基本的な使用方法を知りたい場合は、[ローカルリバースプロキシの起動と最初のクライアント接続](../start/proxy-and-first-client/) を確認してください
