# はじめに

本章では、opencode-notify プラグインを素早くインストールしてデスクトップ通知機能を体験し、その核心的な動作仕組みを理解します。

## 学習パス

### 1. [クイックスタート](./quick-start/)
5分以内にインストールを完了し、最初の通知を体験します。
- OCX によるワンクリックインストール
- 通知機能が正常に動作しているかの確認

### 2. [動作原理](./how-it-works/)
プラグインの核心機能、スマートフィルタリングメカニズム、通知トリガー条件について学びます。
- 4種類の通知タイプとそのトリガータイミングをマスターする
- スマートフィルタリングが通知の殺到を回避する仕組みを理解する

## 前提条件

- OpenCode Agent がインストール済み（[公式サイトからダウンロード](https://opencode.com)）
- 基本的なターミナル操作の知識
