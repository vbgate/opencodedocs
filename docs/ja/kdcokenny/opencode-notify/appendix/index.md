# 付録：イベントタイプと設定リファレンス

本セクションでは、リファレンスドキュメントと設定例を提供し、opencode-notify のイベントタイプと設定オプションを深く理解するのに役立ちます。これらの内容は必要に応じて参照するためのものであり、順番に学習する必要はありません。

## 学習パス

### 1. [イベントタイプの説明](./event-types/)

プラグインが監視する OpenCode イベントタイプとそのトリガー条件を理解します。

- 4 種類のイベントタイプ（session.idle、session.error、permission.updated、tool.execute.before）
- 各イベントのトリガータイミングと処理ロジック
- 親セッションチェック、静音時間チェック、ターミナルフォーカスチェックのフィルタリングルール
- プラットフォーム別の機能差異

### 2. [設定ファイルの例](./config-file-example/)

完全な設定ファイルの例とすべてのフィールドの詳細なコメントを確認します。

- 完全な設定ファイルテンプレート
- notifyChildSessions、sounds、quietHours、terminal などのフィールドの説明
- macOS で利用可能なサウンド効果の完全リスト
- 最小限の設定例
- プラグインを無効化する方法
