# OpenCodeDocs について

> **AI エコシステム初のチュートリアルサイト・ソースコード駆動・100% 実行可能**

## 私たちのビジョン

AI ツールの進化は速く、公式ドキュメントは遅れ、オンラインのチュートリアルは古くなっている——私たちはこれを変えたいと考えています。

OpenCodeDocs はオープンソースプロジェクトのソースコードと公式例を深く解析し、核心ロジックを抽出します。また、自動検証により、すべてのコードが正常に動作することを保証します。

プロジェクトが更新されると、私たちのチュートリアルも同期して更新されます。各チュートリアルには Git Commit バージョンが明記されており、どのバージョンに基づいてコードが生成されたかを確認できます。

**私たちの目標は、AI ツール界の「Verified Standard（検証済み標準）」になることです。**

## オープンソースライセンス

私たちはオープンさが信頼を生むと信じています。コミュニティからの貢献、修正、補完を歓迎します。
