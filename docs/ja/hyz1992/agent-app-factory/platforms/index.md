# プラットフォームと統合

この章では、AI開発環境の設定と必須プラグインのインストール方法を説明し、パイプラインが正常に実行できるようにします。

## 学習パス

### 1. [Claude Code 統合ガイド](./claude-code/)
Claude Code を使用してパイプラインを実行する方法を学びます。権限設定とベストプラクティスを含みます。
- Claude Code の権限設定
- ファイルアクセス制限の理解
- パイプライン実行手順の習得

### 2. [OpenCode とその他のAIアシスタント](./other-ai-assistants/)
OpenCode、Cursor などのAIアシスタントを使用してパイプラインを実行する方法を学びます。
- OpenCode CLI の設定
- Cursor プロジェクトの設定
- 互換性に関する説明

### 3. [必須プラグインのインストール](./plugins/)
superpowers と ui-ux-pro-max プラグインをインストールし、パイプラインが正常に動作するようにします。
- superpowers プラグインのインストール
- ui-ux-pro-max プラグインのインストール
- プラグインの可用性確認
