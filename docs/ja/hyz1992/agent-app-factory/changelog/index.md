# 更新ログ

## セクション概要

本セクションでは Agent App Factory のバージョンアップデートと機能変更を記録し、プロジェクトの発展履歴と重要な改善を把握できるようにします。

## サブページナビゲーション

- **[更新ログ](./release-notes/)** - 完全なバージョン履歴、機能変更、バグ修正記録を確認

## 学習パスの推奨

本セクションはいつでも参照できます。以下のタイミングでの確認を推奨します：
- 初回使用時に現在のバージョンの重要な変更を確認する
- 問題発生時に関連するバグ修正があるか確認する
- 定期的に新バージョンの機能アップデートに注目する
