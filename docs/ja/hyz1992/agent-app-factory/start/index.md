# クイックスタート

このセクションでは AI App Factory を素早く使い始め、コアバリュー、ツールのインストール設定、そして 7 ステージパイプラインの仕組みについて理解し、最初のアプリ生成に備えます。

## 学習パス

以下の順序で学習することを推奨します：

### 1. [クイックスタート: アイデアからアプリへ](../getting-started/)

AI App Factory のコアバリューと前提条件を理解し、完全なケースを通じてアイデアからアプリのエンドツーエンドフローを体験します。

- チェックポイントベースのマルチ Agent 協働メカニズムの理解
- 7 ステージパイプラインの完全なワークフローの把握
- 明確な製品説明の記述テクニックの習得

### 2. [インストールと設定](../installation/)

CLI ツールをグローバルインストールし、AI アシスタント（Claude Code/OpenCode）と関連環境を設定します。

- npm を使用して agent-app-factory をグローバルインストール
- Claude Code または OpenCode の設定
- インストール成功の確認

### 3. [Factory プロジェクトの初期化](../init-project/)

`factory init` コマンドでディレクトリを初期化し、生成されるプロジェクト構造と設定ファイルを理解します。

- プロジェクト初期化コマンドの実行
- プロジェクトディレクトリ構造と設定ファイルの理解
- 製品アイデア入力の準備

### 4. [7 ステージパイプラインの概要](../pipeline-overview/)

Bootstrap から Preview までの完全なパイプラインフローを深く理解し、各ステージの入出力とチェックポイントメカニズムを習得します。

- 7 ステージの目標と成果物の理解
- チェックポイントメカニズムと人工確認フローの習得
- パイプライン実行の制御方法の習得

## 前提条件

このセクションの学習を始める前に、以下が必要です：

::: warning 前提要件

1. **Node.js 環境**: Node.js v18 以上がインストールされていること
2. **AI プログラミングアシスタント**: 以下のツールのいずれかが設定されていること
   - [Claude Code](https://claude.ai/code)（推奨）⭐
   - [OpenCode](https://opencode.design/) または他の Agent モード対応 AI アシスタント
3. **基本的なコマンドライン操作**: ターミナルでコマンドを実行できること

:::

## 次のステップ

このセクションの学習が完了すると、以下を習得しています：

- ✅ AI App Factory のコアの仕組み
- ✅ CLI ツールのインストールと設定
- ✅ プロジェクトの初期化と構造の理解
- ✅ 7 ステージパイプラインの完全なフロー

次は、[プラットフォームと統合](../../platforms/) セクションを学習し、異なる AI アシスタントでパイプラインを実行する方法について理解することを推奨します。
