# 付録

この章では参考ドキュメントと補足資料を提供し、AI App Factory の技術詳細を深く理解するのに役立ちます。

## サブページナビゲーション

### [CLI コマンドリファレンス](../cli-commands/)
完全な CLI コマンドリスト、パラメータ説明と使用例。`init`、`run`、`continue`、`status`、`list`、`reset` などのコマンドが含まれます。

### [ディレクトリ構造詳細](../directory-structure/)
Factory プロジェクトの完全なディレクトリ構造と各ファイルの用途を理解します。`agents/`、`skills/`、`cli/` などのコアディレクトリの役割を含みます。

### [コード規定](../code-standards/)
生成されたコードが従うべき TypeScript コーディング規定、ファイル構造、コメント要件、コミットメッセージ規定を理解します。

### [技術スタック詳細](../tech-stack/)
生成されたアプリが使用する技術スタック：Node.js + Express + Prisma + React Native、および各部分の技術選定理由を理解します。
