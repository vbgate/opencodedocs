# バージョン履歴

本プロジェクトでは、すべてのバージョンにおける機能更新、改善、および修正を記録しています。

---

## v1.0.0 - 2026年1月

### 🎉 初回リリース

これは Agent Skills の最初の正式版であり、完全なスキルパッケージとビルドツールチェーンが含まれています。

#### 新機能

**React パフォーマンス最適化ルール**
- 40以上の React/Next.js パフォーマンス最適化ルール
- 8つの主要カテゴリ：ウォーターフォール排除、バンドル最適化、サーバーサイドパフォーマンス、再レンダリング最適化など
- 影響レベル別分類（CRITICAL > HIGH > MEDIUM > LOW）
- 各ルールに誤り/正解のコード例対比を含む

**Vercel ワンクリックデプロイ**
- 40以上の主流フレームワーク自動検出に対応
- ゼロ認証デプロイフロー
- プレビューリンクと所有権移転リンクの自動生成
- 静的 HTML プロジェクトに対応

**Web デザインガイドライン**
- 100以上の Web インターフェースデザインルール
- アクセシビリティ、パフォーマンス、UX など多角的レビュー
- リモートルール取得（GitHub から）に対応

**ビルドツールチェーン**
- `pnpm build` - AGENTS.md 完全ルールドキュメントを生成
- `pnpm validate` - ルールファイルの完全性を検証
- `pnpm extract-tests` - テストケースを抽出
- `pnpm dev` - 開発フロー（build + validate）

#### 技術スタック

- TypeScript 5.3.0
- Node.js 20+
- pnpm 10.24.0+
- tsx 4.7.0（TypeScript 実行時）

#### ドキュメント

- AGENTS.md 完全ルールガイド
- SKILL.md スキル定義ファイル
- README.md インストールおよび使用説明
- ビルドツール完全ドキュメント

## バージョン命名規則

本プロジェクトは、セマンティックバージョニング（Semantic Versioning）に従います：

- **メジャーバージョン（Major）**: 互換性のない API 変更
- **マイナーバージョン（Minor）**: 後方互換性のある新機能
- **パッチバージョン（Patch）**: 後方互換性のあるバグ修正

例：`1.0.0` は最初の安定バージョンを表します。
