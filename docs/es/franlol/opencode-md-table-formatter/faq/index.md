# Preguntas Frecuentes

## Resumen del Capítulo

Este capítulo cubre las preguntas frecuentes y los métodos de solución de problemas del plugin opencode-md-table-formatter, ayudándote a identificar y resolver rápidamente varios problemas que encuentres durante el proceso de formato de tablas.

---

## Navegación de Subpáginas

### [Preguntas Frecuentes: ¿Qué hacer si las tablas no se formatean](troubleshooting/)

Presentación completa de los problemas comunes y soluciones en el uso del plugin:

- **Tablas no formateadas**: Verificar configuración, requisitos de versión, reiniciar OpenCode
- **Estructura de tabla inválida**: Comprender el significado del error "invalid structure", aprender la sintaxis correcta de tablas
- **Fallo en el formateo**: Manejar el error "table formatting failed" y situaciones anormales
- **Tipos de tablas no compatibles**: Conocer las limitaciones de las tablas HTML, celdas multilínea, etc.
- **Desalineación después del formateo**: Investigar problemas como modo oculto, celdas demasiado largas, código en línea, etc.

---

## Recomendaciones de Ruta de Aprendizaje

Se recomienda consultar este capítulo cuando encuentres problemas reales después de completar [Visión General de Funciones](../start/features/), o usarlo como manual de referencia para la solución de problemas.

Puedes seguir el siguiente orden de aprendizaje:

1. **Cuando encuentres problemas**: Primero identifica el tipo de problema según los síntomas (tablas no formateadas, mensajes de error, etc.)
2. **Investigar paso a paso**: Verifica uno por uno según la lista de verificación de la guía de solución de problemas
3. **Comprensión profunda**: Lee [Especificaciones de Tablas](../advanced/table-spec/) y [Principios del Modo Oculto](../advanced/concealment-mode/) para conocer los detalles técnicos
4. **Limitaciones conocidas**: Consulta [Limitaciones Conocidas](../appendix/limitations/) para conocer los límites del plugin
