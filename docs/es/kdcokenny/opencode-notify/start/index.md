# Introducción

Este capítulo te ayuda a instalar el complemento opencode-notify rápidamente, experimentar las funciones de notificación de escritorio y comprender su mecanismo de funcionamiento principal.

## Ruta de aprendizaje

### 1. [Inicio rápido](./quick-start/)
Completa la instalación y experimenta tu primera notificación en 5 minutos.
- Instalación con un clic mediante OCX
- Verifica que las notificaciones funcionen correctamente

### 2. [Cómo funciona](./how-it-works/)
Comprende las funciones principales del complemento, el mecanismo de filtrado inteligente y las condiciones de activación de notificaciones.
- Domina los cuatro tipos de notificaciones y sus momentos de activación
- Entiende cómo el filtrado inteligente evita el bombardeo de notificaciones

## Requisitos previos

- OpenCode Agent instalado ([descargar desde el sitio web oficial](https://opencode.com))
- Conocimientos básicos de operación de terminal
