# Registro de cambios

Este capítulo registra el historial de versiones y los cambios importantes del plugin opencode-notify, ayudándote a seguir las actualizaciones de funcionalidad, correcciones de problemas y mejoras de configuración.

## Subpáginas

### [Notas de versión](./release-notes/)

Consulta el historial de cambios del plugin registrado en orden cronológico inverso.

**Contenido incluido**:
- Registro de actualizaciones de funcionalidad
- Explicaciones de correcciones de problemas
- Detalles de mejoras de configuración
- Avisos de cambios disruptivos

::: tip Notas sobre la versión
Este plugin se publica a través de OCX y adopta una estrategia sin números de versión, rastreando los cambios a través del historial de commits de Git.
:::

## Requisitos previos

No se requieren requisitos previos, cualquiera puede consultar el registro de cambios.
