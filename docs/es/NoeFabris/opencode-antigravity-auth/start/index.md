# Inicio Rápido

Este capítulo te ayuda a comenzar desde cero con el plugin Antigravity Auth. Conocerás el valor central del plugin, completarás la instalación y la autenticación OAuth, y enviarás tu primera solicitud de modelo para verificar que la configuración es exitosa.

## Ruta de Aprendizaje

Sigue este orden de aprendizaje, cada paso se basa en el anterior:

### 1. [Introducción al Plugin](./what-is-antigravity-auth/)

Conoce el valor central del plugin Antigravity Auth, los escenarios de uso y las advertencias de riesgo.

- Determina si el plugin se adapta a tu caso de uso
- Conoce los modelos de IA compatibles (Claude Opus 4.5, Sonnet 4.5, Gemini 3 Pro/Flash)
- Comprende los riesgos y precauciones de uso

### 2. [Instalación Rápida](./quick-install/)

Completa la instalación del plugin y la primera autenticación en 5 minutos.

- Dos métodos de instalación (con asistente de IA / configuración manual)
- Configuración de definiciones de modelo
- Ejecución de la autenticación OAuth de Google

### 3. [Autenticación OAuth 2.0 PKCE](./first-auth-login/)

Comprende el flujo de autenticación OAuth 2.0 PKCE y completa el primer inicio de sesión.

- Entiende el mecanismo de seguridad de la autenticación PKCE
- Completa el primer inicio de sesión para obtener permisos de acceso a la API
- Conoce el manejo automatizado de la renovación de tokens

### 4. [Primera Solicitud](./first-request/)

Envía la primera solicitud de modelo para verificar que la instalación es exitosa.

- Envía la primera solicitud de modelo Antigravity
- Comprende los parámetros `--model` y `--variant`
- Soluciona errores comunes en solicitudes de modelo

## Requisitos Previos

Antes de comenzar este capítulo, confirma:

- ✅ OpenCode CLI instalado (comando `opencode` disponible)
- ✅ Cuenta de Google disponible (para autenticación OAuth)
