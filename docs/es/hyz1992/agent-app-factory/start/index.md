# Inicio

Este capítulo te ayuda a iniciar rápidamente con AI App Factory, comprender el valor central, instalar y configurar las herramientas, y comprender el principio de funcionamiento de la tubería de 7 etapas, preparándote para generar tu primera aplicación.

## Ruta de Aprendizaje

Se recomienda seguir el siguiente orden para estudiar el contenido de este capítulo:

### 1. [Inicio Rápido: desde la idea hasta la aplicación](../getting-started/)
Comprende el valor central y los requisitos previos de AI App Factory, y experimenta el proceso de extremo a extremo desde la idea hasta la aplicación a través de un caso completo.
- Comprender el mecanismo de colaboración multi-agente basado en puntos de control
- Conocer el flujo de trabajo completo de la tubería de 7 etapas
- Dominar las técnicas para proporcionar descripciones claras del producto

### 2. [Instalación y Configuración](../installation/)
Instala la herramienta CLI globalmente, configura los asistentes de IA (Claude Code/OpenCode) y el entorno relacionado.
- Instalar agent-app-factory globalmente usando npm
- Configurar Claude Code u OpenCode
- Verificar que la instalación sea exitosa

### 3. [Inicializar Proyecto Factory](../init-project/)
Usa el comando `factory init` para inicializar el directorio, y comprender la estructura del proyecto generado y los archivos de configuración.
- Ejecutar el comando de inicialización del proyecto
- Comprender la estructura del directorio del proyecto y los archivos de configuración
- Preparar la entrada de la idea del producto

### 4. [Descripción General de la Tubería de 7 Etapas](../pipeline-overview/)
Comprende profundamente el proceso completo de la tubería desde Bootstrap hasta Preview, las entradas y salidas de cada etapa, y el mecanismo de puntos de control.
- Comprender los objetivos y productos de las 7 etapas
- Dominar el mecanismo de puntos de control y el proceso de confirmación manual
- Aprender a controlar la ejecución de la tubería

## Requisitos Previos

Antes de comenzar a estudiar este capítulo, necesitas:

::: warning Requisitos Previos

1. **Entorno de Node.js**: Node.js v18 o versión superior ya instalado
2. **Asistente de Programación de IA**: Debes configurar una de las siguientes herramientas
   - [Claude Code](https://claude.ai/code) (recomendado) ⭐
   - [OpenCode](https://opencode.design/) u otros asistentes de IA que admitan el modo Agent
3. **Operaciones Básicas de Línea de Comandos**: Poder ejecutar comandos usando la terminal

:::

## Siguiente Paso

Después de completar el estudio de este capítulo, dominarás:

- ✅ El principio de funcionamiento central de AI App Factory
- ✅ La instalación y configuración de la herramienta CLI
- ✅ La inicialización del proyecto y la comprensión de la estructura
- ✅ El proceso completo de la tubería de 7 etapas

A continuación, se recomienda estudiar el capítulo **[Plataformas e Integración](../../platforms/)** para comprender cómo ejecutar la tubería en diferentes asistentes de IA.
