# Apéndice

Esta sección contiene documentación de referencia y material complementario para ayudarte a comprender en profundidad los detalles técnicos de AI App Factory.

## Navegación de subpáginas

### [Referencia de comandos CLI](../cli-commands/)
Lista completa de comandos CLI, descripción de parámetros y ejemplos de uso, incluyendo los comandos `init`, `run`, `continue`, `status`, `list` y `reset`.

### [Estructura de directorios](../directory-structure/)
Conoce la estructura completa de directorios del proyecto Factory y el propósito de cada archivo, incluyendo los roles de los directorios principales como `agents/`, `skills/` y `cli/`.

### [Estándares de código](../code-standards/)
Conoce los estándares de codificación TypeScript que debe seguir el código generado, la estructura de archivos, los requisitos de comentarios y las convenciones de mensajes de commit.

### [Stack tecnológico](../tech-stack/)
Conoce el stack tecnológico utilizado por las aplicaciones generadas: Node.js + Express + Prisma + React Native, así como las razones detrás de cada elección tecnológica.

## Sugerencias de ruta de aprendizaje

La documentación de esta sección es **material de referencia** y no necesita leerse en orden. Se recomienda consultar los capítulos correspondientes según sea necesario mientras estudias el tutorial principal:

- Al estudiar [Instalación y configuración](../../start/installation/), puedes consultar la [Referencia de comandos CLI](../cli-commands/)
- Al estudiar [Inicialización del proyecto Factory](../../start/init-project/), puedes consultar la [Estructura de directorios](../directory-structure/)
- Al estudiar las diversas etapas del pipeline, puedes consultar el [Stack tecnológico](../tech-stack/)
- Cuando necesites comprender los estándares de calidad del código generado, puedes consultar los [Estándares de código](../code-standards/)
