# Preguntas Frecuentes y Mejores Prácticas

La sección de FAQ recopila soluciones a problemas comunes y recomendaciones de mejores prácticas al usar AI App Factory, ayudándote a solucionar problemas rápidamente y mejorar la eficiencia de desarrollo.

## Navegación de Subpáginas

| Curso | Descripción |
| ---- | ---- |

| **[Mejores Prácticas](../best-practices/)** | Aprender técnicas para proporcionar descripciones claras del producto, aprovechar puntos de control, controlar el alcance, iterar paso a paso, etc. |

## Recomendaciones de Ruta de Aprendizaje

El contenido de la sección de FAQ puede consultarse según sea necesario, sin necesidad de seguir un orden estricto:

1. **Consulta cuando encuentres problemas**: Si encuentras algún problema durante el uso, consulta primero "Preguntas Frecuentes y Solución de Problemas"
2. **Lectura esencial para mejorar la eficiencia**: El capítulo "Mejores Prácticas" resume lecciones aprendidas. Se recomienda leerlo después de completar el primer proyecto; la eficiencia de desarrollo mejorará significativamente en proyectos posteriores

## Requisitos Previos

::: tip Recomendación Previo

Antes de leer la sección de FAQ, se recomienda completar lo siguiente:

- ✅ **Capítulo start**: Completar la inicialización del proyecto y ejecutar el pipeline completo al menos una vez
- ✅ **Capítulo platforms**: Configurar al menos un asistente de IA (Claude Code u OpenCode)

:::
