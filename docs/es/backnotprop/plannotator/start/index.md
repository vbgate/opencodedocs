# Inicio rápido

Esta sección te ayuda a empezar con Plannotator desde cero. Aprenderás qué es Plannotator, qué problemas puede resolver y cómo instalar y configurar el plugin en tu Agente de codificación con IA (Claude Code u OpenCode).

## Contenido de esta sección

<div class="grid-cards">

### [Inicio rápido](./getting-started/)

Descubre las funciones principales y casos de uso de Plannotator, y completa tu primera experiencia en 5 minutos. Ideal para todos los nuevos usuarios como primera lección de introducción.

### [Instalar plugin para Claude Code](./installation-claude-code/)

Instala y configura el plugin de Plannotator en Claude Code, compatible con el sistema de plugins y el método de Hook manual, cubriendo configuraciones para entornos locales y remotos.

### [Instalar plugin para OpenCode](./installation-opencode/)

Instala y configura el plugin de Plannotator en OpenCode, habilitando la herramienta `submit_plan` y el comando `/plannotator-review`.

</div>

## Ruta de aprendizaje

::: tip Orden recomendado
1. **Lee primero [Inicio rápido](./getting-started/)**: entiende qué es Plannotator, qué puede hacer y establece una visión general
2. **Luego elige un tutorial de instalación**: selecciona la guía de instalación correspondiente según el Agente de codificación con IA que uses
   - Usas Claude Code → [Instalar plugin para Claude Code](./installation-claude-code/)
   - Usas OpenCode → [Instalar plugin para OpenCode](./installation-opencode/)
:::

Una vez completada la instalación, podrás empezar a usar Plannotator para la revisión de planes y revisión de código.

## Siguientes pasos

Después de completar esta sección, continúa aprendiendo sobre **[Funciones de la plataforma](../platforms/)** y domina el uso completo de la revisión de planes y revisión de código:

- Fundamentos de revisión de planes: aprobar, rechazar, agregar comentarios
- Anotación de imágenes: adjuntar capturas de pantalla y anotarlas en los planes
- Revisión de código: visualizar Git diff y agregar comentarios a nivel de línea
