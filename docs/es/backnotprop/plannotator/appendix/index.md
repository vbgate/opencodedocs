# Apéndice

Este capítulo proporciona referencias técnicas de Plannotator, incluyendo documentación de la API, definiciones de modelos de datos e información de licencia. Este contenido está principalmente dirigido a usuarios que necesitan comprender profundamente los mecanismos internos del sistema o realizar desarrollo secundario.

## Contenido de este Capítulo

| Página | Descripción |
|--- | ---|
| [Referencia de la API](./api-reference/) | Ver todos los endpoints de la API y los formatos de solicitud/respuesta proporcionados por Plannotator |
| [Modelos de Datos](./data-models/) | Conocer los modelos de datos utilizados por Plannotator, incluyendo Annotation, Block, CodeAnnotation, etc. |
| [Licencia](./license/) | Entender la licencia de Plannotator (BSL-1.1) y las restricciones de uso |

## Recomendaciones de Ruta de Aprendizaje

El contenido del apéndice es relativamente independiente, puedes leer selectivamente según tus necesidades:

1. **Para conocer las interfaces de la API** → Lee directamente [Referencia de la API](./api-reference/)
2. **Para comprender las estructuras de datos** → Lee [Modelos de Datos](./data-models/)
3. **Para conocer las restricciones de uso** → Lee [Licencia](./license/)

::: tip Consejo
El contenido del apéndice se utiliza principalmente para consulta, no necesita aprenderse en orden. Se recomienda consultar los capítulos relevantes cuando encuentres problemas específicos.
:::
