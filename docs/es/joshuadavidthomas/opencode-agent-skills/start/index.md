# Inicio Rápido

Este capítulo te ayudará a comenzar desde cero con el plugin OpenCode Agent Skills. Conocerás el valor principal del plugin, completarás la instalación y configuración, y crearás tu primera habilidad.

## Contenido del Capítulo

<div class="grid-cards">

<a href="./what-is-opencode-agent-skills/" class="card">
  <h3>¿Qué es OpenCode Agent Skills?</h3>
  <p>Conoce el valor principal y las características del plugin, incluyendo el descubrimiento dinámico de habilidades, la inyección de contexto, la compresión y recuperación, entre otros mecanismos.</p>
</a>

<a href="./installation/" class="card">
  <h3>Guía de Instalación</h3>
  <p>Completa la instalación del plugin y verifica su funcionamiento. Soporta tres métodos: instalación básica, instalación con versión fija e instalación para desarrollo local.</p>
</a>

<a href="./creating-your-first-skill/" class="card">
  <h3>Crea tu Primera Habilidad</h3>
  <p>Domina la estructura de directorios de habilidades y el formato de SKILL.md, y practica creando y probando una habilidad simple.</p>
</a>

</div>

## Ruta de Aprendizaje

Se recomienda seguir este orden de aprendizaje:

1. **[¿Qué es OpenCode Agent Skills?](./what-is-opencode-agent-skills/)** — Primero comprende qué puede hacer el plugin para tener una visión general
2. **[Guía de Instalación](./installation/)** — Instala el plugin para que OpenCode tenga capacidades de gestión de habilidades
3. **[Crea tu Primera Habilidad](./creating-your-first-skill/)** — Practica creando tu propia primera habilidad

::: tip Tiempo Estimado
Completar este capítulo toma aproximadamente 30-45 minutos.
:::

## Siguiente Paso

Después de completar este capítulo, puedes continuar con **[Funciones de la Plataforma](../platforms/)** para profundizar en el descubrimiento de habilidades, la carga de habilidades, el emparejamiento automático y otras funciones avanzadas.
