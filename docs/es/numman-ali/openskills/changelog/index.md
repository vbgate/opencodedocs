# Registro de Cambios

Esta sección documenta el historial de versiones de OpenSkills, ayudándote a comprender las nuevas funciones, mejoras y correcciones de errores en cada versión.

## Contenido de este Capítulo

| Página | Descripción |
| --- | --- |
| [Registro de Cambios](./changelog/) | Registro completo de cambios de versión, incluyendo nuevas funciones, mejoras, correcciones de errores y actualizaciones de seguridad |

## Cómo Usar el Registro de Cambios

El registro de cambios está ordenado por número de versión en orden descendente (la versión más reciente primero). Cada versión incluye las siguientes categorías:

- **Nuevas Funciones** - Funcionalidades y comandos recién añadidos
- **Mejoras** - Optimizaciones y mejoras de funcionalidades existentes
- **Correcciones de Errores** - Bugs que han sido solucionados
- **Seguridad** - Mejoras relacionadas con la seguridad

::: tip Recomendación de Actualización
Antes de actualizar OpenSkills, se recomienda revisar el registro de cambios para comprender las modificaciones de la versión, prestando especial atención a:
- **Breaking Changes** (Cambios Rupturistas)
- Cómo utilizar las nuevas funciones
- El estado de corrección de problemas conocidos
:::
