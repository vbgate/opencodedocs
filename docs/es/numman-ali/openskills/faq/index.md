# Preguntas Frecuentes

Este capítulo recopila preguntas comunes, soluciones de problemas y explicaciones del diseño durante el uso de OpenSkills, ayudándote a localizar y resolver problemas rápidamente.

## Contenido del Capítulo

<div class="grid-cards">

### [Preguntas Frecuentes](./faq/)

Responde dudas comunes sobre instalación, configuración y uso. Incluye las diferencias entre OpenSkills y Claude Code, ubicación de instalación de habilidades, problemas de sincronización de AGENTS.md, habilidades compartidas entre múltiples proyectos, etc.

### [Solución de Problemas](./troubleshooting/)

Proporciona métodos de diagnóstico y reparación para errores comunes. Cubre pasos detallados de solución para problemas como fallo de Git clone, no encontrar SKILL.md, habilidades no encontradas, errores de permisos, actualizaciones omitidas, etc.

### [¿Por qué CLI?](./why-cli-not-mcp/)

Explica las razones de diseño por las que OpenSkills eligió CLI en lugar de MCP. Compara profundamente las diferencias de posicionamiento entre ambos, entendiendo por qué el sistema de habilidades es más adecuado para el modo de archivos estáticos.

</div>

## Ruta de Aprendizaje

```
¿Tienes problemas?
    │
    ├─ No sé cómo usar ──────────→ Preguntas Frecuentes
    │                           (Conceptos, configuración, consejos de uso)
    │
    ├─ Hay un error ────────────────→ Solución de Problemas
    │                           (Diagnóstico de errores, pasos de reparación)
    │
    └─ Quiero entender el diseño ────────→ ¿Por qué CLI?
                                (Selección técnica, decisiones de arquitectura)
```

**Orden recomendado**:

1. **Primero lee las preguntas frecuentes** → Comprender rápidamente los conceptos principales y consejos de uso
2. **Si hay errores, mira la solución de problemas** → Buscar soluciones según el mensaje de error
3. **Para entender en profundidad, lee el diseño** → Entender las consideraciones técnicas de CLI vs MCP
