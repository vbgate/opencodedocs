# Apéndice

Esta sección proporciona la referencia técnica de OpenSkills, incluyendo la API de línea de comandos, especificaciones de formato de archivos y definiciones de términos. Cuando necesites consultar parámetros específicos de comandos, entender la estructura de los archivos generados o aclarar el significado de algún término, este es tu manual de referencia rápida.

## Contenido de esta sección

| Documento | Descripción |
| --- | --- |
| [Referencia de API CLI](./cli-api/) | Documentación completa de todos los comandos, opciones y parámetros, con ejemplos de uso |
| [Especificación del formato AGENTS.md](./agents-md-format/) | Estructura del archivo AGENTS.md generado por el comando sync, incluyendo etiquetas XML y formato de lista de habilidades |
| [Estructura de archivos](./file-structure/) | Descripción de los directorios y archivos generados por OpenSkills, para entender qué contiene `.openskills/` |
| [Glosario](./glossary/) | Explicación de términos clave en OpenSkills y sistemas de habilidades de IA |

## Ruta de aprendizaje recomendada

El apéndice es material de referencia, no necesitas leerlo de principio a fin. Forma recomendada de uso:

1. **Problemas con comandos** → Consulta [Referencia de API CLI](./cli-api/)
2. **Quieres saber cómo se genera AGENTS.md** → Ve [Especificación del formato AGENTS.md](./agents-md-format/)
3. **No sabes dónde van los archivos** → Revisa [Estructura de archivos](./file-structure/)
4. **Encuentras un término desconocido** → Consulta el [Glosario](./glossary/)
