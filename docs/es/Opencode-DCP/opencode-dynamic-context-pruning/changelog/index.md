# Registro de Cambios

Esta sección documenta la historia de evolución de versiones del plugin DCP, ayudándote a comprender las nuevas funciones, mejoras y correcciones de cada versión para tomar decisiones informadas sobre actualizaciones.

## Contenido de Esta Sección

| Página | Descripción |
| --- | --- |
| [Historial de Versiones](./version-history/) | Consulta el registro completo de actualizaciones desde la v1.0.1 hasta la última versión, incluyendo nuevas funciones, mejoras y correcciones de errores |

## Recomendaciones de Lectura

El registro de cambios es principalmente útil en los siguientes escenarios:

1. **Antes de Actualizar** - Comprender qué cambios trae la nueva versión y evaluar si es necesario actualizar
2. **Resolución de Problemas** - Confirmar en qué versión se introdujo una función específica o en qué versión se corrigió un error
3. **Comprender la Dirección de Desarrollo** - Entender la evolución de funciones y las prioridades de optimización de DCP a través del historial de versiones

::: tip Mantente Actualizado
Se recomienda usar la etiqueta `@latest` en la configuración de OpenCode para obtener automáticamente las últimas funciones y correcciones.
:::
