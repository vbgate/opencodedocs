# Preguntas Frecuentes

Este capítulo recopila las preguntas frecuentes y las mejores prácticas durante el uso de DCP, ayudándote a identificar rápidamente los problemas y optimizar la experiencia de uso.

## Contenido de este Capítulo

<div class="grid-cards">

### [Preguntas Frecuentes y Solución de Problemas](./troubleshooting/)

Resuelve los problemas comunes en el uso de DCP, incluyendo la reparación de errores de configuración, la habilitación de métodos de depuración, el análisis de las causas por las que los Tokens no se reducen, y otras técnicas de solución de problemas.

### [Mejores Prácticas](./best-practices/)

Aprende las mejores estrategias de configuración de DCP, domina el equilibrio entre Prompt Caching y ahorro de Tokens, la configuración de mecanismos de protección, la optimización automática de estrategias y otras técnicas avanzadas.

</div>

## Ruta de Aprendizaje

```
Preguntas Frecuentes y Solución de Problemas → Mejores Prácticas
```

1. **Primero consulta la guía de solución de problemas**: localiza rápidamente la solución cuando encuentres problemas
2. **Luego aprende las mejores prácticas**: optimiza la configuración para maximizar el efecto de ahorro de Tokens
