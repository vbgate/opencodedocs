# Inicio Rápido

Esta sección te ayuda a instalar y comenzar a usar oh-my-opencode rápidamente, desde la configuración básica hasta el orquestador principal, y luego al modo Ultrawork que activa todas las funciones con un clic.

## Ruta de Aprendizaje

Sigue esta secuencia para dominar progresivamente las funciones básicas de oh-my-opencode:

### 1. [Instalación y Configuración Rápidas](./installation/)

Completa la instalación de oh-my-opencode, configuración del Provider y verificación básica.
- Instalar oh-my-opencode
- Configurar AI Provider (Claude, OpenAI, Gemini, etc.)
- Verificar que la configuración es correcta
- Entender la estructura del archivo de configuración

### 2. [Conoce a Sisyphus: El Orquestador Principal](./sisyphus-orchestrator/)

Comprende los conceptos básicos de la orquestación de agentes y aprende el uso básico del agente Sisyphus.
- Qué es un sistema de orquestación de agentes
- Funciones principales del agente Sisyphus
- Uso básico y mejores prácticas
- Entender el flujo de trabajo de colaboración multi-modelo

### 3. [Modo Ultrawork: Activa Todas las Funciones con un Clic](./ultrawork-mode/)

Aprende a usar la palabra clave ultrawork para activar rápidamente todos los agentes profesionales y las funciones de tareas paralelas.
- Qué es el modo Ultrawork
- Cómo activar la palabra clave ultrawork
- Trabajo colaborativo de 10 agentes profesionales
- Gestión de tareas en paralelo en segundo plano

## Requisitos Previos

Antes de comenzar esta sección, asegúrate de:
- Tener Node.js 18+ instalado (se recomienda la versión 20 o superior)
- Estar familiarizado con operaciones básicas de línea de comandos
- Tener al menos una clave API de servicio de IA (Claude, OpenAI o Gemini)

::: tip ¿No tienes clave API?
Si aún no tienes una clave API de servicio de IA, puedes comenzar solicitando Claude, que actualmente es la opción más recomendada.
:::
