# Inicio rápido

Este capítulo te ayuda a familiarizarte rápidamente con el plugin opencode-mystatus, desde la instalación hasta el uso en solo 5 minutos.

## Ruta de aprendizaje

Se recomienda seguir el siguiente orden:

### 1. [Inicio rápido](./quick-start/)

Completa la instalación y configuración del plugin en 5 minutos y consulta la cuota de tu primera plataforma de IA.

- Tres métodos de instalación: que la IA te ayude, instalación manual, instalación desde archivo local
- Configura el comando de barra diagonal `/mystatus`
- Verifica que la instalación sea exitosa

### 2. [Usar mystatus](./using-mystatus/)

Aprende los dos métodos de uso: comandos de barra diagonal y lenguaje natural.

- Uso del comando de barra diagonal `/mystatus`
- Preguntas en lenguaje natural para activar la herramienta mystatus
- Diferencias entre ambos métodos y escenarios de uso

### 3. [Interpretar la salida](./understanding-output/)

Comprende el significado de la barra de progreso, el tiempo de restablecimiento y el formato de salida para múltiples cuentas.

- Significado de la barra de progreso: bloques sólidos vs bloques huecos
- Ciclos de límite de diferentes plataformas (3 horas, 5 horas, mensual)
- Formato de salida en escenarios con múltiples cuentas

## Prerrequisitos

Antes de comenzar, confirma que:

- Has instalado [OpenCode](https://opencode.ai)
- Has configurado al menos la información de autenticación de una plataforma de IA (OpenAI, Zhipu AI, Z.ai, GitHub Copilot o Google Cloud)

## Siguientes pasos

Después de completar este capítulo, puedes continuar con [Funciones de plataforma](../platforms/) para conocer más detalles sobre la consulta de cuotas de cada plataforma.
