# Historial de versiones

Este documento lista todas las versiones y los cambios principales de opencode-mystatus.

## Versión más reciente

### [v1.2.0 - v1.2.4](./v120-v124/)

- Soporte de Google Cloud
- Gestión de múltiples cuentas
- Visualización de cuota de 4 modelos (G3 Pro、G3 Image、G3 Flash、Claude）
- Corrección de múltiples problemas de autenticación de plataforma

## Versiones históricas

### [v1.0.0 - v1.0.1](./v100-v101/)

- Versión inicial
- Consulta de cuota de OpenAI
- Consulta de cuota de Zhipu AI / Z.ai
- Consulta de cuota de GitHub Copilot
- Visualización básica de barras de progreso
- Soporte multilingüe (chino/inglés)
