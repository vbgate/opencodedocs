# 用户服务协议

最后更新日期：2026-01-01  
生效日期：2026-01-01

欢迎使用 OpenCodeDocs！在使用我们的服务之前，请仔细阅读本用户服务协议（以下简称"本协议"）。使用我们的服务即表示您同意接受本协议的约束。

## 1. 服务说明

OpenCodeDocs 是一个专注于 AI 工具教程的开放平台，提供以下服务：

- **教程内容**：开源 AI 工具（如 Claude Code、Cursor、LangChain 等）的使用教程
- **代码验证**：基于源码仓库自动验证教程中的代码可运行性
- **社区功能**：用户可通过 GitHub 参与内容贡献和讨论

我们保留随时修改、暂停或终止部分或全部服务的权利，恕不另行通知。

## 2. 账号注册与管理

### 注册条件
- 您必须年满 16 周岁（或在您所在地区达到法定年龄）
- 提供真实、准确的注册信息
- 每位用户仅可注册一个账号

### 账号安全
- 您有责任保管好账号凭证，不得与他人共享
- 如发现账号被盗用，请立即联系我们
- 因账号管理不善导致的损失，由您自行承担

### 账号注销
您可以随时申请注销账号。注销后，您的个人数据将在 30 天内删除，但公开发布的内容可能仍会保留。

## 3. 用户行为规范

在使用本平台时，您同意不从事以下行为：

- 发布违法、有害、威胁、骚扰、诽谤或侵权的内容
- 冒充他人或虚假陈述身份
- 未经授权访问他人账号或系统
- 干扰或破坏服务的正常运行
- 使用自动化工具大规模抓取内容（合理的 API 使用除外）
- 将教程内容用于商业目的而不遵守许可协议
