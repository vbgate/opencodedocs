# 上手

本章节帮助你快速上手 AI App Factory，了解核心价值、安装配置工具，并理解 7 阶段流水线的工作原理，为生成第一个应用做好准备。

## 学习路径

建议按照以下顺序学习本章节内容：

### 1. [快速开始：从想法到应用](../getting-started/)
了解 AI App Factory 的核心价值和前置要求，通过一个完整的案例体验从想法到应用的端到端流程。
- 理解基于检查点的多 Agent 协作机制
- 了解 7 阶段流水线的完整工作流程
- 掌握提供清晰产品描述的技巧

### 2. [安装与配置](../installation/)
全局安装 CLI 工具，配置 AI 助手（Claude Code/OpenCode）及相关环境。
- 使用 npm 全局安装 agent-app-factory
- 配置 Claude Code 或 OpenCode
- 验证安装是否成功

### 3. [初始化 Factory 项目](../init-project/)
使用 `factory init` 命令初始化目录，了解生成的项目结构和配置文件。
- 执行项目初始化命令
- 理解项目目录结构和配置文件
- 准备产品想法输入

### 4. [7 阶段流水线概览](../pipeline-overview/)
深入理解从 Bootstrap 到 Preview 的完整流水线流程，各阶段的输入输出和检查点机制。
- 理解 7 个阶段的目标和产物
- 掌握检查点机制和人工确认流程
- 学习如何控制流水线执行

## 前置条件

开始本章节学习前，你需要：

::: warning 前置要求

1. **Node.js 环境**：已安装 Node.js v18 或更高版本
2. **AI 编程助手**：必须配置以下工具之一
   - [Claude Code](https://claude.ai/code)（推荐）⭐
   - [OpenCode](https://opencode.design/) 或其他支持 Agent 模式的 AI 助手
3. **基础命令行操作**：能够使用终端执行命令

:::
