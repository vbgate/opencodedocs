# 高级功能

本章节深入讲解 Supermemory 的底层机制和高级配置选项。你将学会如何优化记忆引擎的性能，以及如何根据项目需求定制插件行为。

## 本章内容

<div class="grid-cards">

### [抢占式压缩原理](./compaction/)

深入理解 Token 阈值触发的自动压缩机制。学习 Supermemory 如何在上下文即将溢出前，主动生成结构化总结并持久化保存，防止长会话中的"失忆"问题。

### [深度配置详解](./configuration/)

自定义压缩阈值、关键词触发规则和搜索参数。掌握配置文件的所有选项，让 Supermemory 完美适配你的工作流。

</div>

```
┌─────────────────────────────────────────────────────────────┐
│  第 1 步              第 2 步                                │

│  (理解机制)           (动手定制)                             │
└─────────────────────────────────────────────────────────────┘
```

**建议顺序**：

1. **先学压缩原理**：理解 Supermemory 如何自动管理上下文，这是高级配置的基础。
2. **再学配置详解**：在理解机制后，你才能做出合理的配置决策（比如阈值设多少合适）。
