# 更新日志

本章节记录 DCP 插件的版本演进历史，帮助你了解每个版本的新功能、改进和修复，以便做出升级决策。

## 本章内容

| 页面 | 说明 |
|--- | ---|
| [版本历史](./version-history/) | 查看从 v1.0.1 到最新版本的完整更新记录，包括新功能、改进和 Bug 修复 |

## 阅读建议

更新日志主要用于以下场景：

1. **升级前查阅** - 了解新版本带来了哪些变化，评估是否需要升级
2. **问题排查** - 确认某个功能是在哪个版本引入的，或某个 Bug 是在哪个版本修复的
3. **了解发展方向** - 通过版本历史了解 DCP 的功能演进和优化重点

::: tip 保持最新
推荐在 OpenCode 配置中使用 `@latest` 标签，自动获取最新版本的功能和修复。
:::
