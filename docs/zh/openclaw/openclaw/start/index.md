# 快速上手

欢迎开始使用 OpenClaw！本章节将带你从零开始部署个人 AI 助手。

## 章节概述

在这个章节中，你将学习：

- **入门指南** - 从零开始部署 OpenClaw
- **Gateway 快速启动** - 启动你的 AI 助手服务
- **发送第一条消息** - 与 AI 助手初次对话
- **配置基础** - 理解 openclaw.json 配置结构

## 学习路径

```
入门指南 → Gateway 快速启动 → 发送第一条消息 → 配置基础
   │              │                  │              │
   ▼              ▼                  ▼              ▼
安装和初始化   启动服务          首次对话         进阶配置
```

## 前置要求

在开始之前，请确保你满足以下条件：

- **Node.js** >= 22.12.0
- **pnpm** 10.23.0 或更高版本
- **Git**（用于源码安装）
- **AI API Key**（Anthropic、OpenAI 等）
