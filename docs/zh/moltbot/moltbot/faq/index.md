# 常见问题与优化

本章节帮助你解决使用 Clawdbot 时遇到的各种问题，掌握完整的命令行操作方法，并学习如何优化系统性能。

::: info 学习顺序建议
建议按以下顺序阅读本章节内容：
1. 先学习 **[CLI 命令参考](./cli-commands/)** —— 掌握基础操作和调试工具
2. 再学习 **[故障排除](./troubleshooting/)** —— 遇到问题时能快速定位和解决
3. 最后学习 **[性能优化](./performance-optimization/)** —— 提升系统运行效率
:::

## 子页面导航

### [故障排除](./troubleshooting/)

解决常见问题的全面排查指南，涵盖：
- Gateway 启动失败
- 渠道连接问题（WhatsApp、Telegram、Slack 等）
- 认证错误和权限问题
- 工具调用失败
- 会话管理异常

**适用场景**：当你遇到具体问题，需要快速排查和解决时。

---

### [CLI 命令参考](./cli-commands/)

完整的命令列表和用法说明，包括：
- Gateway 控制命令（启动、重启、状态）
- Agent 交互命令
- 模型管理和配置
- 渠道管理（登录、状态、登出）
- 会话管理（列表、重置、删除、压缩）
- 节点管理（列表、调用、配对）
- 技能管理和配置向导

**适用场景**：需要快速查找某个命令的用法和参数，或学习 CLI 的完整功能。

---

### [性能优化](./performance-optimization/)

提升 Gateway 性能和响应速度的优化技巧，涵盖：
- 提示词缓存（Prompt Caching）降低成本
- 会话修剪（Session Pruning）减少上下文
- 并发控制和资源管理
- 内存优化和垃圾回收
- 日志级别调优

**适用场景**：系统运行缓慢、资源占用过高，或希望提升用户体验时。
