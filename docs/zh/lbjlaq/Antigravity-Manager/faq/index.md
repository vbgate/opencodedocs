# 常见问题

本章节收录了使用 Antigravity Tools 过程中最常遇到的错误码和异常场景，帮助你快速定位问题根因并找到解决方案。

## 本章包含

| 问题类型 | 页面 | 说明 |
|---------|------|------|
| 账号失效 | [invalid_grant 与账号自动禁用](./invalid-grant/) | 账号突然不可用？了解 OAuth Token 失效的原因和恢复流程 |
| 鉴权失败 | [401/鉴权失败](./auth-401/) | 请求被拒绝？检查 auth_mode 配置和 Header 格式 |
| 限流错误 | [429/容量错误](./429-rotation/) | 频繁 429？区分配额不足和上游限流，用轮换策略降低影响 |
| 路径错误 | [404/路径不兼容](./404-base-url/) | 接口 404？解决 Base URL 和 /v1 前缀的拼接问题 |
| 流式异常 | [流式中断/0 Token/签名失效](./streaming-0token/) | 响应中断或空？了解代理的自愈机制和排查路径 |

## 学习路径建议

**按错误码排查**：遇到具体错误时，直接跳转到对应页面。

**系统性学习**：如果你想全面了解可能遇到的问题，建议按以下顺序阅读：

1. **[404/路径不兼容](./404-base-url/)** — 最常见的接入问题，通常是第一个坑
2. **[401/鉴权失败](./auth-401/)** — 路径对了但请求被拒，检查鉴权配置
3. **[invalid_grant 与账号自动禁用](./invalid-grant/)** — 账号层面的问题
4. **[429/容量错误](./429-rotation/)** — 请求成功但被限流
5. **[流式中断/0 Token/签名失效](./streaming-0token/)** — 进阶问题，涉及流式响应和签名机制
