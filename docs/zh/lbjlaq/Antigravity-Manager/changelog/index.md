# 更新日志

本章节记录 Antigravity Tools 的版本演进与发布说明。升级前先看 Changelog，能减少踩到兼容性变化或回归问题的概率。

| 页面 | 说明 |
|--- | ---|

## 学习路径

1. **升级前**：先在 Settings → About 确认当前版本，再去 README 的 Changelog 只看你跨过的版本段落
2. **升级后**：跑一遍 `/healthz` 最小验证，确认代理还能正常工作
