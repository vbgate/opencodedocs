# 快速上手 Superpowers

欢迎来到 Superpowers 快速上手指南。本章节帮助你从零开始安装和配置 Superpowers，了解核心概念，并在 5 分钟内体验 AI 编码代理的强大能力。

## 学习路径

建议按以下顺序学习，逐步掌握 Superpowers 的基础：

### 1. [Superpowers 是什么](./superpowers-intro/)

了解 Superpowers 的核心价值、适用场景和工作原理，弄清楚为什么 AI 代理需要这些技能。

### 2. [快速开始：5 分钟上手](./quick-start/)

通过实际操作快速体验 Superpowers 的核心功能，感受它如何改变 AI 编码代理的工作方式。

### 3. [安装指南](./installation/)

详细的安装步骤，覆盖 Claude Code、OpenCode、Codex 三大平台，确保你能在自己的环境中使用 Superpowers。

### 4. [验证安装](./verification/)

学习如何验证 Superpowers 是否正确安装和加载，排查常见的安装问题。

### 5. [如何使用技能](./using-skills/)

理解技能系统的核心概念、技能优先级和调用规则，为后续的学习打下坚实基础。

## 前置条件

在开始学习本章节前，请确保：

- 你熟悉基本的命令行操作
- 你已经安装并配置了至少一个 AI 编码代理（Claude Code、OpenCode 或 Codex）
- 你有基本的编程经验
