# 附录：技能参考、命令与架构

附录章节提供了 Superpowers 的技术参考文档，包括所有技能的快速查阅、命令完整说明、代理定义详情和架构设计原理。

## 章节概述

本章节包含以下技术参考资料：
- **技能完整参考**：所有 14 个技能的触发条件、使用方法和核心要点
- **命令参考**：斜杠命令（`/brainstorm`、`/write-plan`、`/execute-plan`）的详细说明
- **代理参考**：代码审查代理等代理的定义和职责
- **架构与实现原理**：Superpowers 的架构设计、钩子系统和技能发现机制

## 学习路径建议

### 新手路径
如果您刚刚开始使用 Superpowers，建议按以下顺序阅读：
1. 先完成 [快速开始](../../start/quick-start/) 和 [核心工作流](../../core-workflows/) 章节
2. 在使用过程中，遇到需要快速查阅技能或命令时，再回到附录章节
3. 当您想深入理解 Superpowers 如何工作时，再阅读 [架构与实现原理](../architecture/)

### 进阶路径
如果您已经熟悉 Superpowers 的基本使用，可以：
1. 直接阅读 [技能完整参考](../skills-reference/)，全面了解所有技能
2. 参考 [架构与实现原理](../architecture/)，为编写自定义技能做准备
3. 查阅 [命令参考](../commands-reference/)，优化工作流命令的使用

## 前置条件

阅读附录章节之前，建议您已完成：
- [快速开始](../../start/quick-start/)
- 至少一个核心工作流的学习（如 [设计工作流](../../core-workflows/design-workflow/)）
