# 快速上手

本章节帮助你快速安装和开始使用 oh-my-opencode，从基础配置到主编排器，再到一键激活全部功能的 Ultrawork 模式。

## 学习路径

按照以下顺序学习，循序渐进掌握 oh-my-opencode 的基础功能：

### 1. [快速安装与配置](./installation/)

完成 oh-my-opencode 的安装、Provider 配置和基础验证。
- 安装 oh-my-opencode
- 配置 AI Provider（Claude、OpenAI、Gemini 等）
- 验证配置是否正确
- 理解配置文件结构

### 2. [初识 Sisyphus：主编排器](./sisyphus-orchestrator/)

理解代理编排的核心概念，学习 Sisyphus 代理的基本使用方法。
- 什么是代理编排系统
- Sisyphus 代理的核心功能
- 基本使用方法和最佳实践
- 理解多模型协作的工作流程

### 3. [Ultrawork 模式：一键激活全部功能](./ultrawork-mode/)

学习如何使用 ultrawork 关键词快速激活所有专业代理和并行任务功能。
- 什么是 Ultrawork 模式
- 如何触发 ultrawork 关键词
- 10 位专业代理的协同工作
- 并行后台任务的管理

## 前置条件

开始学习本章节前，请确保：
- 已安装 Node.js 18+（推荐 20 或更高版本）
- 熟悉基本的命令行操作
- 拥有至少一个 AI 服务的 API 密钥（Claude、OpenAI 或 Gemini）

::: tip 没有密钥？
如果你还没有 AI 服务的 API 密钥，可以先从 Claude 开始申请，它是目前最推荐的选择。
:::
