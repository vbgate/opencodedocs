# 版本历史

本文档列出了 opencode-mystatus 的所有版本和主要变更。

## 最新版本

### [v1.2.0 - v1.2.4](./v120-v124/)

- Google Cloud 支持
- 多账号管理
- 4 个模型额度显示（G3 Pro、G3 Image、G3 Flash、Claude）
- 修复了多个平台认证问题

## 历史版本

### [v1.0.0 - v1.0.1](./v100-v101/)

- 初始发布
- OpenAI 额度查询
- 智谱 AI / Z.ai 额度查询
- GitHub Copilot 额度查询
- 基础的进度条显示
- 多语言支持（中文/英文）
