# 附录

本章节提供 OpenSkills 的技术参考资料，包括命令行 API、文件格式规范和术语定义。当你需要查阅具体的命令参数、理解生成文件的结构，或者搞清楚某个术语的含义时，这里是你的速查手册。

| 文档 | 说明 |
|--- | ---|
| [CLI API 参考](./cli-api/) | 所有命令、选项、参数的完整文档，附带使用示例 |

## 学习路径建议

附录是参考资料，不需要从头到尾阅读。推荐的使用方式：

1. **遇到命令问题** → 查 [CLI API 参考](./cli-api/)
2. **想了解 AGENTS.md 怎么生成的** → 看 [AGENTS.md 格式规范](./agents-md-format/)
3. **搞不清文件放在哪** → 翻 [文件结构](./file-structure/)
4. **看到陌生术语** → 查 [术语表](./glossary/)

::: tip 新手建议
如果你刚开始使用 OpenSkills，建议先完成 [快速开始](../start/quick-start/) 教程，遇到问题再回来查阅附录。
:::
