# 更新日志

本章节记录 OpenSkills 的版本变更历史，帮助你了解每个版本的新功能、改进和问题修复。

## 本章内容

| 页面 | 说明 |
| ---- | ---- |
| [更新日志](./changelog/) | 完整的版本变更记录，包括新功能、改进、问题修复和安全更新 |

## 如何使用更新日志

更新日志按版本号倒序排列（最新版本在最前面），每个版本包含以下分类：

- **新增功能** - 新加入的功能和命令
- **改进** - 对现有功能的优化和增强
- **问题修复** - 已修复的 Bug
- **安全性** - 安全相关的改进

::: tip 升级建议
升级 OpenSkills 前，建议先查看更新日志了解版本变化，特别关注：
- **Breaking Changes**（破坏性变更）
- 新功能的使用方法
- 已知问题的修复情况
:::
