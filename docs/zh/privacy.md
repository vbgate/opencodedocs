# 隐私条款

最后更新日期：2026-01-01

感谢您使用 OpenCodeDocs。我们非常重视您的隐私和个人信息保护。本隐私条款将帮助您了解我们如何收集、使用、存储和保护您的个人信息。

## 1. 我们收集的信息

在您使用 OpenCodeDocs 服务时，我们可能会收集以下类型的信息：

### 账号信息
- **邮箱地址**：用于账号注册、登录验证和重要通知
- **用户名**：您选择的显示名称
- **GitHub 账号信息**：如您选择使用 GitHub 登录，我们会获取您的公开信息

### 使用数据
- **浏览记录**：您访问的教程页面和时间
- **交互行为**：代码复制、章节跳转等操作
- **搜索记录**：您在站内的搜索关键词

### 设备信息
- **浏览器类型**：Chrome、Firefox、Safari 等
- **操作系统**：Windows、macOS、Linux 等
- **IP 地址**：用于安全防护和地区优化

## 2. 信息使用方式

我们收集的信息将用于以下目的：

- **提供服务**：确保您能够正常访问和使用教程内容
- **个性化体验**：记住您的阅读进度和偏好设置
- **产品改进**：分析使用模式，优化内容和功能
- **安全保障**：检测异常行为，防止滥用
- **通知发送**：发送教程更新提醒和重要公告（可选择退订）

## 3. 信息共享与披露

我们不会出售您的个人信息。仅在以下情况下可能共享：

- **第三方服务提供商**：如 Vercel（托管）、Plausible（分析），他们仅能访问履行职责所需的信息
- **法律要求**：依据法律法规或政府要求进行披露
- **权益保护**：为保护 OpenCodeDocs 及用户权益的必要情况

## 4. 数据存储与安全

- **存储位置**：数据存储在位于美国和亚太地区的安全服务器
- **加密措施**：传输数据使用 TLS/SSL 加密，敏感数据使用 AES-256 加密存储
- **访问控制**：严格限制内部人员对数据的访问权限
- **保留期限**：账号注销后，个人数据将在 30 天内删除

## 5. 您的权利

根据《个人信息保护法》及 GDPR，您享有以下权利：

- **访问权**：您有权查看我们持有的关于您的个人信息
- **更正权**：您有权要求更正不准确的个人信息
- **删除权**：您有权要求删除您的个人信息
- **导出权**：您有权以通用格式导出您的数据
- **撤回同意**：您有权随时撤回之前给予的同意

如需行使上述权利，请通过下方联系方式与我们联系。

## 6. Cookie 政策

我们使用以下类型的 Cookie：

- **必要 Cookie**：保持登录状态、记住语言偏好
- **分析 Cookie**：统计访问量和使用模式（使用隐私友好的 Plausible）
- **广告 Cookie**：
    - 第三方供应商（包括 Google）使用 Cookie 来根据用户在您的网站或其他网站上的过往访问记录投放广告。
    - Google 使用广告 Cookie，使其及其合作伙伴能够根据用户对您网站和/或互联网上其他网站的访问记录向用户投放广告。
    - 用户可以访问 [Google 广告设置](https://www.google.com/settings/ads) 停用个性化广告。或者，您可以引导用户访问 [www.aboutads.info](http://www.aboutads.info/choices/) 停用第三方供应商用于个性化广告的 Cookie。

您可以通过浏览器设置管理 Cookie。禁用必要 Cookie 可能影响部分功能的正常使用。
