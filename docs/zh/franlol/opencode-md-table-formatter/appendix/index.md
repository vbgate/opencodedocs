# 附录：技术细节与限制说明

本章节包含插件的参考文档和技术细节，帮助你深入理解插件的设计思路、边界限制和性能优化策略。

::: info 学完你能做什么
- 了解插件的已知限制和适用场景
- 掌握缓存机制与性能优化策略
- 理解插件的技术边界和设计取舍
:::

## 本章节包含

### 📚 [已知限制：插件的边界在哪里](./limitations/)

了解插件不支持的功能和技术限制，避免在不支持的场景下使用。包括：
- 不支持 HTML 表格、多行单元格、无分隔行表格
- 不支持合并单元格和无配置选项
- 超大表格性能未验证

**适合人群**：想知道插件能做什么、不能做什么的用户

### 🔧 [技术细节：缓存机制与性能优化](./tech-details/)

深入理解插件的内部实现，包括缓存机制、性能优化策略和代码结构。包括：
- widthCache 数据结构与缓存查找流程
- 自动清理机制与缓存阈值
- 性能优化效果分析

**适合人群**：对插件实现原理感兴趣的开发者

## 学习路径建议

本章节的两个子页面相对独立，可以按需阅读：

1. **快速上手用户**：建议先阅读「已知限制」，了解插件边界后即可停止
2. **深度学习用户**：按顺序阅读 → 「已知限制」→「技术细节」
3. **开发者**：推荐完整阅读，有助于理解插件设计和未来扩展
