# 자주 묻는 질문

## 섹션 개요

이 섹션에서는 opencode-md-table-formatter 플러그인의 자주 묻는 질문과 문제 해결 방법을 다루며, 테이블 포맷팅 과정에서 발생하는 다양한 문제를 빠르게 찾아내고 해결하는 데 도움을 줍니다.

---

## 하위 페이지 탐색

### [자주 묻는 질문: 테이블이 포맷팅되지 않는 문제 해결](troubleshooting/)

플러그인 사용 중 발생하는 일반적인 문제와 해결 방법을 포괄적으로 소개합니다:

- **테이블이 포맷팅되지 않음**: 구성, 버전 요구사항, OpenCode 재시작 확인
- **유효하지 않은 테이블 구조**: "invalid structure" 오류 의미 이해, 올바른 테이블 구문 학습
- **포맷팅 실패**: "table formatting failed" 오류 및 예외 상황 처리
- **지원되지 않는 테이블 유형**: HTML 테이블, 다중 행 셀 등 제한된 시나리오 이해
- **포맷팅 후 정렬되지 않음**: 숨김 모드, 셀 초과 길이, 인라인 코드 등 문제 해결

## 학습 경로 제안

[기능 개요](../start/features/)를 완료한 후 실제 문제가 발생했을 때 이 섹션을 조회하거나 문제 해결 참조 매뉴얼로 사용하는 것을 권장합니다.

다음 순서대로 학습할 수 있습니다:

1. **문제가 발생했을 때**: 증상에 따라 문제 유형(테이블이 포맷팅되지 않음, 오류 메시지 등)을 먼저 파악
2. **단계별 문제 해결**: 문제 해결 가이드의 확인 목록에 따라 하나씩 검증
3. **심층 이해**: [테이블 사양](../advanced/table-spec/) 및 [숨김 모드 원리](../advanced/concealment-mode/)를 읽어 기술 세부사항 이해
4. **알려진 제한 사항**: [알려진 제한 사항](../appendix/limitations/)를 확인하여 플러그인 경계 이해
