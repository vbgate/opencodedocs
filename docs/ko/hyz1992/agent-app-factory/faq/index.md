# 자주 묻는 질문 및 모범 사례

| **[모범 사례](../best-practices/)** | 명확한 제품 설명 제공, 체크포인트 활용, 범위 제어, 단계별 반복 등의 기법 학습 |

## 학습 경로 제안

FAQ 섹션의 내용은 필요에 따라 참조할 수 있으며, 엄격한 순서대로 학습할 필요는 없습니다:

1. **문제 발생 시 참조**: 사용 과정에서 문제가 발생하면 우선「자주 묻는 질문 및 문제 해결」을 확인하세요
2. **효율성 향상을 위한 필독」：「모범 사례」섹션은 경험 교훈을 정리한 것으로, 첫 번째 프로젝트를 완료한 후 읽기를 권장합니다. 이후 프로젝트 개발 효율이 크게 향상될 것입니다

## 선행 조건

::: tip 먼저 완료하세요

FAQ 섹션을 읽기 전 다음 내용을 먼저 완료하는 것을 권장합니다:

- ✅ **start** 섹션: 프로젝트 초기화 및 전체 파이프라인 실행 완료
- ✅ **platforms** 섹션: 최소 하나의 AI 도우미(Claude Code 또는 OpenCode) 구성 완료

:::
