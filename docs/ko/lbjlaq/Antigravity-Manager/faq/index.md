# 자주 묻는 질문

이 장에서는 Antigravity Tools 사용 과정에서 가장 자주 만나는 오류 코드와 예외 시나리오를 수집하여 문제의 근본 원인을 빠르게 찾아내고 해결 방안을 찾도록 도와드립니다.

## 이 장 포함 내용

| 문제 유형 | 페이지 | 설명 |
|--- | --- | ---|
| 계정 비활성화 | [invalid_grant 및 계정 자동 비활성화](./invalid-grant/) | 계정이 갑자기 사용할 수 없게 됨? OAuth Token 실패 원인과 복구 절차 이해 |
| 인증 실패 | [401/인증 실패](./auth-401/) | 요청이 거부됨? auth_mode 구성 및 Header 형식 확인 |
| 속도 제한 오류 | [429/용량 오류](./429-rotation/) | 잦은 429? 할당량 부족과 상류 속도 제한 구분, 로테이션 전략으로 영향 줄이기 |
| 경로 오류 | [404/경로 호환성](./404-base-url/) | 인터페이스 404? Base URL 및 /v1 접두사 연결 문제 해결 |
| 스트리밍 예외 | [스트리밍 중단/0 Token/서명 무효](./streaming-0token/) | 응답 중단 또는 비어 있음? 프록시의 자가 치유 메커니즘 및 문제 해결 경로 이해 |

## 학습 경로 추천

**오류 코드별 문제 해결**: 특정 오류가 발생하면 해당 페이지로 직접 이동하세요.

**체계적 학습**: 가능한 모든 문제를 완전히 이해하려면 다음 순서로 읽으시는 것을 추천합니다:

1. **[404/경로 호환성](./404-base-url/)** — 가장 일반적인 연결 문제, 보통 첫 번째 함정
2. **[401/인증 실패](./auth-401/)** — 경로가 올바르지만 요청이 거부됨, 인증 구성 확인
3. **[invalid_grant 및 계정 자동 비활성화](./invalid-grant/)** — 계정 수준 문제
4. **[429/용량 오류](./429-rotation/)** — 요청 성공하지만 속도 제한됨
5. **[스트리밍 중단/0 Token/서명 무효](./streaming-0token/)** — 고급 문제, 스트리밍 응답 및 서명 메커니즘 포함
