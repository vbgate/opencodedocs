# 부록

이 장에서는 Antigravity Tools의 기술 참조 자료를 모았습니다. API 엔드포인트 요약, 데이터 저장 구조 및 실험적 기능 경계 설명이 포함됩니다. 특정 기술 세부 사항을 빠르게 확인해야 할 때, 이곳이 "사전"입니다.

## 이 장의 내용

| 문서 | 설명 | 적합한 시나리오 |
| --- | --- | --- |
| **[엔드포인트 요약표](./endpoints/)** | 외부 HTTP 라우트 개요: OpenAI/Anthropic/Gemini/MCP 엔드포인트, 인증 모드 및 Header 형식 | 새 클라이언트 연결, 404/401 문제 해결 |
| **[데이터와 모델](./storage-models/)** | 계정 파일 구조, SQLite 통계 라이브러리 테이블 구조, 핵심 필드 정의 | 백업/이전, 직접 DB 조회, 문제 해결 |
| **[z.ai 통합 경계](./zai-boundaries/)** | z.ai에서 구현된 기능 목록 vs 명확히 구현되지 않은 기능 목록 | z.ai 기능 평가, 오용 방지 |

## 학습 경로 제안

```
엔드포인트 요약표 → 데이터와 모델 → z.ai 통합 경계
     ↓              ↓              ↓
  호출할 경로 알기  데이터 위치 알기  경계 위치 알기
```

1. **먼저 엔드포인트 요약표 보기**: 호출할 수 있는 API 목록과 인증 구성 방법 이해
2. **그다음 데이터와 모델 보기**: 데이터 저장 구조 이해, 백업/이전 및 직접 DB 조회 편리하게 수행
3. **마지막으로 z.ai 통합 경계 보기**: z.ai를 사용하는 경우, "구현되지 않은" 기능을 "사용 가능한" 것으로 오인하는 것 방지

::: tip 이 문서는 "필독"이 아닙니다
부록은 참조 자료이지 튜토리얼이 아닙니다. 처음부터 끝까지 읽을 필요 없이, 특정 문제가 발생했을 때 참조하세요.
:::
