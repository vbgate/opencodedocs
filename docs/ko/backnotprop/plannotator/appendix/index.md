# 부록

본 장은 Plannotator의 기술 참고 자료를 제공하며, API 문서, 데이터 모델 정의, 라이선스 정보를 포함합니다. 이 내용은 주로 시스템의 내부 메커니즘을 깊이 이해하거나 2차 개발이 필요한 사용자를 대상으로 합니다.

## 본 장 내용

| 페이지 | 설명 |
| --- | --- |
| [API 참고](./api-reference/) | Plannotator가 제공하는 모든 API 엔드포인트와 요청/응답 형식을 확인합니다 |
| [데이터 모델](./data-models/) | Plannotator가 사용하는 Annotation, Block, CodeAnnotation 등의 데이터 모델을 알아봅니다 |
| [라이선스](./license/) | Plannotator의 라이선스(BSL-1.1)와 사용 제한 사항을 확인합니다 |

## 학습 경로 제안

부록 내용은 상대적으로 독립적이므로, 필요에 따라 선택적으로 읽을 수 있습니다:

1. **API 인터페이스를 알고 싶다면** → 바로 [API 참고](./api-reference/)를 읽으세요
2. **데이터 구조를 이해하고 싶다면** → [데이터 모델](./data-models/)을 읽으세요
3. **사용 제한을 알고 싶다면** → [라이선스](./license/)를 읽으세요

::: tip 팁
부록 내용은 주로 참고용이므로 순차적으로 학습할 필요가 없습니다. 구체적인 문제가 발생했을 때 해당 장을 참조하는 것을 권장합니다.
:::
