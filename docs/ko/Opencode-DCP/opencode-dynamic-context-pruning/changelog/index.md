# 변경 로그

이 섹션은 DCP 플러그인의 버전 진화 역사를 기록하여 각 버전의 새로운 기능, 개선 사항 및 수정 사항을 이해하고 업그레이드 결정을 내릴 수 있도록 도와줍니다.

## 이 장의 내용

| 페이지 | 설명 |
| --- | ---|
| [버전 기록](./version-history/) | v1.0.1부터 최신 버전까지의 전체 업데이트 기록을 확인하세요. 새로운 기능, 개선 사항 및 버그 수정이 포함되어 있습니다. |

## 읽기 권장사항

변경 로그는 주로 다음 시나리오에서 사용됩니다:

1. **업그레이드 전 검토** - 새 버전에 어떤 변경 사항이 있는지 파악하고 업그레이드가 필요한지 평가합니다.
2. **문제 해결** - 특정 기능이 어떤 버전에서 도입되었는지 또는 특정 버그가 어떤 버전에서 수정되었는지 확인합니다.
3. **발전 방향 파악** - 버전 기록을 통해 DCP의 기능 진화 및 최적화 초점을 이해합니다.

::: tip 최신 상태 유지
OpenCode 구성에서 `@latest` 태그를 사용하여 최신 버전의 기능과 수정 사항을 자동으로 가져오는 것을 권장합니다.
:::
