# 부록

본 챕터에서는 DCP의 기술 참고 자료를 제공하며, 내부 아키텍처 설계, 토큰 계산 원리, 전체 API 문서가 포함되어 있습니다. 이 내용은 DCP의 작동 원리를 깊이 이해하거나 2차 개발을 하려는 사용자를 대상으로 합니다.

## 본 챕터 내용

| 문서 | 설명 | 대상 독자 |
| --- | --- | ---|
| [아키텍처 개요](./architecture/) | DCP의 내부 아키텍처, 모듈 종속성 및 호출 체인 이해 | DCP 작동 원리를 이해하려는 사용자 |
| [토큰 계산 원리](./token-calculation/) | DCP가 토큰 사용량과 절약 통계를 계산하는 방법 이해 | 절약 효과를 정확히 평가하려는 사용자 |
| [API 참조](./api-reference/) | 구성 인터페이스, 도구 규격, 상태 관리를 포함한 전체 API 문서 | 플러그인 개발자 |

## 학습 경로

```
아키텍처 개요 → 토큰 계산 원리 → API 참조
   ↓              ↓              ↓
 설계 이해      통계 이해      확장 개발
```

**권장 순서**:

1. **아키텍처 개요**: 먼저 전체적인 인식을 확립하고 DCP의 모듈 구분 및 호출 체인을 이해합니다
2. **토큰 계산 원리**: `/dcp context` 출력의 계산 로직을 이해하고 토큰 분석 방법을 학습합니다
3. **API 참조**: 플러그인 개발이나 2차 개발이 필요한 경우 전체 인터페이스 문서를 참조합니다

::: tip 필요에 따라 읽기
DCP를 잘 사용하기만 하면 되는 경우라면 본 챕터는 건너뛸 수 있습니다. 이 내용은 주로 원리를 깊이 이해하거나 개발이 필요한 사용자를 대상으로 합니다.
:::
