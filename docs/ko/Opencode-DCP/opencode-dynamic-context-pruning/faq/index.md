# 자주 묻는 질문

이 챕터에서는 DCP 사용 과정에서 발생하는 일반적인 질문에 대한 답변과 모범 사례를 수집하여, 빠른 문제 해결과 최적화된 사용 경험을 제공합니다.

## 챕터 내용

<div class="grid-cards">

### [FAQ 및 문제 해결](./troubleshooting/)

DCP 사용 중 일반적인 문제를 해결합니다. 여기에는 구성 오류 수정, 디버깅 모드 활성화, 토큰 감소 실패 원인 분석 등의 문제 해결 기법이 포함됩니다.

### [모범 사례](./best-practices/)

DCP의 최적 구성 전략을 학습하고, 프롬프트 캐싱과 토큰 절약 사이의 균형, 보호 메커니즘 설정, 자동 전략 최적화 등의 고급 기법을 익힙니다.

</div>

## 학습 경로

```
FAQ 및 문제 해결 → 모범 사례
```

1. **먼저 문제 해결 가이드 보기**: 문제 발생 시 빠르게 솔루션을 찾으세요
2. **그런 다음 모범 사례 학습하기**: 구성을 최적화하여 토큰 절약 효과를 극대화하세요
