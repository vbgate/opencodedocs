# 핵심 기능

본 장에서는 DCP의 세 가지 핵심 능력을 심도 있게 다룹니다: 자동 가지치기 전략, LLM 기반 도구 및 Slash 명령어. 이 기능들을 마스터하면 DCP의 Token 최적화 잠재력을 충분히 활용할 수 있습니다.

## 이 장의 내용

<div class="grid-cards">

### [자동 가지치기 전략](./auto-pruning/)

DCP의 세 가지 자동 전략이 어떻게 작동하는지 깊이 이해하세요: 중복 제거 전략, 덮어쓰기 전략 및 오류 정리 전략. 각 전략의 트리거 조건과 적용 시나리오를 알아봅니다.

### [LLM 기반 가지치기 도구](./llm-tools/)

AI가 `discard` 및 `extract` 도구를 자율적으로 호출하여 컨텍스트를 최적화하는 방법을 이해하세요. 이는 DCP의 가장 지능적인 기능으로, AI가 컨텍스트 관리에 능동적으로 참여하게 합니다.

### [Slash 명령어 사용](./commands/)

모든 DCP 명령어의 사용법을 마스터하세요: `/dcp context`로 컨텍스트 상태 확인, `/dcp stats`로 통계 확인, `/dcp sweep`로 수동 가지치기 트리거.

</div>

## 학습 경로

이 장의 내용을 다음 순서대로 학습하는 것을 권장합니다:

```
자동 가지치기 전략 → LLM 기반 도구 → Slash 명령어
      ↓              ↓            ↓
   원리 이해      지능형 가지치기 마스터    모니터링 및 디버깅 학습
```

1. **먼저 자동 가지치기 전략 학습**: 이는 DCP의 기초이며, 세 가지 전략의 작동 원리를 이해합니다
2. **다음으로 LLM 기반 도구 학습**: 자동 전략을 이해한 후, 더 고급화된 AI 주도 가지치기 능력을 학습합니다
3. **마지막으로 Slash 명령어 학습**: 모니터링 및 수동 제어 방법을 마스터하여 디버깅 및 최적화를 용이하게 합니다
