# 부록: 이벤트 유형 및 구성 참고서

본 장에서는 opencode-notify의 이벤트 유형과 구성 옵션을 깊이 있게 이해할 수 있도록 참고 문서와 구성 예제를 제공합니다. 이 내용은 참고 자료로 활용할 수 있으며, 순서대로 학습할 필요가 없습니다.

## 학습 경로

### 1. [이벤트 유형 설명](./event-types/)

플러그인이 수신하는 OpenCode 이벤트 유형 및 트리거 조건을 알아보세요.

- 네 가지 이벤트 유형(session.idle, session.error, permission.updated, tool.execute.before)
- 각 이벤트의 트리거 타이밍 및 처리 로직
- 부모 세션 확인, 무음 시간 확인, 터미널 포커스 확인의 필터링 규칙
- 플랫폼별 기능 차이

### 2. [구성 파일 예제](./config-file-example/)

완전한 구성 파일 예제와 모든 필드의 상세 주석을 확인하세요.

- 완전한 구성 파일 템플릿
- notifyChildSessions, sounds, quietHours, terminal 등 필드 설명
- macOS에서 사용 가능한 효과음 전체 목록
- 최소화 구성 예제
- 플러그인 비활성화 방법
