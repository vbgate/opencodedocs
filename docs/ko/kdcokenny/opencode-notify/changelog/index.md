# 업데이트 로그

이 섹션에서는 opencode-notify 플러그인의 버전 히스토리와 중요한 변경 사항을 기록하여 기능 업데이트, 문제 수정 및 설정 개선 사항을 추적하는 데 도움을 드립니다.

## 하위 페이지

### [업데이트 로그](./release-notes/)

시간 역순으로 기록된 플러그인 변경 히스토리를 확인하세요.

**포함 내용**:
- 기능 업데이트 기록
- 문제 수정 설명
- 설정 개선 상세 정보
- 하위 호환성 변경 알림

::: tip 버전 안내
이 플러그인은 OCX를 통해 배포되며 버전 번호 없이 Git 커밋 히스토리를 통해 변경 사항을 추적하는 전략을 사용합니다.
:::

## 사전 요구사항
