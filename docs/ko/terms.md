# 사용자 서비스 약관

최종 업데이트 일자: 2026-01-01
발효 일자: 2026-01-01

OpenCodeDocs를 이용해 주셔서 감사합니다! 저희 서비스를 사용하기 전에 본 사용자 서비스 약관(이하 "본 약관")을 주의 깊게 읽어보시기 바랍니다. 저희 서비스를 사용하는 것은 본 약관의 규정에 동의하는 것을 의미합니다.

## 1. 서비스 설명

OpenCodeDocs는 AI 도구 튜토리얼에 특화된 오픈 플랫폼으로, 다음과 같은 서비스를 제공합니다:

- **튜토리얼 콘텐츠**: 오픈 소스 AI 도구(Claude Code, Cursor, LangChain 등) 사용 튜토리얼
- **코드 검증**: 소스 코드 저장소를 기반으로 튜토리얼 코드의 실행 가능성을 자동으로 검증
- **커뮤니티 기능**: 사용자는 GitHub를 통해 콘텐츠 기여 및 토론에 참여할 수 있음

저희는 사전 통지 없이 언제든지 전체 또는 일부 서비스를 수정, 일시 중지 또는 종료할 권리를 보유합니다.

## 2. 계정 등록 및 관리

### 등록 조건
- 만 16세 이상이어야 합니다(또는 거주 국가의 법적 성인 연령 도달)
- 정확하고 진실한 등록 정보를 제공해야 합니다
- 사용자마다 하나의 계정만 등록할 수 있습니다

### 계정 보안
- 계정 자격 증명을 안전하게 관리할 책임은 사용자에게 있으며, 타인과 공유해서는 안 됩니다
- 계정이 도용당한 것을 발견하면 즉시 저희에게 연락해 주십시오
- 계정 관리 부실로 인한 손실은 사용자가 부담해야 합니다

### 계정 삭제
언제든지 계정 삭제를 신청할 수 있습니다. 삭제 후 개인 데이터는 30일 이내에 삭제되지만, 공개적으로 게시된 콘텐츠는 보존될 수 있습니다.

## 3. 사용자 행동 규범

본 플랫폼을 사용할 때 다음과 같은 행위에 동의하지 않습니다:

- 불법적, 유해적, 위협적, 괴롭힘, 명예훼손 또는 침해적인 콘텐츠 게시
- 타인을 사칭하거나 신원을 거짓으로 진술
- 승인 없이 타인의 계정 또는 시스템에 액세스
- 서비스의 정상적인 운영을 방해하거나 파괴
- 자동화된 도구로 대규모 콘텐츠 스크래핑(합리적인 API 사용 제외)
- 라이선스 계약을 준수하지 않고 튜토리얼 콘텐츠를 상업적 목적으로 사용

## 4. 지식재산권

### 오픈 소스 라이선스 설명
- **튜토리얼 콘텐츠**: [CC-BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/) 라이선스를 채택하여, 비상업적 공유 및 개조를 허용하지만 저작자 표기 및 동일한 방식으로 공유해야 함
- **웹사이트 구축**: 본 프로젝트는 [VitePress](https://vitepress.dev/)를 기반으로 구축되었으며 MIT 라이선스를 준수함
- **사용자 기여**: 제출한 콘텐츠는 상기 라이선스를 준수하는 전제 하에 플랫폼에서 사용할 수 있도록 기본적으로 라이선스가 부여됨

OpenCodeDocs의 상표, 로고 및 기타 브랜드 식별자는 법적으로 보호되며, 승인 없이 사용할 수 없습니다.

## 5. 면책 조항

- **서비스 중단**: 저희는 서비스의 중단 없는 운영이나 오류가 없음을 보장하지 않습니다. 유지보수, 업그레이드 또는 불가항력으로 인한 서비스 중단에 대해 책임을 지지 않습니다
- **콘텐츠 정확성**: 튜토리얼 콘텐츠는 참고용으로만 제공되며, 모든 환경에서의 사용 가능성을 보장하지 않습니다
- **코드 사용 위험**: 튜토리얼의 코드를 실행하는 것은 사용자의 책임이며, 격리된 환경에서 테스트하는 것을 권장합니다
- **제3자 링크**: 본 사이트에는 제3자 웹사이트에 대한 링크가 포함될 수 있으며, 해당 콘텐츠에 대해 책임을 지지 않습니다

## 6. 약관 수정

저희는 수시로 본 약관을 수정할 수 있습니다. 수정된 약관은 웹사이트에 게시되며, 다음과 같은 방법으로 통지됩니다:

- 웹사이트의 눈에 띄는 위치에 공지
- 등록된 이메일로 통지(해당되는 경우)

서비스를 계속 사용하는 것은 수정된 약관에 동의하는 것을 의미합니다. 수정에 동의하지 않으시는 경우 서비스 사용을 중단하십시오.

## 7. 분쟁 해결

- **적용 법률**: 본 약관은 중화인민공화국 법률의 적용을 받습니다
- **분쟁 해결**: 당사자 간에 우호적으로 협의하여 분쟁을 해결해야 합니다. 협의가 성립되지 않을 경우, 어느 당사자든 피고 소재지의 관할권을 가진 인민법원에 소송을 제기할 수 있습니다
