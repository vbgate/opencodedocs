# 빠른 시작

이 섹션은 oh-my-opencode를 빠르게 설치하고 사용하는 방법을 안내합니다. 기본 설정부터 메인 오케스트레이터, 그리고 모든 기능을 한 번에 활성화하는 Ultrawork 모드까지 단계별로 진행합니다.

## 학습 경로

다음 순서를 따라 oh-my-opencode의 기본 기능을 단계적으로 마스터하세요:

### 1. [빠른 설치 및 설정](./installation/)

oh-my-opencode 설치, Provider 설정, 기본 검증을 완료하세요.
- oh-my-opencode 설치
- AI Provider 설정 (Claude, OpenAI, Gemini 등)
- 설정이 올바른지 확인
- 설정 파일 구조 이해

### 2. [Sisyphus 만나기: 메인 오케스트레이터](./sisyphus-orchestrator/)

에이전트 오케스트레이션의 핵심 개념을 이해하고 Sisyphus 에이전트의 기본 사용법을 배우세요.
- 에이전트 오케스트레이션 시스템이란 무엇인가
- Sisyphus 에이전트의 핵심 기능
- 기본 사용법과 모범 사례
- 다중 모델 협업의 워크플로우 이해

### 3. [Ultrawork 모드: 한 번에 모든 기능 활성화](./ultrawork-mode/)

ultrawork 키워드를 사용하여 모든 전문 에이전트와 병렬 작업 기능을 빠르게 활성화하는 방법을 배우세요.
- Ultrawork 모드란 무엇인가
- ultrawork 키워드를 트리거하는 방법
- 10명의 전문 에이전트 협업
- 병렬 백그라운드 작업 관리

## 전제 조건

이 섹션을 시작하기 전에 다음을 확인하세요:
- Node.js 18+가 설치되어 있어야 합니다 (버전 20 이상 권장)
- 기본 명령줄 작업에 익숙해야 합니다
- 최소 하나의 AI 서비스 API 키가 있어야 합니다 (Claude, OpenAI 또는 Gemini)

::: tip API 키가 없으신가요?
아서 AI 서비스 API 키가 없다면 Claude부터 신청하여 시작하는 것이 현재 가장 권장되는 선택입니다.
:::
