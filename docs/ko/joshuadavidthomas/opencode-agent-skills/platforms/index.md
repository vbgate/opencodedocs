# 플랫폼 기능

이 장에서는 OpenCode Agent Skills의 핵심 기능 모듈을 심층적으로 설명합니다. 스킬 발견, 쿼리, 로드, 자동 추천, 스크립트 실행, 파일 읽기를 포함합니다. 이러한 기능을 마스터하면 플러그인의 스킬 관리 능력을 최대한 활용하여 AI가 개발 작업에 더 효율적으로 도움을 줄 수 있습니다.

## 전제 조건

::: warning 시작하기 전 확인
이 장을 학습하기 전에 다음 사항을 완료했는지 확인하세요:

- [OpenCode Agent Skills 설치](../start/installation/) - 플러그인이 올바르게 설치되고 실행 중인지 확인
- [첫 번째 스킬 만들기](../start/creating-your-first-skill/) - 스킬의 기본 구조 이해
:::

## 학습 경로

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           권장 학습 순서                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1. 스킬 발견 메커니즘  ──→  2. 사용 가능한 스킬 나열  ──→  3. 스킬 로드      │
│         │                     │                    │                    │
│         │                     │                    │                    │
│         ▼                     ▼                    ▼                    │
│   스킬의 출처 이해        스킬 찾기 방법 학습      로드 방법 마스터           │
│                                                                         │
│                              │                                          │
│                              ▼                                          │
│                                                                         │
│   4. 자동 스킬 추천  ←──  5. 스크립트 실행  ←──  6. 파일 읽기            │
│         │                    │                  │                       │
│         ▼                    ▼                  ▼                       │
│   지능형 매칭 이해       자동화 실행          지원 파일 접근               │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**순서대로 학습을 권장합니다**:

1. **먼저 발견 메커니즘 학습** - 스킬이 어디서 오는지, 우선순위가 어떻게 결정되는지 이해
2. **다음으로 스킬 쿼리 학습** - `get_available_skills` 도구 사용법 마스터
3. **그다음 스킬 로드 학습** - `use_skill` 및 XML 인젝션 메커니즘 이해
4. **이어서 자동 추천 학습** - 시맨틱 매칭이 작동하는 방식 이해 (선택 사항, 원리 중심)
5. **마지막으로 스크립트 및 파일 학습** - 이 두 가지는 고급 기능으로 필요에 따라 학습

::: tip 빠른 시작 경로
빠르게 사용해보고 싶다면, 처음 세 과정만 학습하면 됩니다 (발견 → 쿼리 → 로드), 나머지는 필요에 따라 보충하세요.
:::
