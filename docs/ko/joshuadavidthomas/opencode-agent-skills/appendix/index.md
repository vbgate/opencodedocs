# 부록

이 장에서는 OpenCode Agent Skills의 참고자료와 개발 규범을 제공하여 고품질 스킬 작성과 API 인터페이스 빠른 조회를 도와드립니다.

## 이 장의 내용

| 페이지 | 설명 |
|--- | ---|
| [스킬 개발 모범 사례](./best-practices/) | 고품질 스킬 작성을 위한 기술과 규범을 마스터합니다. 네이밍 규범, 설명 작성, 디렉토리 구성, 스크립트 사용 등을 포함합니다 |
| [API 도구 참고](./api-reference/) | 사용 가능한 모든 도구의 상세 매개변수와 반환값을 확인합니다. `get_available_skills`, `read_skill_file`, `run_skill_script`, `use_skill`를 포함합니다 |

## 학습 경로 제안

::: tip 권장 순서
1. **먼저 모범 사례 학습** → 스킬 개발의 규범과 기술을 이해하고 일반적인 오류를 방지합니다
2. **그 다음 API 참고 조회** → 실제 개발에서 문제가 발생할 때 구체적인 매개변수와 반환값을 조회합니다
:::

이 두 내용의 목적이 다릅니다:

- **모범 사례**: 스킬 작성을 시작하기 전에 처음부터 끝까지 읽어보고 올바른 개발 습관을 확립하는 데 적합합니다
- **API 참고**: 필요할 때 구체적인 세부사항을 조회하는 참고서로 사용하기에 적합합니다
