## 학습 경로

다음 순서대로 이 장을 학습하는 것을 권장합니다:

```
일반적인 문제 해결 → 보안 고려사항
```

1. **문제 해결 먼저**: 문제가 발생했을 때, "일반적인 문제 해결"을 먼저 참고하여 빠르게 진단하고 해결하세요
2. **보안 학습**: 플러그인의 보안 메커니즘을 이해하고 안전한 사용 습관을 기르세요

::: tip 문제 발생 시 대처 방법
플러그인을 사용 중에 구체적인 문제가 발생한 경우, "일반적인 문제 해결"로 바로 이동하여 해당 증상을 찾아보세요. 플러그인의 보안 설계를 체계적으로 이해하려면 "보안 고려사항"을 읽어보세요.
:::
