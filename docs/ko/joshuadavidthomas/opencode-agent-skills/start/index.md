# 빠른 시작

이 장에서는 OpenCode Agent Skills 플러그인을 처음부터 사용하는 방법을 안내합니다. 플러그인의 핵심 가치를 이해하고, 설치 및 구성을 완료한 후 첫 번째 스킬을 생성하게 됩니다.

## 학습 경로

다음 순서대로 학습하는 것을 권장합니다:

1. **[OpenCode Agent Skills란 무엇인가요?](./what-is-opencode-agent-skills/)** — 플러그인이 무엇을 할 수 있는지 먼저 이해하고 전체적인 인식을 구축하세요
2. **[설치 가이드](./installation/)** — 플러그인을 설치하여 OpenCode에 스킬 관리 기능을 부여하세요
3. **[첫 번째 스킬 만들기](./creating-your-first-skill/)** — 직접 실습하여 나만의 첫 번째 스킬을 만들어보세요

::: tip 예상 소요 시간
이 장을 완료하는 데 약 30-45분이 소요됩니다.
:::
