# 고급 기능

이 장에서는 Supermemory의 저수준 메커니즘과 고급 설정 옵션을 자세히 설명합니다. 메모리 엔진의 성능을 최적화하고 프로젝트 요구사항에 따라 플러그인 동작을 사용자 정의하는 방법을 배웁니다.

## 이 장의 내용

<div class="grid-cards">

### [선점형 압축 원리](./compaction/)

Token 임계값 트리거 자동 압축 메커니즘을 자세히 이해. Supermemory가 컨텍스트가 오버플로우되기 전에 어떻게 구조화된 요약을 생성하고 영구적으로 저장하여 긴 세션에서 "망각" 문제를 방지하는지 배웁니다.

### [심층 설정 상세](./configuration/)

압축 임계값, 키워드 트리거 규칙, 검색 매개변수를 사용자 정의. 설정 파일의 모든 옵션을 마스터하여 Supermemory를 워크플로우에 완벽하게 맞추세요.

</div>

## 학습 경로

```
┌─────────────────────────────────────────────────────────────┐
│  1단계              2단계                                     │
│  선점형 압축 원리   →   심층 설정 상세                         │
│  (메커니즘 이해)      (사용자 정의 실습)                        │
└─────────────────────────────────────────────────────────────┘
```

**권장 순서**:

1. **먼저 압축 원리 학습**: Supermemory가 자동으로 컨텍스트를 관리하는 방법 이해, 이는 고급 설정의 기초.
2. **다음으로 설정 상세 학습**: 메커니즘 이해 후 합리적인 설정 결정을 내릴 수 있음(예: 임계값을 얼마로 설정하는 것이 적합한지).
