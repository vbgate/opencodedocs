# 변경 로그

본 섹션은 OpenSkills의 버전 변경 이력을 기록하며, 각 버전의 새로운 기능, 개선사항 및 문제 수정사항을 알려드립니다.

## 본 챕터 내용

| 페이지 | 설명 |
| --- | ---|
| [변경 로그](./changelog/) | 새로운 기능, 개선사항, 문제 수정 및 보안 업데이트를 포함한 완전한 버전 변경 기록 |

## 변경 로그 사용 방법

변경 로그는 버전 번호 역순으로 정렬되어 있습니다(최신 버전이 가장 위에). 각 버전은 다음과 같은 카테고리를 포함합니다:

- **새로운 기능** - 새로 추가된 기능과 명령
- **개선사항** - 기존 기능의 최적화 및 강화
- **문제 수정** - 수정된 버그
- **보안** - 보안 관련 개선사항

::: 팁 업그레이드 권장사항
OpenSkills를 업그레이드하기 전에 버전 변경사항을 확인하기 위해 변경 로그를 확인하는 것을 권장합니다. 특히 다음에 주목하세요:
- **Breaking Changes**(호환성이 깨지는 변경)
- 새로운 기능의 사용 방법
- 알려진 문제의 수정 상태
:::
