# 자주 묻는 질문

이 장에서는 OpenSkills 사용 중 발생하는 자주 묻는 질문, 문제 해결 방법 및 설계 철학에 대한 설명을 수집하여 문제를 빠르게 파악하고 해결하는 데 도움을 드립니다.

## 이 장의 내용

<div class="grid-cards">

### [자주 묻는 질문](./faq/)

설치, 구성, 사용 등에 대한 일반적인 질문에 답변합니다. OpenSkills와 Claude Code의 차이점, 스킬 설치 위치, AGENTS.md 동기화 문제, 다중 프로젝트 스킬 공유 등을 포함합니다.

### [문제 해결](./troubleshooting/)

일반적인 오류의 진단 및 수정 방법을 제공합니다. Git clone 실패, SKILL.md를 찾을 수 없음, 스킬을 찾을 수 없음, 권한 오류, 업데이트 건너뛰기 등의 문제에 대한 자세한 해결 단계를 다룹니다.

### [왜 CLI인가?](./why-cli-not-mcp/)

OpenSkills가 MCP 대신 CLI를 선택한 설계 이유를 설명합니다. 두 가지의 위치 차이를 심층 비교하고 스킬 시스템이 정적 파일 모드에 더 적합한 이유를 이해합니다.

</div>

## 학습 경로

```
문제가 발생했나요?
    │
    ├─ 사용 방법을 모름 ──────────→ 자주 묻는 질문
    │                           (개념, 구성, 사용 팁)
    │
    ├─ 오류 발생 ────────────────→ 문제 해결
    │                           (오류 진단, 수정 단계)
    │
    └─ 설계 철학을 알고 싶음 ────────→ 왜 CLI인가?
                                 (기술 선택, 아키텍처 결정)
```

**권장 순서**:

1. **먼저 자주 묻는 질문 보기** → 핵심 개념과 사용 팁을 빠르게 이해
2. **오류 발생 시 문제 해결 보기** → 오류 메시지로 솔루션 검색
3. **심층 이해를 위해 설계 철학 보기** → CLI vs MCP 기술적 고려사항 이해
