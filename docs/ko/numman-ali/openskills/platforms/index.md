# 플랫폼 기능

이 장에서는 OpenSkills CLI의 핵심 명령어와 스킬 관리 기능을 심도 있게 다루어, "사용할 수 있음"에서 "마스터"로 발전하도록 도와드립니다.

## 사전 요구 사항

::: warning 시작 전 확인
이 장을 학습하기 전에 [빠른 시작](../start/) 장을 완료했는지 확인하세요. 특히:
- [OpenSkills 설치](../start/installation/): OpenSkills CLI가 성공적으로 설치되었는지 확인
- [첫 번째 스킬 설치](../start/first-skill/): 기본적인 스킬 설치 프로세스 이해
:::

## 이 장 내용

이 장은 OpenSkills의 모든 핵심 기능을 다루는 6가지 주제를 포함합니다:

### [명령어 상세 설명](./cli-commands/)

7개 명령어, 파라미터 및 플래그를 완전히 소개하고 명령어 요약 표를 제공합니다. 명령어 사용법을 빠르게 찾아야 하는 사용자에게 적합합니다.

### [설치 소스 상세 설명](./install-sources/)

GitHub 저장소, 로컬 경로, 개인 Git 저장소의 세 가지 설치 소스를 자세히 설명합니다. 각 방식의 사용 시나리오, 명령어 형식 및 주의사항을 다룹니다.

### [전역 설치 vs 프로젝트 로컬 설치](./global-vs-project/)

`--global`과 기본(프로젝트 로컬) 설치의 차이를 설명하여, 적절한 설치 위치를 선택하고 스킬 검색 우선순위 규칙을 이해할 수 있도록 돕습니다.

### [설치된 스킬 나열](./list-skills/)

`list` 명령어를 사용하여 설치된 스킬을 보는 방법을 알려드리고 `(project)` 및 `(global)` 위치 태그의 의미를 이해합니다.

### [스킬 업데이트](./update-skills/)

`update` 명령어를 사용하여 설치된 스킬을 새로 고치는 방법을 안내합니다. 모든 스킬 업데이트와 지정된 스킬 업데이트를 지원하여 스킬을 원본 저장소와 동기화합니다.

### [스킬 삭제](./remove-skills/)

대화형 `manage` 명령어와 스크립트 방식 `remove` 명령어의 두 가지 삭제 방법을 소개하여 스킬 라이브러리를 효율적으로 관리할 수 있도록 돕습니다.

## 학습 경로

요구사항에 따라 적절한 학습 경로를 선택하세요:

### 경로 A: 체계적 학습(초보자 추천)

순서대로 모든 내용을 학습하여 완전한 지식 체계를 구축하세요:

```
명령어 상세 설명 → 설치 소스 → 전역 vs 프로젝트 → 스킬 나열 → 스킬 업데이트 → 스킬 삭제
```

### 경로 B: 필요에 따른 참조

현재 작업에 따라 바로 이동하세요:

| 하고 싶은 것 | 이 문서 읽기 |
|--- | ---|
| 특정 명령어 사용법 찾기 | [명령어 상세 설명](./cli-commands/) |
| 개인 저장소에서 스킬 설치 | [설치 소스 상세 설명](./install-sources/) |
| 전역 또는 프로젝트 설치 결정 | [전역 vs 프로젝트](./global-vs-project/) |
| 설치된 스킬 확인 | [설치된 스킬 나열](./list-skills/) |
| 스킬을 최신 버전으로 업데이트 | [스킬 업데이트](./update-skills/) |
| 사용하지 않는 스킬 정리 | [스킬 삭제](./remove-skills/) |
