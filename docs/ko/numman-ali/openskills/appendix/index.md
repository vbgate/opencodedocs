# 부록

이 장에서는 OpenSkills의 기술 참고 자료를 제공합니다. 여기에는 명령줄 API, 파일 형식 규격 및 용어 정의가 포함됩니다. 특정 명령 매개변수를 확인하거나 생성된 파일의 구조를 이해하거나 특정 용어의 의미를 파악해야 할 때 이곳이 바로 빠른 참고 가이드입니다.

## 이 장의 내용

| 문서 | 설명 |
| --- | --- |
| [CLI API 참고](./cli-api/) | 모든 명령, 옵션, 매개변수의 완전한 문서와 사용 예제 |
| [AGENTS.md 형식 규격](./agents-md-format/) | sync 명령이 생성하는 AGENTS.md 파일 구조, XML 태그 및 스킬 목록 형식 포함 |
| [파일 구조](./file-structure/) | OpenSkills가 생성하는 디렉토리와 파일 설명, `.openskills/`에 무엇이 있는지 이해하는 데 도움 |
| [용어집](./glossary/) | OpenSkills와 AI 스킬 시스템의 핵심 용어 설명 |

## 학습 경로 권장사항

부록은 참고 자료이므로 처음부터 끝까지 읽을 필요가 없습니다. 권장되는 사용 방법:

1. **명령 문제가 있을 때** → [CLI API 참고](./cli-api/) 확인
2. **AGENTS.md 생성 방식을 알고 싶을 때** → [AGENTS.md 형식 규격](./agents-md-format/) 참고
3. **파일 위치가 헷갈릴 때** → [파일 구조](./file-structure/) 확인
4. **낯선 용어를 봤을 때** → [용어집](./glossary/) 검색

::: tip 초보자 권장사항
OpenSkills를 처음 사용하는 경우, 먼저 [빠른 시작](../start/quick-start/) 튜토리얼을 완료한 후 문제가 발생하면 이 부록을 참조하세요.
:::
