# Quick Start

This chapter helps you get started with the OpenCode Agent Skills plugin from scratch. You'll learn about the plugin's core value, complete the installation and configuration, and create your first skill.

## What You'll Learn

<div class="grid-cards">

<a href="./what-is-opencode-agent-skills/" class="card">
  <h3>What is OpenCode Agent Skills?</h3>
  <p>Learn about the plugin's core value and features, including dynamic skill discovery, context injection, compression and recovery mechanisms.</p>
</a>

<a href="./installation/" class="card">
  <h3>Installation Guide</h3>
  <p>Complete plugin installation and verify it works, supporting basic installation, fixed version installation, and local development installation.</p>
</a>

<a href="./creating-your-first-skill/" class="card">
  <h3>Create Your First Skill</h3>
  <p>Master the skill directory structure and SKILL.md format specification, then create and test a simple skill hands-on.</p>
</a>

</div>

## Learning Path

We recommend learning in the following order:

1. **[What is OpenCode Agent Skills?](./what-is-opencode-agent-skills/)** — First understand what the plugin can do and build an overall understanding
2. **[Installation Guide](./installation/)** — Install the plugin to give OpenCode skill management capabilities
3. **[Create Your First Skill](./creating-your-first-skill/)** — Hands-on practice creating your very first skill

::: tip Estimated Time
This chapter takes approximately 30-45 minutes to complete.
:::

## Next Steps

After completing this chapter, you can continue to **[Platform Features](../platforms/)** to learn more about advanced features like skill discovery mechanisms, skill loading, and automatic matching.
