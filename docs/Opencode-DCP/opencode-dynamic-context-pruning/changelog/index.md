# Changelog

This chapter records the version evolution history of the DCP plugin, helping you understand the new features, improvements, and fixes in each version so you can make upgrade decisions.

## Chapter Contents

| Page | Description |
|------|-------------|
| [Version History](./version-history/) | View complete update records from v1.0.1 to the latest version, including new features, improvements, and bug fixes |

## Reading Recommendations

The changelog is primarily used for the following scenarios:

1. **Before Upgrading** - Learn what changes the new version brings, evaluate whether to upgrade
2. **Troubleshooting** - Confirm which version introduced a feature or which version fixed a bug
3. **Understand Development Direction** - Learn DCP's feature evolution and optimization focus through version history

::: tip Stay Up-to-Date
We recommend using the `@latest` tag in your OpenCode configuration to automatically get the latest version's features and fixes.
:::
