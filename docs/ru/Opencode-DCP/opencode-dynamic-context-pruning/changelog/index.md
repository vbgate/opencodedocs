# Журнал изменений

В этом разделе представлена история версий плагина DCP, которая поможет вам узнать о новых функциях, улучшениях и исправлениях каждой версии и принять обоснованное решение об обновлении.

## Содержание раздела

| Страница | Описание |
| --- | --- |
| [История версий](./version-history/) | Полный список обновлений от v1.0.1 до последней версии, включая новые функции, улучшения и исправления ошибок |

## Рекомендации по чтению

Журнал изменений полезен в следующих случаях:

1. **Перед обновлением** — узнайте о нововведениях последней версии и оцените необходимость обновления
2. **При поиске проблемы** — проверьте, в какой версии была добавлена определённая функция или исправлена ошибка
3. **Для понимания развития** — через историю версий проследите эволюцию функциональности DCP и приоритеты оптимизации

::: tip Рекомендация по обновлению
Рекомендуется использовать тег `@latest` в настройках OpenCode для автоматической загрузки последней версии с новыми функциями и исправлениями.
:::
