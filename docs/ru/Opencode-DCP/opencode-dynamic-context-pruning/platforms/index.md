# Ключевые функции

В этой главе подробно рассматриваются три ключевые возможности DCP: автоматические стратегии обрезки, инструменты на базе LLM и команды Slash. Освоив эти функции, вы сможете в полной мере использовать потенциал DCP для оптимизации токенов.

## Содержание главы

<div class="grid-cards">

### [Автоматические стратегии обрезки](./auto-pruning/)

Глубокое понимание работы трёх автоматических стратегий DCP: стратегия дедупликации, стратегия перезаписи и стратегия очистки ошибок. Узнайте условия срабатывания и сценарии применения каждой стратегии.

### [Инструменты обрезки на базе LLM](./llm-tools/)

Поймите, как ИИ самостоятельно вызывает инструменты `discard` и `extract` для оптимизации контекста. Это самая интеллектуальная функция DCP, позволяющая ИИ активно участвовать в управлении контекстом.

### [Использование команд Slash](./commands/)

Освойте все команды DCP: `/dcp context` для просмотра состояния контекста, `/dcp stats` для просмотра статистики, `/dcp sweep` для ручного запуска обрезки.

</div>

## Путь обучения

Рекомендуется изучать содержание этой главы в следующем порядке:

```
Автоматические стратегии обрезки → Инструменты на базе LLM → Команды Slash
↓                      ↓                      ↓
Понимание принципов    Освоение интеллектуальной обрезки    Изучение мониторинга и отладки
```

1. **Сначала изучите автоматические стратегии обрезки**: это основа DCP, поймите принцип работы трёх стратегий
2. **Затем изучите инструменты на базе LLM**: после понимания автоматических стратегий изучите более продвинутые возможности активной обрезки ИИ
3. **Наконец, изучите команды Slash**: освойте методы мониторинга и ручного управления для удобной отладки и оптимизации
