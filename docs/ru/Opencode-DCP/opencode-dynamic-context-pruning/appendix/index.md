# Приложение

Эта глава предоставляет техническую документацию по DCP, включая дизайн внутренней архитектуры, принципы расчета токенов и полную документацию API. Этот материал ориентирован на пользователей, желающих глубоко понять принципы работы DCP или проводить вторичную разработку.

## Содержание главы

| Документ | Описание | Для кого |
| --- | --- | --- |
| [Обзор архитектуры](./architecture/) | Узнайте о внутренней архитектуре DCP, зависимостях модулей и цепочках вызовов | Пользователям, желающим понять принципы работы DCP |
| [Принципы расчета токенов](./token-calculation/) | Поймите, как DCP рассчитывает использование токенов и статистику экономии | Пользователям, желающим точно оценить эффект экономии |
| [Справочник API](./api-reference/) | Полная документация API, включая интерфейсы конфигурации, спецификации инструментов, управление состоянием | Разработчикам плагинов |

## Путь обучения

```
Обзор архитектуры → Принципы расчета токенов → Справочник API
       ↓                    ↓                      ↓
  Понимание дизайна    Понимание статистики    Разработка расширений
```

**Рекомендуемый порядок**:

1. **Обзор архитектуры**: сначала сформируйте общее понимание, узнайте о разделении модулей DCP и цепочках вызовов
2. **Принципы расчета токенов**: узнайте логику вычислений, выводимую командой `/dcp context`, научитесь анализировать распределение токенов
3. **Справочник API**: если вам нужно разработать плагин или провести вторичную разработку, ознакомьтесь с полной документацией интерфейсов

Если вы просто хотите эффективно использовать DCP, можно пропустить эту главу. Этот материал в основном ориентирован на пользователей, желающих глубоко понять принципы или провести разработку.
:::
