# Приложение

В этой главе представлена техническая документация Plannotator, включая документацию API, определения моделей данных и информацию о лицензии. Этот контент предназначен для пользователей, которым необходимо глубоко понять внутренние механизмы системы или доработать её.

## Рекомендации по обучению

Содержание приложения является относительно независимым, вы можете выборочно читать его по мере необходимости:

1. **Хотите узнать о интерфейсах API** → сразу читайте [Справочник по API](./api-reference/)
2. **Хотите понять структуры данных** → читайте [Модели данных](./data-models/)
3. **Хотите узнать об ограничениях использования** → читайте [Лицензия](./license/)

::: tip Совет
Содержимое приложения предназначено в основном для справки, его не нужно изучать по порядку. Рекомендуется обращаться к соответствующим разделам при возникновении конкретных вопросов.
:::
