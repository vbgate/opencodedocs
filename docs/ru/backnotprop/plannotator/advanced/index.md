# Продвинутые функции

Освоив базовые операции рецензирования планов и кода, эта глава познакомит вас с продвинутыми возможностями Plannotator: совместной работой в команде, интеграцией с приложениями для заметок, поддержкой удалённой разработки и гибкой настройкой переменных окружения.

::: warning Предварительные условия
Перед изучением этой главы убедитесь, что вы завершили:
- [Быстрый старт](../start/getting-started/): знакомство с основными концепциями Plannotator
- [Основы рецензирования планов](../platforms/plan-review-basics/): освоение базовых операций рецензирования планов
:::

## Содержание главы

| Урок | Описание | Сценарии использования |
| --- | --- | --- |
| [URL-шеринг](./url-sharing/) | Шеринг планов и комментариев через URL для командной работы без бэкенда | Необходимость поделиться результатами рецензирования с командой |
| [Интеграция с Obsidian](./obsidian-integration/) | Автоматическое сохранение одобренных планов в Obsidian vault | Управление базой знаний в Obsidian |
| [Интеграция с Bear](./bear-integration/) | Сохранение планов в Bear через x-callback-url | Пользователи macOS/iOS с приложением Bear |
| [Удалённый режим](./remote-mode/) | Использование в удалённых средах: SSH, devcontainer, WSL | Сценарии удалённой разработки |
| [Переменные окружения](./environment-variables/) | Обзор всех доступных переменных окружения и их назначения | Необходимость кастомизации поведения Plannotator |

## Рекомендуемый путь обучения

```
┌─────────────────────────────────────────────────────────────────┐
│  Рекомендуемый порядок изучения                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. URL-шеринг       ← Наиболее часто используется,           │
│       ↓                 необходим для командной работы          │
│  2. Интеграция с приложениями для заметок                       │
│       ← Выберите в зависимости от используемого приложения      │
│     ├─ Интеграция с Obsidian  (пользователи Obsidian)          │
│     └─ Интеграция с Bear      (пользователи Bear)               │
│       ↓                                                         │
│  3. Удалённый режим  ← Если работаете в удалённой среде         │
│       ↓                                                         │
│  4. Переменные окружения                                        │
│       ← Для глубокой кастомизации настроек                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

::: tip Изучайте по мере необходимости
Не обязательно проходить все материалы по порядку. Если вам нужна только функция шеринга, изучите только URL-шеринг; если вы не используете удалённую разработку, можете пропустить раздел об удалённом режиме.
:::
