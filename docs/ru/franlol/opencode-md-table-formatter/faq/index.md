# Часто задаваемые вопросы

## Обзор главы

Эта глава охватывает распространенные проблемы и методы устранения неполадок плагина opencode-md-table-formatter, помогая вам быстро находить и решать различные проблемы, возникающие в процессе форматирования таблиц.

---

## Навигация по подстраницам

### [FAQ: Что делать, если таблица не форматируется](troubleshooting/)

Полное введение в распространенные проблемы и решения при использовании плагина:

- **Таблица не форматируется**: проверка конфигурации, требований к версии, перезапуск OpenCode
- **Недопустимая структура таблицы**: понимание смысла ошибки "invalid structure", изучение правильного синтаксиса таблиц
- **Сбой форматирования**: обработка ошибок "table formatting failed" и аномальных ситуаций
- **Неподдерживаемые типы таблиц**: понимание ограничений для HTML-таблиц, многострочных ячеек и других сценариев
- **Невыровненное форматирование**: выявление скрытых режимов, сверхдлинных ячеек, встроенного кода и других проблем

## Рекомендации по обучению

Рекомендуется изучить этот раздел после ознакомления с [обзором функций](../start/features/) и обратиться к нему при возникновении реальных проблем или использовать как справочник по устранению неполадок.

Вы можете учиться в следующем порядке:

1. **При возникновении проблем**: сначала определите тип проблемы по симптомам (таблица не форматируется, сообщения об ошибках и т. д.)
2. **Постепенная диагностика**: последовательно проверяйте каждый пункт из контрольного списка руководства по устранению неполадок
3. **Глубокое понимание**: изучите [спецификацию таблиц](../advanced/table-spec/) и [принципы скрытого режима](../advanced/concealment-mode/) для понимания технических деталей
4. **Известные ограничения**: ознакомьтесь с [ограничениями](../appendix/limitations/) для понимания границ работы плагина
