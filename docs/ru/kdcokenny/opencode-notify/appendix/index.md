# Приложение: Справочник по типам событий и конфигурации

Этот раздел содержит справочную документацию и примеры конфигурации, которые помогут вам глубоко понять типы событий и параметры конфигурации плагина opencode-notify. Этот материал предназначен для справки и не требует последовательного изучения.

## Учебный путь

### 1. [Типы событий](./event-types/)

Узнайте о типах событий OpenCode, которые отслеживает плагин, и об условиях их срабатывания.

- Четыре типа событий (session.idle, session.error, permission.updated, tool.execute.before)
- Условия срабатывания и логика обработки для каждого типа событий
- Фильтрационные правила: проверка родительской сессии, проверка тихих часов, проверка фокуса терминала
- Различия в функциональности на разных платформах

### 2. [Пример файла конфигурации](./config-file-example/)

Ознакомьтесь с полным примером файла конфигурации и детальными комментариями ко всем полям.

- Полный шаблон файла конфигурации
- Описание полей notifyChildSessions, sounds, quietHours, terminal и других
- Полный список доступных звуков для macOS
- Пример минимальной конфигурации
- Способы отключения плагина
