# Приложение

Эта глава предоставляет технические справочные материалы OpenSkills, включая API командной строки, спецификации форматов файлов и определения терминов. Когда вам нужно проверить конкретные параметры команд, понять структуру сгенерированных файлов или выяснить значение определенного термина, здесь находится ваше справочное руководство.

## Содержание главы

| Документ | Описание |
|--- | ---|
| [Справочник по CLI API](./cli-api/) | Полная документация всех команд, опций и параметров с примерами использования |
| [Спецификация формата AGENTS.md](./agents-md-format/) | Структура файла AGENTS.md, генерируемого командой sync, включая XML-теги и формат списка навыков |
| [Структура файлов](./file-structure/) | Описание каталогов и файлов, создаваемых OpenSkills, поможет понять, что находится в `.openskills/` |
| [Глоссарий](./glossary/) | Объяснение ключевых терминов в OpenSkills и системе AI навыков |

## Рекомендации по обучению

Приложение - это справочный материал, его не нужно читать от начала до конца. Рекомендуемые способы использования:

1. **Возникли проблемы с командами** → Смотрите [Справочник по CLI API](./cli-api/)
2. **Хотите понять, как генерируется AGENTS.md** → Читайте [Спецификацию формата AGENTS.md](./agents-md-format/)
3. **Не знаете, куда поместить файлы** → Смотрите [Структуру файлов](./file-structure/)
4. **Видите незнакомые термины** → Смотрите [Глоссарий](./glossary/)

::: tip Совет для новичков
Если вы только начинаете использовать OpenSkills, рекомендуем сначала пройти учебник [Быстрый старт](/ru/numman-ali/openskills/start/quick-start/), а при возникновении проблем вернуться к этому приложению для справки.
:::
