# Журнал изменений

В этом разделе записана история версий OpenSkills, которая поможет вам узнать о новых функциях, улучшениях и исправлениях ошибок в каждой версии.

## Содержание главы

| Страница | Описание |
| --- | --- |
| [Журнал изменений](./changelog/) | Полная история изменений версий, включая новые функции, улучшения, исправления ошибок и обновления безопасности |

## Как использовать журнал изменений

Журнал изменений отсортирован по номерам версий в обратном порядке (новейшая версия в начале). Каждая версия содержит следующие категории:

- **Новые функции** — добавленные функции и команды
- **Улучшения** — оптимизация и расширение существующих функций
- **Исправления ошибок** — устранённые баги
- **Безопасность** — улучшения, связанные с безопасностью

::: tip Рекомендации по обновлению
Перед обновлением OpenSkills рекомендуется просмотреть журнал изменений, чтобы узнать об изменениях в версии, особенно обратите внимание на:
- **Breaking Changes** (критические изменения)
- Методы использования новых функций
- Состояние исправления известных проблем
:::
