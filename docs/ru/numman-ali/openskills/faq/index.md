# Часто задаваемые вопросы

В этой главе собраны часто задаваемые вопросы, решения проблем и объяснения концепций дизайна OpenSkills, которые помогут вам быстро находить и решать проблемы.

## Содержание главы

<div class="grid-cards">

### [Ответы на часто задаваемые вопросы](./faq/)

Ответы на распространённые вопросы по установке, настройке и использованию. Включая различия между OpenSkills и Claude Code, расположение установленных навыков, проблемы синхронизации AGENTS.md, совместное использование навыков в нескольких проектах и многое другое.

### [Устранение неполадок](./troubleshooting/)

Предоставляет методы диагностики и исправления распространённых ошибок. Охватывает подробные шаги по решению проблем: ошибки Git clone, отсутствие SKILL.md, навык не найден, ошибки прав доступа, пропуск обновлений и другие.

### [Почему CLI?](./why-cli-not-mcp/)

Объясняет причины выбора CLI вместо MCP в дизайне OpenSkills. Углублённое сравнение различий в позиционировании двух подходов, понимание того, почему система навыков лучше подходит для режима статических файлов.

</div>

```
    │                                           (диагностика ошибок, шаги исправления)
    │

                                                (выбор технологии, архитектурные решения)
```

**Рекомендуемый порядок**:

1. **Сначала прочтите FAQ** → Быстрое ознакомление с основными концепциями и советами по использованию
2. **При ошибке обратитесь к устранению неполадок** → Поиск решения по тексту ошибки
3. **Для углублённого понимания читайте о концепции дизайна** → Понимание технических аспектов CLI vs MCP
