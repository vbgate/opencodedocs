# Приложение

Этот раздел предоставляет справочные материалы и стандарты разработки для OpenCode Agent Skills, помогая вам писать качественные навыки и быстро находить справочные данные по API интерфейсам.

## Содержание главы

| Страница | Описание |
|--- | ---|
| [Лучшие практики разработки навыков](./best-practices/) | Освойте техники и стандарты написания качественных навыков, включая соглашения по именованию, написание описаний, организацию каталогов, использование скриптов и т.д. |
| [Справочник по API инструментам](./api-reference/) | Просмотрите подробные параметры и возвращаемые значения всех доступных инструментов, включая `get_available_skills`, `read_skill_file`, `run_skill_script`, `use_skill` |

## Рекомендации по учебному пути

::: tip Рекомендуемый порядок
1. **Сначала изучите лучшие практики** → Узнайте стандарты и техники разработки навыков, чтобы избежать распространённых ошибок
2. **Затем ознакомьтесь с API справочником** → При возникновении проблем в процессе разработки, просмотрите конкретные параметры и возвращаемые значения
:::

Эти два раздела имеют разное назначение:

- **Лучшие практики**: Подходит для предварительного ознакомления перед началом написания навыков, чтобы сформировать правильные привычки разработки
- **Справочник API**: Подходит как справочник для быстрого поиска конкретных деталей при необходимости
