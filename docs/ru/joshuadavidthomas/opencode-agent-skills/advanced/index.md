# Продвинутые функции

Эта глава подробно описывает продвинутые возможности OpenCode Agent Skills, включая совместимость с Claude Code, интеграцию рабочих процессов Superpowers, систему приоритетов пространств имён и механизмы восстановления после сжатия контекста. После освоения этого материала вы сможете лучше управлять сложной экосистемой навыков и обеспечивать их доступность в длительных сессиях.

## Предварительные требования

::: warning Перед началом убедитесь
Перед изучением этой главы убедитесь, что вы выполнили следующие задачи:

- [Установка OpenCode Agent Skills](../start/installation/) - Плагин правильно установлен и работает
- [Создание вашего первого навыка](../start/creating-your-first-skill/) - Понимание базовой структуры навыков
- [Детальный обзор механизма обнаружения навыков](../platforms/skill-discovery-mechanism/) - Понимание того, из каких мест обнаруживаются навыки
- [Загрузка навыков в контекст сессии](../platforms/loading-skills-into-context/) - Освоение использования инструмента `use_skill`
:::

## Путь обучения

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        Рекомендуемый порядок обучения                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1. Совместимость Claude Code  ──→  2. Интеграция Superpowers  ──→  3. Пространства имён │
│         │                              │                               │                   │
│         ▼                              ▼                               ▼                   │
│   Повторное использование навыков  Включение руководства по      Точный контроль   │
│   Claude                           рабочим процессам            источников навыков   │
│                                                                         │
│                                  │                                      │
│                                  ▼                                      │
│                                                                         │
│                    4. Восстановление после сжатия контекста             │
│                                  │                                      │
│                                  ▼                                      │
│                     Сохранение навыков в длительных сессиях              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Рекомендуется изучать в следующем порядке**:

1. **Сначала изучите совместимость с Claude Code** - Если у вас есть навыки Claude Code или вы хотите использовать навыки из магазина плагинов Claude, это первый шаг
2. **Затем изучите интеграцию Superpowers** - Для пользователей, которым нужны строгие рекомендации по рабочим процессам, понимание того, как включить и настроить
3. **Затем изучите пространства имён** - Когда количество навыков увеличивается и возникают конфликты с одноимёнными навыками, это ключевой момент
4. **Наконец, изучите восстановление после сжатия** - Понимание того, как навыки остаются доступными в длительных сессиях, концептуальное содержимое

::: tip Обучение по потребностям
- **Миграция из Claude Code**: Сосредоточьтесь на уроке 1 (совместимость) и уроке 3 (пространства имён)
- **Желание стандартизации рабочих процессов**: Сосредоточьтесь на уроке 2 (Superpowers)
- **Конфликты навыков**: Непосредственно перейдите к уроку 3 (пространства имён)
- **Потеря навыков в длительных сессиях**: Непосредственно перейдите к уроку 4 (восстановление после сжатия)
:::
