# Быстрый старт

Этот раздел поможет вам начать работу с плагином OpenCode Agent Skills с нуля. Вы узнаете о ключевых преимуществах плагина, выполните установку и настройку, а также создадите свой первый навык.

## Путь обучения

Рекомендуется изучать материалы в следующем порядке:

1. **[Что такое OpenCode Agent Skills?](./what-is-opencode-agent-skills/)** — Сначала узнайте, что может делать плагин, чтобы сформировать общее представление
2. **[Руководство по установке](./installation/)** — Установите плагин, чтобы OpenCode мог управлять навыками
3. **[Создайте свой первый навык](./creating-your-first-skill/)** — Приступайте к практике и создайте свой первый навык

::: tip Примерное время
На изучение этого раздела потребуется примерно 30-45 минут.
:::

## Следующий шаг

После завершения этого раздела вы можете продолжить изучение **[Функций платформы](../platforms/)**, чтобы глубже понять механизмы обнаружения навыков, загрузки навыков, автоматического сопоставления и другие расширенные функции.
