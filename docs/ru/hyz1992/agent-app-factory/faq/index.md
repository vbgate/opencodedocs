# Часто задаваемые вопросы и лучшие практики

Раздел FAQ содержит решения распространенных проблем и рекомендации по лучшим практикам при использовании AI App Factory, помогая вам быстро устранять неполадки и повышать эффективность разработки.

## Подстраничная навигация

| Урок | Описание |
| ---- | ---- |
| **[Часто задаваемые вопросы и устранение неполадок](../troubleshooting/)** | Решение распространенных проблем при инициализации, выполнении и развертывании |
| **[Лучшие практики](../best-practices/)** | Изучение навыков предоставления четких описаний продуктов, использования контрольных точек, контроля области действия, поэтапной итерации и других методов |

## Рекомендации по пути обучения

Содержимое раздела FAQ можно просматривать по необходимости, не обязательно изучать в строгом порядке:

1. **Консультация при возникновении проблем**: Если вы столкнулись с какими-либо проблемами в процессе использования, в первую очередь ознакомьтесь с «Часто задаваемые вопросы и устранение неполадок»
2. **Обязательно для повышения эффективности**: В разделе «Лучшие практики» обобщены уроки и выводы. Рекомендуется прочитать его после завершения первого проекта, эффективность разработки последующих проектов значительно повысится

## Предварительные условия

::: tip Рекомендуется сначала завершить

Перед чтением раздела FAQ рекомендуется сначала завершить следующее:

- ✅ **start**: Завершите инициализацию проекта и запустите полный конвейер
- ✅ **platforms**: Настройте хотя бы одного AI-помощника (Claude Code или OpenCode)

:::
