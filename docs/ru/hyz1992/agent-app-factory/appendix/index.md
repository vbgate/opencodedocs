# Приложение

Этот раздел содержит справочные документы и дополнительные материалы, которые помогут вам глубже понять технические детали AI App Factory.

## Навигация по подразделам

### [Справочник по CLI-командам](../cli-commands/)
Полный список CLI-команд с описанием параметров и примерами использования, включая команды `init`, `run`, `continue`, `status`, `list` и `reset`.

### [Подробная структура каталогов](../directory-structure/)
Узнайте полную структуру каталогов проекта Factory и назначение каждого файла, включая назначение основных каталогов `agents/`, `skills/`, `cli/` и других.

### [Стандарты кода](../code-standards/)
Узнайте о стандартах кодирования TypeScript, структуре файлов, требованиях к комментариям и стандартах для сообщений коммитов, которым должен соответствовать генерируемый код.

### [Подробное описание технического стека](../tech-stack/)
Узнайте о техническом стеке, используемом в генерируемых приложениях: Node.js + Express + Prisma + React Native, а также о причинах выбора технологий для каждой части.
