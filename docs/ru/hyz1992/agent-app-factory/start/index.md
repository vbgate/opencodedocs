# Начало работы

Этот раздел поможет вам быстро освоить AI App Factory: узнать о ключевых преимуществах, установить и настроить инструменты, а также понять принцип работы 7-этапного конвейера, чтобы подготовиться к созданию первого приложения.

## Путь обучения

Рекомендуется изучать материалы этого раздела в следующем порядке:

### 1. [Быстрый старт: от идеи до приложения](../getting-started/)
Узнайте о ключевых преимуществах AI App Factory и предварительных требованиях, а также пройдите полный цикл от идеи до готового приложения на практическом примере.
- Поймите механизм многопользовательского взаимодействия агентов на основе контрольных точек
- Изучите полный рабочий процесс 7-этапного конвейера
- Освойте технику составления четкого описания продукта

### 2. [Установка и настройка](../installation/)
Установите CLI-инструмент глобально и настройте AI-ассистента (Claude Code/OpenCode) и связанное окружение.
- Установите agent-app-factory глобально через npm
- Настройте Claude Code или OpenCode
- Проверьте успешность установки

### 3. [Инициализация проекта Factory](../init-project/)
Инициализируйте директорию с помощью команды `factory init`, изучите структуру создаваемого проекта и файлы конфигурации.
- Выполните команду инициализации проекта
- Поймите структуру директорий и файлы конфигурации
- Подготовьте описание идеи продукта

### 4. [Обзор 7-этапного конвейера](../pipeline-overview/)
Углубленно изучите полный процесс конвейера — от Bootstrap до Preview, включая входные и выходные данные каждого этапа и механизм контрольных точек.
- Поймите цели и результаты 7 этапов
- Освойте механизм контрольных точек и процесс ручного подтверждения
- Научитесь управлять выполнением конвейера

## Предварительные требования

Перед началом изучения этого раздела вам понадобится:

::: warning Предварительные требования

1. **Среда Node.js**: установлен Node.js v18 или выше
2. **AI-ассистент для программирования**: необходимо настроить один из следующих инструментов
   - [Claude Code](https://claude.ai/code) (рекомендуется) ⭐
   - [OpenCode](https://opencode.design/) или другой AI-ассистент с поддержкой режима Agent
3. **Базовые навыки работы с командной строкой**: умение выполнять команды в терминале

:::
