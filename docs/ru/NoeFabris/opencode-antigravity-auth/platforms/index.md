# Функции платформы

Эта глава поможет вам глубже понять модели, систему квот и функции платформы, поддерживаемые плагином Antigravity Auth. Вы научитесь выбирать подходящую модель, настраивать возможности Thinking, включать Google Search и максимизировать использование квот.

## Предварительные условия

::: warning Перед началом убедитесь
Перед изучением этой главы убедитесь, что вы завершили:
- [Быстрая установка](../start/quick-install/) — установка плагина и первая аутентификация
- [Первый запрос](../start/first-request/) — успешная отправка первого запроса к модели
:::

## Путь обучения

Изучайте материалы в следующем порядке —循序渐进掌握平台功能：

### 1. [Доступные модели](./available-models/)

Узнайте обо всех доступных моделях и их конфигурациях вариантов

- Ознакомьтесь с Claude Opus 4.5, Sonnet 4.5 и Gemini 3 Pro/Flash
- Поймите распределение моделей между двумя пулами квот Antigravity и Gemini CLI
- Освойте использование параметра `--variant`

### 2. [Двойная система квот](./dual-quota-system/)

Поймите принцип работы двойной системы квот Antigravity и Gemini CLI

- Узнайте, как каждая учётная запись обладает двумя независимыми пулами квот Gemini
- Включите автоматическую конфигурацию fallback для удвоения квот
- Явно укажите использование определённого пула квот для конкретной модели

### 3. [Google Search Grounding](./google-search-grounding/)

Включите Google Search для моделей Gemini, чтобы повысить точность фактов

- Позвольте Gemini искать актуальную информацию в сети
- Настройте порог поиска для управления частотой поисков
- Выберите подходящую конфигурацию в зависимости от требований задачи

### 4. [Thinking модели](./thinking-models/)

Освойте конфигурацию и использование Thinking моделей Claude и Gemini 3

- Настройте thinking budget для Claude
- Используйте thinking level Gemini 3 (minimal/low/medium/high)
- Поймите interleaved thinking и стратегию сохранения блоков размышлений
