# Приложение

В этом разделе представлены технические справочные материалы плагина Antigravity Auth, включая проектирование архитектуры, спецификацию API, формат хранения и полное руководство по конфигурации, которые помогут вам глубоко понять внутренние механизмы плагина.

## Путь обучения

### 1. [Обзор архитектуры](./architecture-overview/)

Познакомьтесь с модульной структурой плагина и процессом обработки запросов.

- Слоистая модульная конструкция и разделение ответственности
- Полный путь запроса от OpenCode до Antigravity API
- Балансировка нагрузки между несколькими аккаунтами и механизм восстановления сессий

### 2. [Спецификация API](./api-spec/)

Углубитесь в технические детали Antigravity API.

- Единый интерфейс шлюза и конфигурация конечных точек
- Форматы запросов/ответов и ограничения JSON Schema
- Конфигурация Thinking-моделей и правила вызова функций

### 3. [Формат хранения](./storage-schema/)

Узнайте о структуре файлов хранения аккаунтов и управлении версиями.

- Расположение файлов хранения и значение каждого поля
- Эволюция версий v1/v2/v3 и автоматическая миграция
- Методы миграции конфигурации аккаунтов между машинами

### 4. [Все опции конфигурации](./all-config-options/)

Полное справочное руководство по всем опциям конфигурации.

- Значения по умолчанию и сценарии применения для 30+ опций конфигурации
- Методы переопределения конфигурации через переменные окружения
- Лучшие комбинации конфигураций для разных сценариев использования
