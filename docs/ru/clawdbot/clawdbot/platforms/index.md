# Мультиканальные платформы

Clawdbot через единую управляющую плоскость Gateway поддерживает различные каналы связи и платформы, позволяя вам взаимодействовать с AI-помощником на знакомом интерфейсе.

## Обзор главы

Эта глава представляет все каналы связи и платформы, поддерживаемые Clawdbot, включая мессенджеры (WhatsApp, Telegram, Slack, Discord и др.), мобильные узлы (iOS, Android) и десктопные приложения (macOS). Узнайте, как настроить эти каналы, чтобы AI-помощник бесшовно интегрировался в ваш ежедневный рабочий процесс.

## Навигация по подпунктам

### Обзор каналов

- **[Обзор мультиканальной системы](channels-overview/)** - Узнайте о всех каналах связи, поддерживаемых Clawdbot, и их особенностях, освойте базовые концепции настройки каналов.

### Мессенджеры

- **[WhatsApp](whatsapp/)** - Настройка и использование канала WhatsApp (на базе Baileys), поддерживает подключение устройств и управление группами.
- **[Telegram](telegram/)** - Настройка и использование канала Telegram (на базе grammY Bot API), настройка Bot Token и Webhook.
- **[Slack](slack/)** - Настройка и использование канала Slack (на базе Bolt), интеграция в ваше рабочее пространство.
- **[Discord](discord/)** - Настройка и использование канала Discord (на базе discord.js), поддерживает серверы и каналы.
- **[Google Chat](googlechat/)** - Настройка и использование канала Google Chat, интеграция с Google Workspace.
- **[Signal](signal/)** - Настройка и использование канала Signal (на базе signal-cli), защищенная коммуникация с конфиденциальностью.
- **[iMessage](imessage/)** - Настройка и использование канала iMessage (эксклюзивно для macOS), интеграция с приложением Сообщения на macOS.
- **[LINE](line/)** - Настройка и использование канала LINE (Messaging API), взаимодействие с пользователями LINE.

### Веб и нативные приложения

- **[Интерфейс WebChat](webchat/)** - Используйте встроенный интерфейс WebChat для взаимодействия с AI-помощником, без необходимости настройки внешних каналов.
- **[Приложение macOS](macos-app/)** - Узнайте о функциях приложения macOS в строке меню, включая Voice Wake, Talk Mode и удаленное управление.
- **[Узел iOS](ios-node/)** - Настройте узел iOS для выполнения локальных операций на устройстве (Camera, Canvas, Voice Wake).
- **[Узел Android](android-node/)** - Настройте узел Android для выполнения локальных операций на устройстве (Camera, Canvas).

## Рекомендуемый порядок обучения

В зависимости от вашего сценария использования, рекомендуется следующий порядок обучения:

### Быстрое начало для новичков

Если вы впервые используете Clawdbot, рекомендуется следующий порядок обучения:

1. **[Обзор мультиканальной системы](channels-overview/)** - Сначала ознакомьтесь с общей архитектурой и концепцией каналов
2. **[Интерфейс WebChat](webchat/)** - Самый простой способ начать использование без настройки
3. **Выберите популярный канал** - Выберите в соответствии с вашими ежедневными привычками:
   - Ежедневные переписки → [WhatsApp](whatsapp/) или [Telegram](telegram/)
   - Командная работа → [Slack](slack/) или [Discord](discord/)
   - Пользователи macOS → [iMessage](imessage/)

### Интеграция мобильных устройств

Если вы хотите использовать Clawdbot на мобильных устройствах:

1. **[Узел iOS](ios-node/)** - Настройте локальные функции на iPhone/iPad
2. **[Узел Android](android-node/)** - Настройте локальные функции на устройствах Android
3. **[Приложение macOS](macos-app/)** - Используйте приложение macOS как центр управления

### Корпоративное развертывание

Если вам нужно развернуть в командной среде:

1. **[Slack](slack/)** - Интеграция с рабочим пространством команды
2. **[Discord](discord/)** - Создание сервера сообщества
3. **[Google Chat](googlechat/)** - Интеграция с Google Workspace

## Часто задаваемые вопросы

::: details Могу ли я использовать несколько каналов одновременно?
Да! Clawdbot поддерживает одновременное включение нескольких каналов. Вы можете принимать и отправлять сообщения на разных каналах, все сообщения обрабатываются через единый Gateway.
:::

::: details Какой канал наиболее рекомендуется?
Это зависит от вашего сценария использования:
- **WebChat** - Самый простой, не требует настройки
- **WhatsApp** - Подходит для общения с друзьями и семьей
- **Telegram** - Стабильный Bot API, подходит для автоматических ответов
- **Slack/Discord** - Подходит для командной работы
:::

::: details Требуется ли оплата за настройку каналов?
Большинство каналов бесплатны, но некоторые могут иметь стоимость:
- WhatsApp Business API - могут возникнуть расходы
- Google Chat - требуется аккаунт Google Workspace
- Другие каналы - обычно бесплатны, достаточно получить Bot Token
:::
