# Дополнительные функции

Этот раздел описывает дополнительные параметры конфигурации opencode-mystatus, подходит для пользователей, нуждающихся в большей настройке.

## Список функций

### [Конфигурация Google Cloud](./google-setup/)

Настройка и управление несколькими аккаунтами Google Cloud Antigravity:

- Добавление нескольких аккаунтов Google Cloud
- Соответствие 4 моделей (G3 Pro, G3 Image, G3 Flash, Claude)
- Различие projectId и managedProjectId
- Решение проблемы недостатка квот одного аккаунта

### [Конфигурация аутентификации Copilot](./copilot-auth/)

Решение проблем аутентификации GitHub Copilot:

- Различие между OAuth Token и Fine-grained PAT
- Решение проблемы недостатка прав OAuth Token
- Создание Fine-grained PAT и настройка типа подписки
- Конфигурация файла copilot-quota-token.json

### [Поддержка нескольких языков](./i18n-setup/)

Понимание механизма автоматического определения языка:

- Принцип автоматического определения языка системы
- Механизм возврата к Intl API и переменным окружения
- Как переключать язык вывода (китайский/английский)
