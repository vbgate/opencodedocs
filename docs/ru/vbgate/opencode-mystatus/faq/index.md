# Частые вопросы

Этот раздел собрал частые проблемы и решения при использовании opencode-mystatus.

## Категории проблем

### [Поиск и устранение неисправностей](./troubleshooting/)

Решите различные проблемы при запросе:

- Невозможно прочитать файл аутентификации
- Token истёк или недостаточно прав
- Сбой запроса API или тайм-аут
- Обработка специфических ошибок разных платформ

### [Безопасность и конфиденциальность](./security/)

Узнайте о механизмах безопасности плагина:

- Только чтение локальных файлов
- Автоматическая маскировка API Key
- Вызов только официальных интерфейсов
- Нет загрузки или хранения данных

## Быстрое поиск

Согласно сообщению об ошибке быстро найдите решение:

| Ключевые слова ошибки | Возможная причина | Решение |
|-----------|---------|---------|
| `auth.json not found` | Файл аутентификации не существует | [Поиск и устранение](./troubleshooting/) |
| `Token expired` | Token истёк | [Поиск и устранение](./troubleshooting/) |
| `Permission denied` | Недостаточно прав | [Конфигурация аутентификации Copilot](../advanced/copilot-auth/) |
| `project_id missing` | Незавершённая конфигурация Google Cloud | [Конфигурация Google Cloud](../advanced/google-setup/) |
| `Request timeout` | Проблемы с сетью | [Поиск и устранение](./troubleshooting/) |
