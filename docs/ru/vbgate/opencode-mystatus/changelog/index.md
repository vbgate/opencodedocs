# История версий

Этот документ перечисляет все версии и основные изменения opencode-mystatus.

## Последняя версия

### [v1.2.0 - v1.2.4](./v120-v124/)

- Поддержка Google Cloud
- Управление несколькими аккаунтами
- Отображение квот 4 моделей (G3 Pro, G3 Image, G3 Flash, Claude)
- Исправлены несколько проблем аутентификации платформ

## Исторические версии

### [v1.0.0 - v1.0.1](./v100-v101/)

- Начальный выпуск
- Запрос квот OpenAI
- Запрос квот Zhipu AI / Z.ai
- Запрос квот GitHub Copilot
- Базовый индикатор прогресса
- Поддержка нескольких языков (китайский/английский)
