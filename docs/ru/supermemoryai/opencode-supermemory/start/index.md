# Быстрый старт

Эта глава поможет вам установить и настроить плагин opencode-supermemory с нуля, чтобы ваш OpenCode Agent получил возможность постоянного запоминания. После завершения этой главы Agent сможет запоминать архитектуру проекта и ваши настройки предпочтений.

## Путь обучения

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   1. Быстрый старт          2. Инициализация проекта             │
│   ─────────────   →   ─────────────                             │

│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**Рекомендуемый порядок обучения**:

1. **[Быстрый старт](./getting-started/)**: Сначала выполните установку плагина и настройку API Key — это предпосылка для использования всех функций.
2. **[Инициализация проекта](./initialization/)**: После установки выполните команду инициализации, чтобы Agent ознакомился с вашим проектом.

## Предварительные условия

Перед началом изучения этой главы убедитесь, что:

- ✅ Установлен [OpenCode](https://opencode.ai), и команда `opencode` доступна в терминале
- ✅ Зарегистрирован аккаунт [Supermemory](https://console.supermemory.ai) и получен API Key
