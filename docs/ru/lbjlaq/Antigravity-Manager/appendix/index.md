# Приложение

В этом разделе собраны технические справочные материалы Antigravity Tools, включая шпаргалку по API-эндпоинтам, структуру хранения данных и описание границ экспериментальной функциональности. Когда вам нужно быстро найти конкретную техническую деталь, здесь находится ваш «словарь».

## Содержание главы

| Документ | Описание | Подходящие сценарии |
|--- | --- | ---|
| **[Шпаргалка по эндпоинтам](./endpoints/)** | Обзор внешних HTTP-маршрутов: эндпоинты OpenAI/Anthropic/Gemini/MCP, режимы аутентификации и форматы заголовков | Подключение нового клиента, устранение неполадок 404/401 |
| **[Данные и модели](./storage-models/)** | Структура файлов аккаунтов, структура таблиц базы статистики SQLite, спецификация ключевых полей | Резервное копирование/миграция, прямой запрос к базе, устранение неполадок |
| **[Границы интеграции z.ai](./zai-boundaries/)** | Список реализованных и явно нереализованных функций z.ai | Оценка возможностей z.ai, избегание неправильного использования |

## Рекомендуемый путь обучения

```
Шпаргалка по эндпоинтам → Данные и модели → Границы интеграции z.ai
          ↓                     ↓                       ↓
   Знать какой путь вызвать   Знать где данные      Знать где границы
```

1. **Сначала посмотрите шпаргалку по эндпоинтам**: узнайте, какие API можно вызывать, и как настроить аутентификацию
2. **Затем посмотрите «Данные и модели»**: узнайте структуру хранения данных, что удобно для резервного копирования, миграции и прямого запроса к базе для устранения неполадок
3. **Наконец посмотрите «Границы интеграции z.ai»**: если вы используете z.ai, эта статья поможет вам не принять «нереализованное» за «доступное»

::: tip Эти документы не являются «обязательными к чтению»
Приложение — это справочные материалы, а не учебник. Вам не нужно читать его от начала до конца, просто обращайтесь к нему при возникновении конкретных вопросов.
:::
