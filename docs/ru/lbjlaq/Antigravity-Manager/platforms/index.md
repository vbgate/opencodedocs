# Платформы и интеграция

Основная возможность Antigravity Tools — преобразовывать протоколы нескольких платформ AI в унифицированную локальную API-шлюз. В этой главе подробно описываются способы подключения, границы совместимости и лучшие практики каждого протокола.

## Содержание главы

| Урок | Описание |
|-----|------|
| [OpenAI-совместимый API](./openai/) | Стратегия реализации `/v1/chat/completions` и `/v1/responses`, безболезненный доступ OpenAI SDK |
| [Anthropic-совместимый API](./anthropic/) | Ключевой контракт `/v1/messages` с Claude Code, поддержка цепочки размышлений, системных подсказок и других основных возможностей |
| [Gemini-нативный API](./gemini/) | Конечные точки `/v1beta/models` и доступ к конечным точкам Google SDK, поддержка совместимости `x-goog-api-key` |
| [Imagen 3 генерация изображений](./imagen/) | Автоматическое отображение параметров `size`/`quality` OpenAI Images, поддержка произвольных соотношений сторон |
| [Транскрибация аудио](./audio/) | Ограничения `/v1/audio/transcriptions` и обработка больших пакетов |
| [Конечные точки MCP](./mcp/) | Предоставление Web Search/Reader/Vision в качестве вызываемых инструментов |
| [Туннель Cloudflared](./cloudflared/) | Одним кликом безопасно открывать локальный API в публичную сеть (не является безопасным по умолчанию) |

## Рекомендуемый путь обучения

::: tip Рекомендуемый порядок
Сначала изучайте протокол, который используете:
:::

**Быстрый выбор**:

| Ваш сценарий | Рекомендуется начать с |
|---------|---------|
| Используете Claude Code CLI | [Anthropic-совместимый API](./anthropic/) |
| Используете OpenAI Python SDK | [OpenAI-совместимый API](./openai/) |
| Используете официальный Google SDK | [Gemini-нативный API](./gemini/) |
| Нужен ИИ-рисование | [Imagen 3 генерация изображений](./imagen/) |
| Нужна распознавание речи в текст | [Транскрибация аудио](./audio/) |
| Нужен поиск в сети / чтение веб-страниц | [Конечные точки MCP](./mcp/) |
| Нужен удаленный доступ | [Туннель Cloudflared](./cloudflared/) |
