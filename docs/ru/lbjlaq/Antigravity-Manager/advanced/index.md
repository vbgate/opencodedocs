# Расширенная конфигурация

В этой главе подробно рассматриваются расширенные функции Antigravity Tools: управление конфигурацией, политики безопасности, планирование аккаунтов, маршрутизация моделей, управление квотами, мониторинг и статистика, а также варианты развертывания на сервере. После изучения этого содержания вы сможете улучшить Antigravity Tools от состояния «работает» до состояния «удобный, стабильный и пригодный для эксплуатации».

## Рекомендации по учебному пути

::: tip Рекомендуемый порядок
В этой главе много материала, рекомендуется изучать по модулям порциями:
:::

**Первый этап: конфигурация и безопасность (обязательно)**

```
Полное описание конфигурации → Безопасность и приватность
config                     security
```

Сначала ознакомьтесь с системой конфигурации (какие настройки требуют перезапуска, какие обновляются в горячем режиме), затем изучите настройки безопасности (особенно при открытии доступа к локальной сети/общественной сети).

**Второй этап: планирование и маршрутизация (рекомендуется)**

```
Высокодоступное планирование → Маршрутизация моделей
scheduling                  model-router
```

Научитесь использовать минимальное количество аккаунтов для максимальной стабильности, затем используйте маршрутизацию моделей для экранирования изменений от вышестоящих поставщиков.

**Третий этап: квоты и мониторинг (по необходимости)**

```
Управление квотами → Proxy Monitor → Token Stats
quota               monitoring      token-stats
```

Предотвратите незаметное исчерпание квот, превратите черный ящик вызовов в наблюдаемую систему, оптимизируйте количественные затраты.

**Четвертый этап: стабильность и развертывание (расширенный уровень)**

```
Стабильность длинных сессий → Системные возможности → Развертывание на сервере
context-compression         system            deployment
```

Решите скрытые проблемы длинных сессий, сделайте клиент похожим на продукт, затем изучите развертывание на сервере.

**Быстрый выбор**:

| Ваш сценарий | Рекомендуется сначала прочитать |
|---------|---------|
| Нестабильная ротация нескольких аккаунтов | [Высокодоступное планирование](./scheduling/) |
| Хотите зафиксировать определенное имя модели | [Маршрутизация моделей](./model-router/) |
| Квоты всегда заканчиваются | [Управление квотами](./quota/) |
| Хотите просмотреть журналы запросов | [Proxy Monitor](./monitoring/) |
| Хотите статистику потребления токенов | [Token Stats](./token-stats/) |
| В длинных диалогах часто возникают ошибки | [Стабильность длинных сессий](./context-compression/) |
| Нужно открыть доступ к локальной сети | [Безопасность и приватность](./security/) |
| Нужно развернуть на сервере | [Развертывание на сервере](./deployment/) |

## Предварительные требования

:::
