# О OpenCodeDocs

> **Первый учебный сайт экосистемы ИИ · Управляемый исходным кодом · 100% работоспособный**

## Наше видение

Инструменты ИИ развиваются невероятно быстро, официальная документация отстает, а онлайн-руководства устаревают — мы хотим это изменить.

OpenCodeDocs глубоко анализирует исходный код и официальные примеры проектов с открытым исходным кодом, выделяет основную логику и гарантирует работоспособность каждой строки кода с помощью автоматической проверки.

При обновлении проектов наши руководства синхронизируются. Каждое руководство четко указывает версию Git Commit, чтобы вы знали, на основе какой версии был создан код.

**Наша цель — стать «Verified Standard» в мире инструментов ИИ.**

## Лицензия с открытым исходным кодом

Мы верим, что открытость порождает доверие. Приветствуем вклад сообщества, исправление ошибок и дополнение материалов.
