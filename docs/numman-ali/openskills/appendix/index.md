# Appendix

This section provides technical reference materials for OpenSkills, including command-line API, file format specifications, and terminology definitions. When you need to look up specific command parameters, understand the structure of generated files, or clarify the meaning of a term, this is your quick reference.

## Recommended Usage

The appendix is a reference section and doesn't need to be read from start to finish. Recommended usage:

1. **Encounter command issues** → Check [CLI API Reference](./cli-api/)
2. **Want to understand how AGENTS.md is generated** → Read [AGENTS.md Format Specification](./agents-md-format/)
3. **Unsure where files are located** → Browse [File Structure](./file-structure/)
4. **See unfamiliar terms** → Look up [Glossary](./glossary/)

::: tip Beginner Recommendation
If you're new to OpenSkills, it's recommended to complete the [Quick Start](../start/quick-start/) tutorial first, then come back to consult the appendix when you encounter issues.
:::
