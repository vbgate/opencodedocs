## How to Use the Changelog

The changelog is sorted in reverse order by version number (latest version first), and each version includes the following categories:

- **New Features** - Newly added features and commands
- **Improvements** - Optimizations and enhancements to existing features
- **Bug Fixes** - Fixed bugs
- **Security** - Security-related improvements

::: tip Upgrade Recommendations
Before upgrading OpenSkills, it's recommended to check the changelog first to understand version changes, especially:
- **Breaking Changes**
- Usage methods for new features
- Fix status of known issues
:::
