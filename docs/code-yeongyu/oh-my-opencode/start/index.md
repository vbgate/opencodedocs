# Quick Start

This section helps you quickly install and start using oh-my-opencode, from basic configuration to the main orchestrator, and then to the Ultrawork mode that activates all features with one click.

## Learning Path

Follow this sequence to progressively master the basic functions of oh-my-opencode:

### 1. [Quick Installation and Configuration](./installation/)

Complete the installation of oh-my-opencode, Provider configuration, and basic verification.
- Install oh-my-opencode
- Configure AI Provider (Claude, OpenAI, Gemini, etc.)
- Verify configuration is correct
- Understand configuration file structure

### 2. [Meet Sisyphus: The Main Orchestrator](./sisyphus-orchestrator/)

Understand the core concepts of agent orchestration and learn the basic usage of the Sisyphus agent.
- What is an agent orchestration system
- Core functions of the Sisyphus agent
- Basic usage and best practices
- Understand the workflow of multi-model collaboration

### 3. [Ultrawork Mode: Activate All Features with One Click](./ultrawork-mode/)

Learn how to use the ultrawork keyword to quickly activate all professional agents and parallel task features.
- What is Ultrawork mode
- How to trigger the ultrawork keyword
- Collaborative work of 10 professional agents
- Management of parallel background tasks

## Prerequisites

Before starting this section, make sure:
- Node.js 18+ is installed (version 20 or higher is recommended)
- Familiar with basic command-line operations
- Have at least one AI service API key (Claude, OpenAI, or Gemini)

::: tip No API key?
If you don't have an AI service API key yet, you can start by applying for Claude, which is currently the most recommended choice.
:::
