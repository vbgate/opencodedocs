## Quick Help

### Having Issues?

1. **Run diagnostics first**
   ```bash
   openclaw doctor
   ```

2. **Check status**
   ```bash
   openclaw status
   ```

3. **View logs**
   ```bash
   openclaw logs --follow
   ```

### Common Commands Cheat Sheet

| Command | Purpose |
|---------|---------|
| `openclaw doctor` | Run system diagnostics |
| `openclaw status` | View system status |
| `openclaw logs` | View logs |
| `openclaw config validate` | Validate configuration |
| `openclaw --help` | Show help |
