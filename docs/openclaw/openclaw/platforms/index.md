# Platforms & Channels

This section covers how to integrate OpenClaw with various messaging platforms, enabling your AI assistant to respond across multiple channels.

## Supported Channels

OpenClaw supports a wide range of messaging channels:

### Core Channels (Built-in)

- **WhatsApp** - via Baileys
- **Telegram** - Bot API integration
- **Discord** - discord.js bot
- **Slack** - Bolt framework
- **Signal** - libsignal protocol
- **iMessage** - native support

### Extended Channels

- **BlueBubbles** - iMessage bridge
- **MS Teams** - Microsoft Graph API
- **Matrix** - decentralized chat
- **Zalo** - Vietnam's mainstream platform
- **IRC** - classic chat protocol

## Channel Selection Guide

Choose the right channel based on your use case:

| Use Case | Recommended Channels |
| --- | --- |
| Personal Assistant | WhatsApp, Telegram |
| Team Collaboration | Slack, Discord |
| Privacy First | Signal |
| Enterprise | MS Teams, Slack |
| Apple Ecosystem | BlueBubbles |

## Configuration Overview

The configuration process for each channel is generally similar:

1. Create Bot/App on the platform
2. Obtain Token/Credentials
3. Configure in OpenClaw
4. Test connection

For detailed steps, refer to each channel's dedicated tutorial.
