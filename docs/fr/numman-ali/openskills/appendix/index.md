# Annexe

Ce chapitre fournit les références techniques d'OpenSkills, y compris l'API de ligne de commande, les spécifications de format de fichier et les définitions des termes. Lorsque vous devez consulter des paramètres de commande spécifiques, comprendre la structure des fichiers générés, ou clarifier la signification d'un terme, ceci est votre manuel de référence rapide.

## Contenu du Chapitre

| Document | Description |
|--- | ---|
| [Référence de l'API CLI](./cli-api/) | Documentation complète de toutes les commandes, options et paramètres, avec exemples d'utilisation |
| [Spécification du Format AGENTS.md](./agents-md-format/) | Structure du fichier AGENTS.md généré par la commande sync, incluant les balises XML et le format de la liste des compétences |
| [Structure des Fichiers](./file-structure/) | Explication des répertoires et fichiers générés par OpenSkills, vous aidant à comprendre ce qui se trouve dans `.openskills/` |
| [Glossaire](./glossary/) | Explication des termes clés dans OpenSkills et le système de compétences IA |

## Recommandations de Parcours d'Apprentissage

L'annexe est une référence, il n'est pas nécessaire de la lire du début à la fin. Mode d'utilisation recommandé :

1. **Problème de commande** → Consultez [Référence de l'API CLI](./cli-api/)
2. **Comprendre comment AGENTS.md est généré** → Voir [Spécification du Format AGENTS.md](./agents-md-format/)
3. **Incertitude sur l'emplacement des fichiers** → Parcourir [Structure des Fichiers](./file-structure/)
4. **Terme inconnu** → Consulter [Glossaire](./glossary/)

::: tip Conseil pour les Débutants
Si vous venez de commencer à utiliser OpenSkills, il est recommandé de compléter d'abord le tutoriel [Démarrage Rapide](../start/quick-start/) et de revenir consulter l'annexe en cas de problème.
:::
