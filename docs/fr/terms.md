## 1. Description du service

OpenCodeDocs est une plateforme ouverte dédiée aux tutoriels d'outils IA, offrant les services suivants :

- **Contenu des tutoriels** : Tutoriels d'utilisation pour les outils IA open source (tels que Claude Code, Cursor, LangChain, etc.)
- **Vérification du code** : Vérification automatique de l'exécutabilité du code dans les tutoriels basée sur les dépôts de code source
- **Fonctionnalités communautaires** : Les utilisateurs peuvent participer aux contributions et discussions via GitHub

Nous nous réservons le droit de modifier, suspendre ou mettre fin à tout ou partie des services sans préavis.
