# À propos d'OpenCodeDocs

> **Premier site de tutoriels de l'écosystème IA · Basé sur le code source · 100% exécutable**

## Notre vision

Les outils IA évoluent rapidement, la documentation officielle prend du retard et les tutoriels en ligne sont obsolètes — nous voulons changer tout ça.

OpenCodeDocs analyse en profondeur le code source des projets open source et les exemples officiels, extrait la logique principale, et garantit que chaque ligne de code fonctionne grâce à une vérification automatisée.

Lorsque les projets sont mis à jour, nos tutoriels sont également synchronisés. Chaque tutoriel indique clairement la version Git Commit, pour que vous sachiez sur quelle version le code est basé.

**Notre objectif est de devenir la « Verified Standard » de l'écosystème des outils IA.**
