# Annexe

Cette section fournit les références techniques de Plannotator, y compris la documentation API, les définitions des modèles de données et les informations de licence. Ces contenus s'adressent principalement aux utilisateurs ayant besoin de comprendre en profondeur les mécanismes internes du système ou d'effectuer un développement secondaire.

## Contenu de cette section

| Page | Description |
| --- | --- |
| [Référence API](./api-reference/) | Consultez tous les points d'accès API fournis par Plannotator et les formats de requête/réponse |
| [Modèles de données](./data-models/) | Comprendre les modèles de données utilisés par Plannotator, y compris Annotation, Block, CodeAnnotation, etc. |
| [Licence](./license/) | Découvrir la licence de Plannotator (BSL-1.1) et les restrictions d'utilisation |

## Conseils de parcours d'apprentissage

Le contenu de l'annexe est relativement indépendant, vous pouvez le consulter sélectivement selon vos besoins :

1. **Souhaitez-vous comprendre les interfaces API ?** → Lisez directement [Référence API](./api-reference/)
2. **Souhaitez-vous comprendre les structures de données ?** → Lisez [Modèles de données](./data-models/)
3. **Souhaitez-vous comprendre les restrictions d'utilisation ?** → Lisez [Licence](./license/)

::: tip Astuce
Le contenu de l'annexe est principalement destiné à la consultation et n'a pas besoin d'être appris dans l'ordre. Il est recommandé de consulter les sections pertinentes lorsque vous rencontrez des problèmes spécifiques.
:::
