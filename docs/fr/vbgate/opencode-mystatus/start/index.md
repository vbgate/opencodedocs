# Démarrage rapide

Ce chapitre vous aide à démarrer rapidement avec le plugin opencode-mystatus, de l'installation à l'utilisation en 5 minutes.

## Parcours d'apprentissage

Nous vous recommandons de suivre l'ordre suivant :

### 1. [Démarrage rapide](./quick-start/)

Complétez l'installation et la configuration du plugin en 5 minutes, et consultez le quota de votre première plateforme IA.

- Trois méthodes d'installation : laissez l'IA vous installer, installation manuelle, installation depuis un fichier local
- Configuration de la commande slash `/mystatus`
- Vérification de l'installation réussie

### 2. [Utiliser mystatus](./using-mystatus/)

Apprenez les deux méthodes d'utilisation : la commande slash et le langage naturel.

- Utilisation de la commande slash `/mystatus`
- Déclenchement de l'outil mystatus par une question en langage naturel
- Différences entre les deux méthodes et scénarios d'utilisation

### 3. [Comprendre la sortie](./understanding-output/)

Comprenez les barres de progression, les temps de réinitialisation et le format de sortie multi-comptes.

- Signification des barres de progression : blocs pleins vs blocs vides
- Cycles de limite de différentes plateformes (3 heures, 5 heures, mensuel)
- Format de sortie dans les scénarios multi-comptes

## Conditions préalables

Avant de commencer, assurez-vous de :

- Avoir installé [OpenCode](https://opencode.ai)
- Avoir configuré les informations d'authentification d'au moins une plateforme IA (OpenAI, Zhipu AI, Z.ai, GitHub Copilot ou Google Cloud)

## Étapes suivantes

Une fois ce chapitre terminé, vous pouvez continuer avec [Fonctionnalités de plateforme](../platforms/) pour approfondir les détails de consultation des quotas de chaque plateforme.
