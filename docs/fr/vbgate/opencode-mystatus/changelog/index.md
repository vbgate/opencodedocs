# Historique des versions

Ce document répertorie toutes les versions d'opencode-mystatus et leurs modifications principales.

## Dernière version

### [v1.2.0 - v1.2.4](./v120-v124/)

- Support Google Cloud
- Gestion multi-comptes
- Affichage des quotas de 4 modèles (G3 Pro, G3 Image, G3 Flash, Claude)
- Correction de plusieurs problèmes d'authentification de plateforme

## Versions antérieures

### [v1.0.0 - v1.0.1](./v100-v101/)

- Publication initiale
- Consultation des quotas OpenAI
- Consultation des quotas Zhipu AI / Z.ai
- Consultation des quotas GitHub Copilot
- Affichage de base des barres de progression
- Support multilingue (chinois/anglais)
