# Prise en main

Ce chapitre vous aide à installer rapidement le plugin opencode-notify, à découvrir les fonctionnalités de notifications de bureau, et à comprendre son mécanisme de fonctionnement principal.

## Parcours d'apprentissage

### 1. [Démarrage rapide](./quick-start/)
Installez et découvrez votre première notification en 5 minutes.
- Installation en un clic via OCX
- Vérification que les notifications fonctionnent correctement

### 2. [Fonctionnement](./how-it-works/)
Comprenez les fonctionnalités principales du plugin, le mécanisme de filtrage intelligent et les conditions de déclenchement des notifications.
- Maîtrisez les quatre types de notifications et leurs moments de déclenchement
- Comprenez comment le filtrage intelligent évite le bombardement de notifications

## Prérequis

- OpenCode Agent installé ([télécharger sur le site officiel](https://opencode.com))
- Connaissance de base des opérations en terminal
