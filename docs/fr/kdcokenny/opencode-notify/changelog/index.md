# Journal des modifications

Cette section enregistre l'historique des versions et les changements importants du plugin opencode-notify, vous aidant à suivre les mises à jour des fonctionnalités, les corrections de bugs et les améliorations de configuration.

## Sous-pages

### [Journal des modifications](./release-notes/)

Consultez l'historique des changements du plugin en ordre chronologique inverse.

**Contenu inclus** :
- Enregistrements des mises à jour des fonctionnalités
- Explications des corrections de bugs
- Détails des améliorations de configuration
- Avertissements concernant les changements cassants

::: tip Note de version
Ce plugin est publié via OCX et adopte une stratégie sans numéro de version, suivant les changements à travers l'historique des commits Git.
:::

## Prérequis

Aucun prérequis n'est nécessaire, tout le monde peut consulter le journal des modifications.
