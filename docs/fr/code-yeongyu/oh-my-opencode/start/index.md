# Démarrage Rapide

Cette section vous aide à installer et à commencer à utiliser oh-my-opencode rapidement, de la configuration de base à l'orchestrateur principal, jusqu'au mode Ultrawork qui active toutes les fonctionnalités en un clic.

## Parcours d'Apprentissage

Suivez cette séquence pour maîtriser progressivement les fonctions de base d'oh-my-opencode :

### 1. [Installation et Configuration Rapides](./installation/)

Terminez l'installation d'oh-my-opencode, la configuration du Provider et la vérification de base.
- Installer oh-my-opencode
- Configurer le AI Provider (Claude, OpenAI, Gemini, etc.)
- Vérifier que la configuration est correcte
- Comprendre la structure du fichier de configuration

### 2. [Découverte de Sisyphus: L'Orchestrateur Principal](./sisyphus-orchestrator/)

Comprenez les concepts fondamentaux de l'orchestration d'agents et apprenez l'utilisation de base de l'agent Sisyphus.
- Qu'est-ce qu'un système d'orchestration d'agents
- Fonctions principales de l'agent Sisyphus
- Utilisation de base et meilleures pratiques
- Comprendre le flux de travail de la collaboration multi-modèles

### 3. [Mode Ultrawork: Activer Toutes les Fonctionnalités en un Clic](./ultrawork-mode/)

Apprenez à utiliser le mot-clé ultrawork pour activer rapidement tous les agents professionnels et les fonctionnalités de tâches parallèles.
- Qu'est-ce que le mode Ultrawork
- Comment déclencher le mot-clé ultrawork
- Travail collaboratif de 10 agents professionnels
- Gestion des tâches parallèles en arrière-plan

## Prérequis

Avant de commencer cette section, assurez-vous que :
- Node.js 18+ est installé (version 20 ou supérieure recommandée)
- Vous êtes familiarisé avec les opérations de base en ligne de commande
- Vous avez au moins une clé API de service AI (Claude, OpenAI ou Gemini)

::: tip Pas de clé API ?
Si vous n'avez pas encore de clé API de service AI, vous pouvez commencer par demander Claude, qui est actuellement le choix le plus recommandé.
:::
