# Politique de Confidentialité

Dernière mise à jour : 2026-01-01

Merci d'utiliser OpenCodeDocs. Nous accordons une grande importance à votre vie privée et à la protection de vos informations personnelles. Cette politique de confidentialité vous aidera à comprendre comment nous collectons, utilisons, stockons et protégeons vos informations personnelles.

## 1. Informations que nous collectons

Lorsque vous utilisez les services d'OpenCodeDocs, nous pouvons collecter les types d'informations suivants :

### Informations de compte
- **Adresse e-mail** : Utilisée pour l'inscription au compte, la vérification de connexion et les notifications importantes
- **Nom d'utilisateur** : Le nom d'affichage que vous avez choisi
- **Informations du compte GitHub** : Si vous choisissez de vous connecter via GitHub, nous récupérerons vos informations publiques

### Données d'utilisation
- **Historique de navigation** : Les pages de tutoriels que vous visitez et les heures de visite
- **Comportements d'interaction** : Opérations telles que la copie de code et la navigation entre les chapitres
- **Historique de recherche** : Les mots-clés que vous recherchez sur le site

### Informations sur l'appareil
- **Type de navigateur** : Chrome, Firefox, Safari, etc.
- **Système d'exploitation** : Windows, macOS, Linux, etc.
- **Adresse IP** : Utilisée pour la protection de sécurité et l'optimisation régionale

## 2. Utilisation des informations

Les informations que nous collectons sont utilisées aux fins suivantes :

- **Fourniture du service** : Assurer que vous pouvez accéder et utiliser normalement le contenu des tutoriels
- **Expérience personnalisée** : Mémoriser votre progression de lecture et vos préférences
- **Amélioration du produit** : Analyser les modèles d'utilisation pour optimiser le contenu et les fonctionnalités
- **Sécurité** : Détecter les comportements anormaux et prévenir les abus
- **Envoi de notifications** : Envoyer des rappels de mise à jour des tutoriels et des annonces importantes (possibilité de se désabonner)

## 3. Partage et divulgation des informations

Nous ne vendons pas vos informations personnelles. Le partage ne peut avoir lieu que dans les cas suivants :

- **Prestataires de services tiers** : Tels que Vercel (hébergement) et Plausible (analyse), qui ne peuvent accéder qu'aux informations nécessaires à l'accomplissement de leurs fonctions
- **Exigences légales** : Divulgation conformément aux lois, règlements ou exigences gouvernementales
- **Protection des droits** : Situations nécessaires pour protéger les droits d'OpenCodeDocs et des utilisateurs

## 4. Stockage et sécurité des données

- **Emplacement de stockage** : Les données sont stockées sur des serveurs sécurisés situés aux États-Unis et dans la région Asie-Pacifique
- **Mesures de chiffrement** : Les données en transit sont chiffrées avec TLS/SSL, les données sensibles sont stockées avec un chiffrement AES-256
- **Contrôle d'accès** : Accès strictement limité du personnel interne aux données
- **Durée de conservation** : Après la suppression du compte, les données personnelles seront supprimées dans un délai de 30 jours

## 5. Vos droits

Conformément à la Loi sur la protection des informations personnelles et au RGPD, vous bénéficiez des droits suivants :

- **Droit d'accès** : Vous avez le droit de consulter les informations personnelles que nous détenons vous concernant
- **Droit de rectification** : Vous avez le droit d'exiger la correction d'informations personnelles inexactes
- **Droit à l'effacement** : Vous avez le droit d'exiger la suppression de vos informations personnelles
- **Droit à la portabilité** : Vous avez le droit d'exporter vos données dans un format courant
- **Droit de retrait du consentement** : Vous avez le droit de retirer à tout moment le consentement précédemment donné

Si vous souhaitez exercer ces droits, veuillez nous contacter via les coordonnées ci-dessous.

## 6. Politique relative aux cookies

Nous utilisons les types de cookies suivants :

- **Cookies nécessaires** : Maintenir l'état de connexion, mémoriser les préférences linguistiques
- **Cookies d'analyse** : Statistiques sur le volume de visites et les modèles d'utilisation (en utilisant Plausible respectueux de la vie privée)
- **Cookies publicitaires** :
    - Les fournisseurs tiers, y compris Google, utilisent des cookies pour diffuser des publicités en fonction des visites précédentes de l'utilisateur sur votre site ou sur d'autres sites.
    - Google utilise des cookies publicitaires pour permettre à Google et à ses partenaires de diffuser des publicités aux utilisateurs en fonction de leurs visites sur votre site et/ou d'autres sites sur Internet.
    - Les utilisateurs peuvent visiter [Google Ad Settings](https://www.google.com/settings/ads) pour désactiver les publicités personnalisées. Alternativement, vous pouvez orienter les utilisateurs vers [www.aboutads.info](http://www.aboutads.info/choices/) pour désactiver les cookies de fournisseurs tiers utilisés pour les publicités personnalisées.

Vous pouvez gérer les cookies via les paramètres de votre navigateur. La désactivation des cookies nécessaires peut affecter le fonctionnement normal de certaines fonctionnalités.
