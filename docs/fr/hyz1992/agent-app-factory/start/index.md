# Démarrage

Ce chapitre vous aide à démarrer rapidement avec AI App Factory, à comprendre sa valeur ajoutée, à installer et configurer les outils, et à appréhender le fonctionnement du pipeline en 7 étapes pour préparer la génération de votre première application.

## Parcours d'apprentissage

Nous vous recommandons de suivre ce chapitre dans l'ordre suivant :

### 1. [Démarrage rapide : de l'idée à l'application](../getting-started/)
Découvrez la valeur ajoutée d'AI App Factory et les prérequis, puis expérimentez le processus complet de l'idée à l'application à travers un cas concret.
- Comprendre le mécanisme de collaboration multi-agents basé sur les points de contrôle
- Découvrir le workflow complet du pipeline en 7 étapes
- Maîtriser les techniques pour fournir une description produit claire

### 2. [Installation et configuration](../installation/)
Installez l'outil CLI globalement et configurez l'assistant IA (Claude Code/OpenCode) ainsi que l'environnement associé.
- Installer agent-app-factory globalement via npm
- Configurer Claude Code ou OpenCode
- Vérifier que l'installation est réussie

### 3. [Initialisation d'un projet Factory](../init-project/)
Utilisez la commande `factory init` pour initialiser le répertoire, et découvrez la structure du projet généré ainsi que les fichiers de configuration.
- Exécuter la commande d'initialisation du projet
- Comprendre la structure des répertoires et les fichiers de configuration
- Préparer la description de votre idée produit

### 4. [Vue d'ensemble du pipeline en 7 étapes](../pipeline-overview/)
Approfondissez votre compréhension du pipeline complet, du Bootstrap au Preview, avec les entrées/sorties de chaque étape et le mécanisme des points de contrôle.
- Comprendre les objectifs et les livrables de chacune des 7 étapes
- Maîtriser le mécanisme des points de contrôle et le processus de validation manuelle
- Apprendre à contrôler l'exécution du pipeline

## Prérequis

Avant de commencer ce chapitre, vous devez disposer de :

::: warning Prérequis

1. **Environnement Node.js** : Node.js v18 ou version supérieure installé
2. **Assistant de programmation IA** : L'un des outils suivants doit être configuré
   - [Claude Code](https://claude.ai/code) (recommandé) ⭐
   - [OpenCode](https://opencode.design/) ou tout autre assistant IA supportant le mode Agent
3. **Compétences de base en ligne de commande** : Capacité à exécuter des commandes dans un terminal

:::

À l'issue de ce chapitre, vous maîtriserez :

- ✅ Les principes de fonctionnement d'AI App Factory
- ✅ L'installation et la configuration de l'outil CLI
- ✅ L'initialisation de projet et la compréhension de sa structure
- ✅ Le processus complet du pipeline en 7 étapes

Ensuite, nous vous recommandons d'étudier le chapitre **[Plateformes et intégrations](../../platforms/)** pour découvrir comment exécuter le pipeline sur différents assistants IA.
