# Journal des modifications

## Vue d'ensemble du chapitre

Ce chapitre documente les mises à jour de version et les changements de fonctionnalités d'Agent App Factory, vous aidant à comprendre l'évolution du projet et les améliorations importantes.

## Navigation des sous-pages

- **[Notes de version](./release-notes/)** - Consultez l'historique complet des versions, les changements de fonctionnalités et les corrections de bugs

## Recommandations de parcours d'apprentissage

Ce chapitre peut être consulté à tout moment, il est recommandé de :
- Connaître les changements importants de la version actuelle lors de la première utilisation
- Vérifier s'il existe des corrections de bugs liées lorsque vous rencontrez des problèmes
- Suivre régulièrement les mises à jour des fonctionnalités des nouvelles versions
