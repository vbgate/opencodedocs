# 常見問題

本章節收錄了使用 Antigravity Tools 過程中最常遇到的錯誤碼和異常情境，幫助你快速定位問題根因並找到解決方案。

## 本章包含

| 問題類型 | 頁面 | 說明 |
|--- | --- | ---|
| 帳號失效 | [invalid_grant 與帳號自動禁用](./invalid-grant/) | 帳號突然不可用？了解 OAuth Token 失效的原因和恢復流程 |
| 鑑權失敗 | [401/鑑權失敗](./auth-401/) | 請求被拒絕？檢查 auth_mode 設定和 Header 格式 |
| 限流錯誤 | [429/容量錯誤](./429-rotation/) | 頻繁 429？區分配額不足和上游限流，用輪換策略降低影響 |
| 路徑錯誤 | [404/路徑不相容](./404-base-url/) | 介面 404？解決 Base URL 和 /v1 前綴的拼接問題 |
| 串流異常 | [串流中斷/0 Token/簽名失效](./streaming-0token/) | 響應中斷或空？了解代理的自愈機制和排查路徑 |

## 學習路徑建議

**按錯誤碼排查**：遇到具體錯誤時，直接跳轉到對應頁面。

**系統性學習**：如果你想全面了解可能遇到的問題，建議按以下順序閱讀：

1. **[404/路徑不相容](./404-base-url/)** — 最常見的接入問題，通常是第一個坑
2. **[401/鑑權失敗](./auth-401/)** — 路徑對了但請求被拒，檢查鑑權設定
3. **[invalid_grant 與帳號自動禁用](./invalid-grant/)** — 帳號層面的問題
4. **[429/容量錯誤](./429-rotation/)** — 請求成功但被限流
5. **[串流中斷/0 Token/簽名失效](./streaming-0token/)** — 進階問題，涉及串流響應和簽名機制
