# 更新日誌

本章節記錄 Antigravity Tools 的版本演進與發布說明。升級前先看 Changelog，能減少踩到相容性變化或回歸問題的機率。

## 學習路徑

1. **升級前**：先在 Settings → About 確認目前版本，再去 README 的 Changelog 只看你跨過的版本段落
2. **升級後**：跑一遍 `/healthz` 最小驗證，確認代理還能正常運作
