# 平台功能

本章節深入講解 OpenCode Agent Skills 的核心功能模組，包括技能發現、查詢、載入、自動推薦、腳本執行和檔案讀取。掌握這些功能後，你就能充分利用外掛的技能管理能力，讓 AI 更高效地服務於你的開發工作。

## 前置條件

::: warning 開始前請確認
在學習本章節之前，請確保你已完成：

- [安裝 OpenCode Agent Skills](../start/installation/) - 外掛已正確安裝並執行
- [創建你的第一個技能](../start/creating-your-first-skill/) - 了解技能的基本結構
:::

## 學習路徑

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           推薦學習順序                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1. 技能發現機制  ──→  2. 列出可用技能  ──→  3. 載入技能               │
│         │                     │                    │                    │
│         │                     │                    │                    │
│         ▼                     ▼                    ▼                    │
│   理解技能從哪來        學會查找技能          掌握載入方法               │
│                                                                         │
│                              │                                          │
│                              ▼                                          │
│                                                                         │
│   4. 自動技能推薦  ←──  5. 執行腳本  ←──  6. 讀取檔案                   │
│         │                    │                  │                       │
│         ▼                    ▼                  ▼                       │
│   了解智能匹配          運行自動化          存取支援檔案                 │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**建議按順序學習**：

1. **先學發現機制** - 理解技能從哪裡來，優先級如何決定
2. **再學查詢技能** - 掌握 `get_available_skills` 工具的使用
3. **然後學載入技能** - 理解 `use_skill` 和 XML 注入機制
4. **接著學自動推薦** - 了解語義匹配如何工作（可選，偏原理）
5. **最後學腳本和檔案** - 這兩個是進階功能，按需學習

::: tip 快速上手路徑
如果你只想快速用起來，可以只學前三課（發現 → 查詢 → 載入），其他按需補充。
:::
