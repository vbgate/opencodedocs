## 學習路徑

建議按以下順序學習本章內容：

```
常見問題排查 → 安全性說明
```

1. **先學排查**：遇到問題時，先查閱「常見問題排查」快速定位和解決
2. **再學安全**：了解外掛的安全機制，養成安全使用習慣

如果你正在使用外掛時遇到了具體問題，直接跳到「常見問題排查」查詢對應症狀。如果想系統了解外掛的安全設計，再閱讀「安全性說明」。
:::
