# 更新日誌

本章節記錄 OpenSkills 的版本變更歷史，協助您了解每個版本的新功能、改進和問題修復。

## 本章內容

| 頁面 | 說明 |
| --- | --- |
| [更新日誌](./changelog/) | 完整的版本變更記錄，包括新功能、改進、問題修復和安全更新 |

## 如何使用更新日誌

更新日誌按版本號倒序排列（最新版本在最前面），每個版本包含以下分類：

- **新增功能** - 新加入的功能和指令
- **改進** - 對現有功能的最佳化和增強
- **問題修復** - 已修復的 Bug
- **安全性** - 安全相關的改進

::: tip 升級建議
升級 OpenSkills 前，建議先查看更新日誌了解版本變化，特別關注：
- **Breaking Changes**（破壞性變更）
- 新功能的使用方法
- 已知問題的修復情況
:::
