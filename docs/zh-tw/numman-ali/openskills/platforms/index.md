# 平台功能

本章節深入講解 OpenSkills CLI 的核心指令和技能管理功能，幫助你從「會用」到「精通」。

## 前置條件

::: warning 開始前請確認
在學習本章節之前，請確保你已完成 [快速開始](../start/) 章節，特別是：
- [安裝 OpenSkills](../start/installation/)：已成功安裝 OpenSkills CLI
- [安裝第一個技能](../start/first-skill/)：了解基本的技能安裝流程
:::

## 本章內容

本章節包含 6 個主題，涵蓋 OpenSkills 的所有核心功能：

### [指令詳解](./cli-commands/)

完整介紹所有 7 個指令、參數和標誌，提供指令速查表。適合需要快速查找指令用法的用戶。

### [安裝來源詳解](./install-sources/)

詳細講解三種安裝來源：GitHub 儲存庫、本地路徑、私有 Git 儲存庫。每種方式的適用場景、指令格式和注意事項。

### [全域安裝 vs 專案本地安裝](./global-vs-project/)

解釋 `--global` 和預設（專案本地）安裝的區別，幫助你選擇合適的安裝位置，理解技能搜尋優先順序規則。

### [列出已安裝技能](./list-skills/)

教會你使用 `list` 指令查看已安裝技能，理解 `(project)` 和 `(global)` 位置標籤的含義。

### [更新技能](./update-skills/)

指導你使用 `update` 指令重新整理已安裝技能，支援全部更新和指定技能更新，保持技能與來源儲存庫同步。

### [刪除技能](./remove-skills/)

介紹兩種刪除方式：互動式 `manage` 指令和腳本化 `remove` 指令，幫助你高效管理技能庫。
