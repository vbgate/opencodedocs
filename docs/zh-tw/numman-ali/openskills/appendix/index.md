# 附錄

本章節提供 OpenSkills 的技術參考資料，包括指令列 API、檔案格式規範和術語定義。當你需要查閱具體的指令參數、理解產生檔案的結構，或者搞清楚某個術語的含義時，這裡是你的速查手冊。

## 本章內容

| 文件 | 說明 |
| --- | --- |
| [CLI API 參考](./cli-api/) | 所有指令、選項、參數的完整文件，附帶使用範例 |
| [AGENTS.md 格式規範](./agents-md-format/) | sync 指令產生的 AGENTS.md 檔案結構，包括 XML 標籤和技能清單格式 |
| [檔案結構](./file-structure/) | OpenSkills 產生的目錄和檔案說明，幫你理解 `.openskills/` 裡都有什麼 |
| [術語表](./glossary/) | OpenSkills 和 AI 技能系統中的關鍵術語解釋 |

## 學習路徑建議

附錄是參考資料，不需要從頭到尾閱讀。推薦的使用方式：

1. **遇到指令問題** → 查 [CLI API 參考](./cli-api/)
2. **想了解 AGENTS.md 怎麼產生的** → 看 [AGENTS.md 格式規範](./agents-md-format/)
3. **搞不清檔案放在哪** → 翻 [檔案結構](./file-structure/)
4. **看到陌生術語** → 查 [術語表](./glossary/)

::: tip 新手建議
如果你剛開始使用 OpenSkills，建議先完成 [快速開始](../start/quick-start/) 教學，遇到問題再回來查閱附錄。
:::
