# 附錄：技術細節與限制說明

本章節包含外掛的參考文件和技術細節，幫助你深入理解外掛的設計思路、邊界限制和效能優化策略。

::: info 學完你能做什麼
- 了解外掛的已知限制和適用場景
- 掌握快取機制與效能優化策略
- 理解外掛的技術邊界和設計取捨
:::

## 本章節包含

### 📚 [已知限制：外掛的邊界在哪裡](./limitations/)

了解外掛不支援的功能和技術限制，避免在不支援的場景下使用。包括：
- 不支援 HTML 表格、多列儲存格、無分隔列表格
- 不支援合併儲存格和無設定選項
- 超大表格效能未驗證

**適合人群**：想知道外掛能做什麼、不能做什麼的使用者

### 🔧 [技術細節：快取機制與效能優化](./tech-details/)

深入理解外掛的內部實作，包括快取機制、效能優化策略和程式碼結構。包括：
- widthCache 資料結構與快取查詢流程
- 自動清理機制與快取閾值
- 效能優化效果分析

**適合人群**：對外掛實作原理感興趣的開發者

## 學習路徑建議

本章節的兩個子頁面相對獨立，可以按需閱讀：

1. **快速上手使用者**：建議先閱讀「已知限制」，了解外掛邊界後即可停止
2. **深度學習使用者**：按順序閱讀 → 「已知限制」→「技術細節」
3. **開發者**：推薦完整閱讀，有助於理解外掛設計和未來擴充
