## 學習路徑

### 1. [快速開始](./quick-start/)
在 5 分鐘內完成安裝並體驗第一個通知。
- 透過 OCX 一鍵安裝
- 驗證通知功能是否正常運作

### 2. [工作原理](./how-it-works/)
了解外掛的核心功能、智慧過濾機制和通知觸發條件。
- 掌握四種通知類型及其觸發時機
- 理解智慧過濾如何避免通知轟炸
