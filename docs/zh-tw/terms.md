# 用戶服務協議

最後更新日期：2026-01-01
生效日期：2026-01-01

歡迎使用 OpenCodeDocs！在使用我們的服務之前，請仔細閱�本用戶服務協議（以下簡稱「本協議」）。使用我們的服務即表示您同意接受本協議的約束。

## 1. 服務說明

OpenCodeDocs 是一個專注於 AI 工具教學的開放平台，提供以下服務：

- **教學內容**：開源 AI 工具（如 Claude Code、Cursor、LangChain 等）的使用教學
- **程式碼驗證**：基於原始碼倉庫自動驗證教學中的程式碼可執行性
- **社群功能**：使用者可透過 GitHub 參與內容貢獻和討論

我們保留隨時修改、暫停或終止部分或全部服務的權利，恕不另行通知。

## 2. 帳號註冊與管理

### 註冊條件
- 您必須年滿 16 歲（或在您所在地區達到法定年齡）
- 提供真實、準確的註冊資訊
- 每位使用者僅可註冊一個帳號

### 帳號安全
- 您有責任保管好帳號憑證，不得與他人共享
- 如發現帳號被盜用，請立即聯繫我們
- 因帳號管理不善導致的損失，由您自行承擔

### 帳號註銷
您可以隨時申請註銷帳號。註銷後，您的個人資料將在 30 天內刪除，但公開發佈的內容可能仍會保留。

## 3. 使用者行為規範

在使用本平台時，您同意不從事以下行為：

- 發佈違法、有害、威脅、騷擾、誹謗或侵權的內容
- 冒充他人或虛假陳述身份
- 未經授權存取他人帳號或系統
- 干擾或破壞服務的正常執行
- 使用自動化工具大規模擷取內容（合理的 API 使用除外）
- 將教學內容用於商業目的而不遵守授權協議
