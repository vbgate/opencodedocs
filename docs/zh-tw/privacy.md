# 隱私條款

最後更新日期：2026-01-01

感謝您使用 OpenCodeDocs。我們非常重視您的隱私和個人資訊保護。本隱私條款將幫助您了解我們如何收集、使用、儲存和保護您的個人資訊。

## 1. 我們收集的資訊

在您使用 OpenCodeDocs 服務時，我們可能會收集以下類型的資訊：

### 帳號資訊
- **電子郵件**：用於帳號註冊、登入驗證和重要通知
- **使用者名稱**：您選擇的顯示名稱
- **GitHub 帳號資訊**：如您選擇使用 GitHub 登入，我們會取得您的公開資訊

### 使用資料
- **瀏覽記錄**：您造訪的教學頁面和時間
- **互動行為**：程式碼複製、章節跳轉等操作
- **搜尋記錄**：您在站內的搜尋關鍵字

### 裝置資訊
- **瀏覽器類型**：Chrome、Firefox、Safari 等
- **作業系統**：Windows、macOS、Linux 等
- **IP 位址**：用於安全防護和地區優化

## 2. 資訊使用方式

我們收集的資訊將用於以下目的：

- **提供服務**：確保您能夠正常造訪和使用教學內容
- **個人化體驗**：記住您的閱讀進度和偏好設定
- **產品改進**：分析使用模式，優化內容和功能
- **安全保障**：偵測異常行為，防止濫用
- **通知發送**：發送教學更新提醒和重要公告（可選擇退訂）

## 3. 資訊共享與揭露

我們不會出售您的個人資訊。僅在以下情況下可能共享：

- **第三方服務提供商**：如 Vercel（託管）、Plausible（分析），他們僅能存取履行職責所需的資訊
- **法律要求**：依據法律法規或政府要求進行揭露
- **權益保護**：為保護 OpenCodeDocs 及使用者權益的必要情況

## 4. 資料儲存與安全

- **儲存位置**：資料儲存於位於美國和亞太地區的安全伺服器
- **加密措施**：傳輸資料使用 TLS/SSL 加密，敏感資料使用 AES-256 加密儲存
- **存取控制**：嚴格限制內部人員對資料的存取權限
- **保留期限**：帳號註銷後，個人資料將在 30 天內刪除

## 5. 您的權利

根據《個人資訊保護法》及 GDPR，您享有以下權利：

- **存取權**：您有權查看我們持有的關於您的個人資訊
- **更正權**：您有權要求更正不準確的個人資訊
- **刪除權**：您有權要求刪除您的個人資訊
- **匯出權**：您有權以通用格式匯出您的資料
- **撤回同意**：您有權隨時撤回之前給予的同意

如需行使上述權利，請透過下方聯絡方式與我們聯繫。

## 6. Cookie 政策

我們使用以下類型的 Cookie：

- **必要 Cookie**：保持登入狀態、記住語言偏好
- **分析 Cookie**：統計造訪量和使用模式（使用隱私友善的 Plausible）
- **廣告 Cookie**：
    - 第三方供應商（包括 Google）使用 Cookie 來根據使用者在您的網站或其他網站上的過往造訪記錄投放廣告。
    - Google 使用廣告 Cookie，使其及其合作夥伴能夠根據使用者對您網站和/或網際網路上其他網站的造訪記錄向使用者投放廣告。
    - 使用者可以造訪 [Google 廣告設定](https://www.google.com/settings/ads) 停用個人化廣告。或者，您可以引導使用者造訪 [www.aboutads.info](http://www.aboutads.info/choices/) 停用第三方供應商用於個人化廣告的 Cookie。

您可以透過瀏覽器設定管理 Cookie。停用必要 Cookie 可能影響部分功能的正常使用。
