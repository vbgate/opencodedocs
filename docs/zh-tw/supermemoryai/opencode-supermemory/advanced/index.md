# 進階功能

本章節深入講解 Supermemory 的底層機制和進階配置選項。你將學會如何優化記憶引擎的效能，以及如何根據專案需求定製外掛行為。

## 本章內容

<div class="grid-cards">

### [搶佔式壓縮原理](./compaction/)

深入理解 Token 閾值觸發的自動壓縮機制。學習 Supermemory 如何在上下文即將溢出前，主動產生結構化總結並持久化儲存，防止長會話中的「失憶」問題。

### [深度配置詳解](./configuration/)

自訂壓縮閾值、關鍵詞觸發規則和搜尋參數。掌握配置檔案的所有選項，讓 Supermemory 完美適配你的工作流程。

</div>

## 學習路徑

```
┌─────────────────────────────────────────────────────────────┐
│  第 1 步              第 2 步                                │
│  搶佔式壓縮原理   →   深度配置詳解                           │
│  (理解機制)           (動手定製)                             │
└─────────────────────────────────────────────────────────────┘
```

**建議順序**：

1. **先學壓縮原理**：理解 Supermemory 如何自動管理上下文，這是進階配置的基礎。
2. **再學配置詳解**：在理解機制後，你才能做出合理的配置決策（比如閾值設多少合適）。
