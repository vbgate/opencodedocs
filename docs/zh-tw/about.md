# 關於 OpenCodeDocs

> **AI 生態第一教學站 · 原始碼驅動 · 100% 可執行**

## 我們的願景

AI 工具迭代快速，官方文件落後，網上教學過時——我們要改變這一切。

OpenCodeDocs 深入解析開源專案原始碼與官方範例，提煉核心邏輯，並透過自動化驗證確保每一行程式碼都能執行。

當專案更新時，我們的教學同步更新。每篇教學都清楚標註 Git Commit 版本，讓你知道程式碼基於哪個版本產生。

**我們的目標是成為 AI 工具圈的「Verified Standard」。**

## 開源協議

我們相信開放帶來信任。歡迎社群貢獻、糾錯、補充。
