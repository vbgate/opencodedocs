# 更新日誌

本節記錄 DCP 外掛程式的版本演進歷史，幫助你了解每個版本的新功能、改進和修復，以便做出升級決策。

## 本章內容

| 頁面 | 說明 |
|--- | ---|
| [版本歷史](./version-history/) | 查看從 v1.0.1 到最新版本的完整更新記錄，包括新功能、改進和 Bug 修復 |

## 閱讀建議

更新日誌主要用於以下場景：

1. **升級前查閱** - 了解新版本帶來了哪些變化，評估是否需要升級
2. **問題排查** - 確認某個功能是在哪個版本引入的，或某個 Bug 是在哪個版本修復的
3. **了解發展方向** - 通過版本歷史了解 DCP 的功能演進和優化重點

::: tip 保持最新
推薦在 OpenCode 配置中使用 `@latest` 標籤，自動獲取最新版本的功能和修復。
:::
