# 版本歷史

本文檔列出了 opencode-mystatus 的所有版本和主要變更。

## 最新版本

### [v1.2.0 - v1.2.4](./v120-v124/)

- Google Cloud 支援
- 多帳號管理
- 4 個模型額度顯示（G3 Pro、G3 Image、G3 Flash、Claude）
- 修復了多個平台認證問題

## 歷史版本

### [v1.0.0 - v1.0.1](./v100-v101/)

- 初始發布
- OpenAI 額度查詢
- 智證 AI / Z.ai 額度查詢
- GitHub Copilot 額度查詢
- 基礎的進度條顯示
- 多語言支援（中文/英文）
