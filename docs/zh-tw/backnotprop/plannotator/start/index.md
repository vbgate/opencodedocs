# 快速開始

本章節幫助你從零開始使用 Plannotator。你將瞭解 Plannotator 是什麼、它能解決什麼問題，並在你使用的 AI Coding Agent（Claude Code 或 OpenCode）中完成安裝設定。

## 本章內容

<div class="grid-cards">

### [快速開始](./getting-started/)

瞭解 Plannotator 的核心功能和使用情境，在 5 分鐘內完成首次體驗。適合所有新使用者作為入門第一課。

### [安裝 Claude Code 外掛](./installation-claude-code/)

在 Claude Code 中安裝並設定 Plannotator 外掛，支援外掛系統和手動 Hook 兩種方式，涵蓋本機和遠端環境設定。

### [安裝 OpenCode 外掛](./installation-opencode/)

在 OpenCode 中安裝並設定 Plannotator 外掛，啟用 `submit_plan` tool 和 `/plannotator-review` 指令。

</div>

## 學習路徑

::: tip 建議順序
1. **先讀 [快速開始](./getting-started/)**：瞭解 Plannotator 是什麼、能做什麼，建立整體認知
2. **再選一個安裝教學**：根據你使用的 AI Coding Agent 選擇對應的安裝指南
   - 使用 Claude Code → [安裝 Claude Code 外掛](./installation-claude-code/)
   - 使用 OpenCode → [安裝 OpenCode 外掛](./installation-opencode/)
:::

完成安裝後，你就可以開始使用 Plannotator 進行計畫審查和程式碼審查了。

## 下一步

完成本章節後，繼續學習 **[平台功能](../platforms/)**，掌握計畫審查和程式碼審查的完整用法：

- 計畫審查基礎：批准、拒絕、新增註解
- 圖像標註：在計畫中附加截圖並標註
- 程式碼審查：視覺化 Git diff，新增行級註解
