# Schnellstart

Dieses Kapitel hilft Ihnen, das OpenCode Agent Skills Plugin von Grund auf zu nutzen. Sie erfahren mehr über den Kernwert des Plugins, schließen die Installation und Konfiguration ab und erstellen Ihren ersten Skill.

## Inhalt dieses Kapitels

<div class="grid-cards">

<a href="./what-is-opencode-agent-skills/" class="card">
  <h3>Was ist OpenCode Agent Skills?</h3>
  <p>Verstehen Sie den Kernwert und die Funktionsmerkmale des Plugins, einschließlich Mechanismen wie dynamische Skill-Erkennung, Kontext-Injektion, Komprimierung und Wiederherstellung.</p>
</a>

<a href="./installation/" class="card">
  <h3>Installationsanleitung</h3>
  <p>Schließen Sie die Plugin-Installation ab und überprüfen Sie den Betrieb. Unterstützt werden Basis-Installation, Installation mit fester Version und lokale Entwicklungsinstallation.</p>
</a>

<a href="./creating-your-first-skill/" class="card">
  <h3>Erstellen Sie Ihren ersten Skill</h3>
  <p>Beherrschen Sie die Skill-Verzeichnisstruktur und das SKILL.md-Format, und erstellen und testen Sie einen einfachen Skill praktisch.</p>
</a>

</div>

## Lernpfad

Es wird empfohlen, in der folgenden Reihenfolge zu lernen:

1. **[Was ist OpenCode Agent Skills?](./what-is-opencode-agent-skills/)** — Verstehen Sie zuerst, was das Plugin kann, und bilden Sie sich ein Gesamtbild
2. **[Installationsanleitung](./installation/)** — Installieren Sie das Plugin, damit OpenCode über Skill-Management-Fähigkeiten verfügt
3. **[Erstellen Sie Ihren ersten Skill](./creating-your-first-skill/)** — Praktische Übung: Erstellen Sie Ihren ersten eigenen Skill

::: tip Geschätzte Zeit
Das Abschließen dieses Kapitels dauert etwa 30-45 Minuten.
:::

## Nächste Schritte

Nach Abschluss dieses Kapitels können Sie mit **[Plattformfunktionen](../platforms/)** fortfahren, um fortgeschrittene Funktionen wie Skill-Erkennungsmechanismen, Skill-Laden und automatische Zuordnung tiefer zu verstehen.
