# Schnellstart

Dieses Kapitel hilft Ihnen, das opencode-supermemory-Plugin von Grund auf zu installieren und zu konfigurieren, damit Ihr OpenCode Agent über persistente Speicherfähigkeiten verfügt. Nach Abschluss dieses Kapitels wird der Agent in der Lage sein, Ihre Projektarchitektur und Ihre Einstellungen zu merken.

## Voraussetzungen

Bevor Sie mit diesem Kapitel beginnen, stellen Sie sicher:

- ✅ Sie haben [OpenCode](https://opencode.ai) installiert und der `opencode`-Befehl ist im Terminal verfügbar.
- ✅ Sie haben ein [Supermemory](https://console.supermemory.ai)-Konto registriert und den API Key erhalten.
