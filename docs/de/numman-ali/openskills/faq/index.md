# Häufige Fragen

Dieses Kapitel sammelt häufige Fragen, Fehlerbehebungsmaßnahmen und Design-Erklärungen bei der Nutzung von OpenSkills und hilft Ihnen dabei, Probleme schnell zu lokalisieren und zu lösen.

## Inhalte dieses Kapitels

<div class="grid-cards">

### [Häufige Fragen](./faq/)

Beantwortet häufige Fragen zu Installation, Konfiguration und Nutzung. Umfasst den Unterschied zwischen OpenSkills und Claude Code, den Speicherort von Skills, AGENTS.md-Synchronisationsprobleme, Skills-Sharing zwischen mehreren Projekten und mehr.

### [Fehlerbehebung](./troubleshooting/)

Bietet Diagnose- und Reparaturmethoden für häufige Fehler. Deckt detaillierte Lösungsschritte für Git clone-Fehler, fehlende SKILL.md, nicht gefundene Skills, Berechtigungsfehler, übersprungene Updates und andere Probleme ab.

### [Warum CLI?](./why-cli-not-mcp/)

Erklärt die Designentscheidung von OpenSkills für CLI statt MCP. Vergleicht die Positionierungsunterschiede zwischen beiden und hilft Ihnen zu verstehen, warum das Skill-System für statische Dateimodus besser geeignet ist.

</div>

## Lernpfad

```
Probleme?
    │
    ├─ Unklarheiten zur Nutzung ────→ Häufige Fragen
    │                               (Konzepte, Konfiguration, Nutzungstipps)
    │
    ├─ Fehler aufgetreten ──────────→ Fehlerbehebung
    │                               (Fehlerdiagnose, Reparaturschritte)
    │
    └─ Designphilosophie verstehen → Warum CLI?
                                    (Technologieauswahl, Architekturentscheidungen)
```

**Empfohlene Reihenfolge**:

1. **Zuerst Häufige Fragen lesen** → Schnelles Verständnis der Kernkonzepte und Nutzungstipps
2. **Bei Fehlern Fehlerbehebung konsultieren** → Lösungsschritte basierend auf Fehlermeldungen finden
3. **Designphilosophie für tiefere Einblicke lesen** → Technische Überlegungen zu CLI vs. MCP verstehen
