# Anhang

Dieser Abschnitt bietet technische Referenzdokumentation für OpenSkills, einschließlich Befehlszeilen-API, Dateiformatspezifikationen und Begriffsdefinitionen. Wenn Sie spezifische Befehlsparameter nachschlagen, die Struktur generierter Dateien verstehen oder die Bedeutung eines Begriffs klären müssen, ist dies Ihr Schnellreferenzhandbuch.

## Inhalt dieses Kapitels

| Dokument | Beschreibung |
| --- | --- |
| [CLI-API-Referenz](./cli-api/) | Vollständige Dokumentation aller Befehle, Optionen und Parameter mit Verwendungsbeispielen |
| [AGENTS.md-Format](./agents-md-format/) | Die Struktur der von sync-Befehl generierten AGENTS.md-Datei, einschließlich XML-Tags und Skill-Listenformat |
| [Dateistruktur](./file-structure/) | Erläuterung der von OpenSkills generierten Verzeichnisse und Dateien – was befindet sich in `.openskills/`? |
| [Glossar](./glossary/) | Erklärungen der Schlüsselbegriffe in OpenSkills und KI-Skill-Systemen |

## Empfohlener Lernpfad

Der Anhang ist Referenzmaterial und muss nicht von Anfang bis Ende gelesen werden. Empfohlene Verwendung:

1. **Probleme mit Befehlen** → Siehe [CLI-API-Referenz](./cli-api/)
2. **Verstehen der AGENTS.md-Generierung** → Siehe [AGENTS.md-Format](./agents-md-format/)
3. **Unklarheit über Dateispeicherorte** → Siehe [Dateistruktur](./file-structure/)
4. **Unbekannte Begriffe** → Siehe [Glossar](./glossary/)

::: tip Tipp für Einsteiger
Wenn Sie gerade erst mit OpenSkills beginnen, empfehlen wir, zunächst das Tutorial [Schnellstart](../start/quick-start/) zu absolvieren und bei Problemen zum Anhang zurückzukehren.
:::
