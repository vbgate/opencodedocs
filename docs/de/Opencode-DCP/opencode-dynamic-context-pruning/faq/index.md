# Häufig gestellte Fragen

In diesem Kapitel finden Sie Antworten auf häufige Fragen und Best Practices zur Verwendung von DCP, die Ihnen helfen, Probleme schnell zu identifizieren und Ihre Benutzererfahrung zu optimieren.

## Kapitelinhalt

<div class="grid-cards">

### [Häufig gestellte Fragen und Fehlerbehebung](./troubleshooting/)

Lösen Sie häufige Probleme bei der Verwendung von DCP, einschließlich der Behebung von Konfigurationsfehlern, dem Aktivieren von Debugging-Methoden und der Analyse von Gründen für nicht reduzierte Token-Kosten sowie weitere Fehlerbehebungstechniken.

### [Best Practices](./best-practices/)

Lernen Sie die besten Konfigurationsstrategien für DCP kennen, meistern Sie die Balance zwischen Prompt Caching und Token-Einsparungen, den Schutzmechanismen und die automatische Strategieoptimierung sowie weitere fortgeschrittene Techniken.
