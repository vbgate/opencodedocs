# Änderungsprotokoll

Dieses Kapitel dokumentiert den Versionsverlauf und wichtige Änderungen des opencode-notify-Plugins und hilft Ihnen, Funktionsupdates, Problembehebungen und Konfigurationsoptimierungen zu verfolgen.

## Unterseiten

### [Änderungsprotokoll](./release-notes/)

Zeigen Sie die Änderungshistorie des Plugins in umgekehrter chronologischer Reihenfolge an.

**Enthaltener Inhalt**:
- Funktionsupdate-Protokoll
- Problembehebungsbeschreibung
- Konfigurationsoptimierungsdetails
- Warnung vor breaking changes

::: tip Versionshinweis
Dieses Plugin wird über OCX veröffentlicht und verwendet eine versionsfreie Strategie. Änderungen werden über die Git-Commit-Historie verfolgt.
:::

## Voraussetzungen

Keine Voraussetzungen erforderlich. Jeder kann das Änderungsprotokoll einsehen.
