# Benutzervereinbarung

Zuletzt aktualisiert am: 2026-01-01  
Inkrafttreten: 2026-01-01

Willkommen bei OpenCodeDocs! Bevor Sie unsere Dienste nutzen, lesen Sie bitte diese Benutzervereinbarung (im Folgenden "Vereinbarung" genannt) sorgfältig durch. Die Nutzung unserer Dienste bedeutet, dass Sie sich an die Bedingungen dieser Vereinbarung halten.

## 1. Beschreibung der Dienste

OpenCodeDocs ist eine offene Plattform, die sich auf Tutorials für KI-Tools spezialisiert hat und folgende Dienste anbietet:

- **Tutorial-Inhalte**: Anleitungen zur Nutzung von Open-Source-KI-Tools (wie Claude Code, Cursor, LangChain usw.)
- **Code-Verifizierung**: Automatische Verifizierung der Ausführbarkeit von Code in Tutorials basierend auf Quellcode-Repositories
- **Community-Funktionen**: Benutzer können über GitHub an Content-Beiträgen und Diskussionen teilnehmen

Wir behalten uns das Recht vor, Teile oder alle Dienste jederzeit ohne Vorankündigung zu ändern, auszusetzen oder zu beenden.

## 2. Konto-Registrierung und -Verwaltung

### Registrierungsvoraussetzungen
- Sie müssen mindestens 16 Jahre alt sein (oder das gesetzliche Alter in Ihrer Region erreicht haben)
- Sie müssen echte, genaue Registrierungsinformationen angeben
- Jeder Benutzer darf nur ein Konto registrieren

### Kontosicherheit
- Sie sind verantwortlich für die sichere Aufbewahrung Ihrer Kontozugangsdaten und dürfen diese nicht mit anderen teilen
- Wenn Sie feststellen, dass Ihr Konto unbefugt genutzt wurde, kontaktieren Sie uns bitte umgehend
- Verluste, die durch unsachgemäße Kontoverwaltung entstehen, gehen zu Ihren Lasten

### Konto-Löschung
Sie können jederzeit die Löschung Ihres Kontos beantragen. Nach der Löschung werden Ihre persönlichen Daten innerhalb von 30 Tagen gelöscht, aber öffentlich veröffentlichte Inhalte möglicherweise noch erhalten bleiben.

## 3. Verhaltensregeln für Benutzer

Bei der Nutzung dieser Plattform stimmen Sie zu, folgende Handlungen zu unterlassen:

- Veröffentlichung von illegalen, schädlichen, bedrohlichen, belästigenden, verleumderischen oder urheberrechtsverletzenden Inhalten
- Vortäuschen einer anderen Person oder falsche Angaben zu Ihrer Identität machen
- Unbefugter Zugriff auf Konten oder Systeme anderer Personen
- Störung oder Unterbrechung des ordnungsgemäßen Betriebs der Dienste
- Verwenden von Automatisierungstools, um Inhalte großflächig zu extrahieren (ausgenommen angemessene API-Nutzung)
- Verwenden von Tutorial-Inhalten für kommerzielle Zwecke ohne Einhaltung der Lizenzvereinbarung

Wir können diese Vereinbarung jederzeit ändern. Die geänderte Vereinbarung wird auf der Website veröffentlicht und Ihnen wie folgt mitgeteilt:

- Bekanntmachung an einer hervorgehobenen Stelle auf der Website

Die weitere Nutzung unserer Dienste bedeutet, dass Sie die geänderte Vereinbarung akzeptieren. Wenn Sie den Änderungen nicht zustimmen, stellen Sie die Nutzung der Dienste ein.
