# Schnellstart

In diesem Kapitel lernen Sie, Plannotator von Grund auf zu verwenden. Sie erfahren, was Plannotator ist, welche Probleme es lösen kann, und Sie werden die Installation und Konfiguration in Ihrem verwendeten AI Coding Agent (Claude Code oder OpenCode) abschließen.

## Inhalt dieses Kapitels

<div class="grid-cards">

### [Schnellstart](./getting-started/)

Erfahren Sie die Kernfunktionen und Anwendungsfälle von Plannotator und absolvieren Sie Ihr erstes Erlebnis in 5 Minuten. Geeignet für alle neuen Benutzer als erste Einführungsstunde.

### [Claude Code Plugin installieren](./installation-claude-code/)

Installieren und konfigurieren Sie das Plannotator-Plugin in Claude Code, mit Unterstützung für Plugin-System und manuelle Hook-Methoden, einschließlich lokaler und Remote-Umgebungskonfiguration.

### [OpenCode Plugin installieren](./installation-opencode/)

Installieren und konfigurieren Sie das Plannotator-Plugin in OpenCode, aktivieren Sie das `submit_plan` Tool und den `/plannotator-review` Befehl.

</div>

## Lernpfad

::: tip Empfohlene Reihenfolge
1. **Zuerst [Schnellstart](./getting-started/) lesen**: Erfahren Sie, was Plannotator ist, was es kann, und erstellen Sie ein umfassendes Verständnis
2. **Dann ein Installations-Tutorial wählen**: Wählen Sie die entsprechende Installationsanleitung basierend auf Ihrem verwendeten AI Coding Agent
   - Claude Code verwenden → [Claude Code Plugin installieren](./installation-claude-code/)
   - OpenCode verwenden → [OpenCode Plugin installieren](./installation-opencode/)
:::

Nach Abschluss der Installation können Sie beginnen, Plannotator für Planprüfung und Codeprüfung zu verwenden.

## Nächste Schritte

Nach Abschluss dieses Kapitels fahren Sie mit **[Plattformfunktionen](../platforms/)** fort, um die vollständige Verwendung von Planprüfung und Codeprüfung zu beherrschen:

- Grundlagen der Planprüfung: Genehmigen, Ablehnen, Kommentare hinzufügen
- Bildmarkierung: Screenshots an den Plan anhängen und markieren
- Codeprüfung: Git diff visualisieren, zeilenweise Kommentare hinzufügen
