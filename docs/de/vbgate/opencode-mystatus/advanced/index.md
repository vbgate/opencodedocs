# Erweiterte Funktionen

Dieses Kapitel stellt die erweiterten Konfigurationsoptionen von opencode-mystatus vor, geeignet für Benutzer, die mehr Anpassungsmöglichkeiten benötigen.

## Funktionsliste

### [Google Cloud-Konfiguration](./google-setup/)

Konfigurieren und verwalten Sie mehrere Google Cloud Antigravity-Konten:

- Mehrere Google Cloud-Konten hinzufügen
- Zuordnungsbeziehung der 4 Modelle (G3 Pro, G3 Image, G3 Flash, Claude)
- Unterschied zwischen projectId und managedProjectId
- Lösung des Problems unzureichender Modellkredite eines einzelnen Kontos

### [Copilot-Authentifizierungskonfiguration](./copilot-auth/)

Lösen Sie GitHub Copilot-Authentifizierungsprobleme:

- Unterschied zwischen OAuth-Token und Fine-grained PAT
- Lösung des Problems unzureichender OAuth-Token-Berechtigungen
- Erstellen von Fine-grained PAT und Konfiguration des Abonnementtyps
- Konfiguration der Datei `copilot-quota-token.json`

### [Mehrsprachige Unterstützung](./i18n-setup/)

Erfahren Sie über den automatischen Spracherkennungsmechanismus:

- Funktionsweise der automatischen Erkennung der Systemsprache
- Fallback-Mechanismus von Intl-API und Umgebungsvariablen
- Wie Sie die Ausgabesprache wechseln (Chinesisch/Englisch)
