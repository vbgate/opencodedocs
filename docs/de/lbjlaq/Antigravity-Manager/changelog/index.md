# Änderungsprotokoll

Dieses Kapitel dokumentiert die Versionenvielfalt und Release-Hinweise von Antigravity Tools. Bevor Sie ein Upgrade durchführen, sollten Sie das Changelog lesen. Dies reduziert die Wahrscheinlichkeit, auf Kompatibilitätsänderungen oder Regressionen zu stoßen.

## Kapitelinhalt

| Seite | Beschreibung |
| --- | --- |
| [Versionsverlauf](./release-notes/) | Wie Sie das in das README eingebettete Changelog lesen, schnell die Auswirkungen eines Upgrades bewerten und nach dem Upgrade verifizieren |

## Lernpfad

1. **Vor dem Upgrade**: Bestätigen Sie zuerst die aktuelle Version unter Settings → About und lesen Sie dann im Changelog des README nur die Abschnitte der übersprungenen Versionen
2. **Nach dem Upgrade**: Führen Sie eine minimale Validierung mit `/healthz` durch, um sicherzustellen, dass der Proxy weiterhin ordnungsgemäß funktioniert

- Wenn Sie noch nicht installiert haben oder die Installationsmethode kennenlernen möchten, lesen Sie [Installation und Upgrade](../start/installation/)
- Wenn Sie die grundlegende Verwendung des Proxies kennenlernen möchten, lesen Sie [Lokalen Reverse-Proxy starten und ersten Client verbinden](../start/proxy-and-first-client/)
