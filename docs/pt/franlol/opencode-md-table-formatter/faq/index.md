# Perguntas Frequentes

## Visão geral do capítulo

Este capítulo abrange perguntas frequentes e métodos de solução de problemas do plugin opencode-md-table-formatter, ajudando você a localizar e resolver rapidamente vários problemas encontrados durante o processo de formatação de tabelas.

---

## Navegação de subpáginas

### [Perguntas Frequentes: O que fazer quando a tabela não é formatada](troubleshooting/)

Apresenta abrangentemente problemas comuns e soluções no uso do plugin:

- **Tabela não formatada**: Verificar configuração, requisitos de versão, reiniciar OpenCode
- **Estrutura de tabela inválida**: Entender o significado do erro "invalid structure", aprender a sintaxe correta de tabelas
- **Falha na formatação**: Tratar erro "table formatting failed" e situações anormais
- **Tipos de tabelas não suportados**: Entender cenários de limitação como tabelas HTML, células multilinha, etc.
- **Desalinhamento após formatação**: Investigar problemas como modo de ocultação, células muito longas, código em linha, etc.

---

## Sugestão de caminho de aprendizado

Recomenda-se consultar este capítulo ao encontrar problemas práticos após concluir [Visão Geral de Recursos](../start/features/), ou usá-lo como manual de referência para solução de problemas.

Você pode aprender na seguinte ordem:

1. **Ao encontrar problemas**: Primeiro localize o tipo de problema com base nos sintomas (tabela não formatada, mensagens de erro, etc.)
2. **Investigar passo a passo**: Verifique um por um de acordo com a lista de verificação do guia de solução de problemas
3. **Entender profundamente**: Leia [Especificações de Tabelas](../advanced/table-spec/) e [Princípio do Modo de Ocultação](../advanced/concealment-mode/) para entender detalhes técnicos
4. **Limitações conhecidas**: Veja [Limitações Conhecidas](../../appendix/limitations/) para entender os limites do plugin
