# Registro de Alterações

Este capítulo registra o histórico de alterações de versão do OpenSkills, ajudando você a entender os novos recursos, melhorias e correções de cada versão.

## Conteúdo deste Capítulo

| Página | Descrição |
|--- | ---|
| [Registro de Alterações](./changelog/) | Registro completo de alterações de versão, incluindo novos recursos, melhorias, correções de problemas e atualizações de segurança |

## Como Usar o Registro de Alterações

O registro de alterações é organizado em ordem decrescente de número de versão (a versão mais recente fica primeiro), e cada versão contém as seguintes categorias:

- **Novidades** - Recursos e comandos recém-adicionados
- **Melhorias** - Otimizações e aprimoramentos de funcionalidades existentes
- **Correções de Problemas** - Bugs corrigidos
- **Segurança** - Melhorias relacionadas à segurança

::: tip Recomendação de Atualização
Antes de atualizar o OpenSkills, recomendamos que você primeiro consulte o registro de alterações para entender as mudanças de versão, prestando atenção especial a:
- **Breaking Changes** (alterações que quebram a compatibilidade)
- Métodos de uso dos novos recursos
- Situação de correções de problemas conhecidos
:::
