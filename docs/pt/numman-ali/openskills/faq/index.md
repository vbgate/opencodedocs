# Perguntas Frequentes

Este capítulo reúne perguntas frequentes, soluções de solução de problemas e explicações sobre a filosofia de design durante o uso do OpenSkills, ajudando você a localizar e resolver problemas rapidamente.

## Conteúdo deste Capítulo

<div class="grid-cards">

### [Perguntas Frequentes](./faq/)

Responda a dúvidas comuns sobre instalação, configuração e uso. Inclui diferenças entre OpenSkills e Claude Code, local de instalação de habilidades, problemas de sincronização do AGENTS.md, compartilhamento de habilidades entre múltiplos projetos, etc.

### [Solução de Problemas](./troubleshooting/)

Forneça métodos de diagnóstico e correção para erros comuns. Cubra etapas detalhadas de resolução para falhas de Git clone, não encontrar SKILL.md, habilidades não encontradas, erros de permissão, atualizações puladas, etc.

### [Por que CLI?](./why-cli-not-mcp/)

Explique os motivos de design do OpenSkills escolher CLI em vez de MCP. Compare profundamente as diferenças de posicionamento entre os dois, entenda por que o sistema de habilidades é mais adequado para o modo de arquivo estático.

</div>

## Caminho de Aprendizado

```
Encontrou problemas?
    │
    ├─ Não sabe como usar ──────────→ Perguntas Frequentes
    │                                  (conceitos, configuração, dicas de uso)
    │
    ├─ Ocorreu um erro ──────────────→ Solução de Problemas
    │                                  (diagnóstico de erro, etapas de correção)
    │
    └─ Quer entender a filosofia de design ─→ Por que CLI?
                                        (seleção técnica, decisões de arquitetura)
```

**Ordem recomendada**:

1. **Primeiro veja Perguntas Frequentes** → Entenda rapidamente conceitos centrais e dicas de uso
2. **Encontrar erros, veja Solução de Problemas** → Encontre soluções com base nas mensagens de erro
3. **Quer entender profundamente, veja a filosofia de design** → Entenda considerações técnicas CLI vs MCP
