# Apêndice

Este capítulo fornece referências técnicas do OpenSkills, incluindo API de linha de comando, especificações de formato de arquivo e definições de termos. Quando você precisa consultar parâmetros de comando específicos, entender a estrutura dos arquivos gerados, ou esclarecer o significado de um determinado termo, este é o seu manual rápido de consulta.

## Conteúdo do Capítulo

| Documento | Descrição |
|--- | ---|
| [Referência da CLI API](./cli-api/) | Documentação completa de todos os comandos, opções e parâmetros, com exemplos de uso |
| [Especificação de Formato AGENTS.md](./agents-md-format/) | Estrutura do arquivo AGENTS.md gerado pelo comando sync, incluindo tags XML e formato da lista de habilidades |
| [Estrutura de Arquivos](./file-structure/) | Descrição dos diretórios e arquivos gerados pelo OpenSkills, ajudando você a entender o que há dentro de `.openskills/` |
| [Glossário](./glossary/) | Explicação de termos-chave no OpenSkills e no sistema de habilidades de IA |

## Sugestões de Caminho de Aprendizado

O apêndice é um material de referência, não precisa ser lido do começo ao fim. Maneiras recomendadas de uso:

1. **Encontrou problemas com comandos** → Consulte [Referência da CLI API](./cli-api/)
2. **Quer entender como o AGENTS.md é gerado** → Veja [Especificação de Formato AGENTS.md](./agents-md-format/)
3. **Não sabe onde os arquivos estão colocados** → Consulte [Estrutura de Arquivos](./file-structure/)
4. **Encontrou termos desconhecidos** → Consulte [Glossário](./glossary/)

::: tip Sugestão para Iniciantes
Se você está começando a usar o OpenSkills, recomenda-se completar primeiro o tutorial [Início Rápido](../start/quick-start/) e voltar ao apêndice para consultar quando encontrar problemas.
:::
