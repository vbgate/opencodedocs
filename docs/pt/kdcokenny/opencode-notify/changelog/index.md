# Registro de Atualizações

Este capítulo registra o histórico de versões e alterações importantes do plugin opencode-notify, ajudando você a acompanhar atualizações de recursos, correções de problemas e melhorias de configuração.

## Subpáginas

### [Registro de Atualizações](./release-notes/)

Veja o histórico de alterações do plugin em ordem cronológica inversa.

**Conteúdo incluído**:
- Registro de atualizações de recursos
- Descrição de correções de problemas
- Detalhes de melhorias de configuração
- Avisos sobre alterações interruptivas

::: tip Notas sobre versões
Este plugin é publicado via OCX e utiliza uma estratégia sem numeração de versão, rastreando as alterações através do histórico de commits do Git.
:::

## Pré-requisitos

Não há pré-requisitos, qualquer pessoa pode consultar o registro de atualizações.
