# Registro de Atualizações

Este capítulo registra o histórico de evolução de versões do plugin DCP, ajudando você a entender os novos recursos, melhorias e correções de cada versão, para que possa tomar decisões sobre atualizações.

## Conteúdo do Capítulo

| Página | Descrição |
|--- | ---|
| [Histórico de Versões](./version-history/) | Veja o registro completo de atualizações de v1.0.1 até a versão mais recente, incluindo novos recursos, melhorias e correções de bugs |

## Recomendações de Leitura

O registro de atualizações é útil principalmente nos seguintes cenários:

1. **Antes de atualizar** - Entenda quais mudanças a nova versão trouxe e avalie se é necessário atualizar
2. **Solução de problemas** - Confirme em qual versão um recurso foi introduzido ou em qual versão um bug foi corrigido
3. **Compreender a direção de desenvolvimento** - Através do histórico de versões, entenda a evolução funcional e os focos de otimização do DCP

::: tip Mantenha-se Atualizado
Recomendamos usar a tag `@latest` na configuração do OpenCode para obter automaticamente os recursos e correções da versão mais recente.
:::
