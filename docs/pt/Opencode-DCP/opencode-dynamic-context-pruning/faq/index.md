# Perguntas Frequentes

Este capítulo reúne respostas a perguntas frequentes e recomendações de melhores práticas durante o uso do DCP, ajudando você a diagnosticar problemas rapidamente e otimizar sua experiência de uso.

## Conteúdo deste Capítulo

<div class="grid-cards">

### [Perguntas Frequentes e Solução de Problemas](./troubleshooting/)

Resolva problemas comuns ao usar o DCP, incluindo correção de erros de configuração, ativação de métodos de depuração, análise de razões para redução insuficiente de Tokens e outras técnicas de solução de problemas.

### [Melhores Práticas](./best-practices/)

Aprenda as melhores estratégias de configuração do DCP, domine o equilíbrio entre Prompt Caching e economia de Tokens, configuração de mecanismos de proteção, otimização automática de estratégias e outras técnicas avançadas.

</div>

## Caminho de Aprendizagem

```
Perguntas Frequentes e Solução de Problemas → Melhores Práticas
```

1. **Primeiro consulte o guia de solução de problemas**: localize rapidamente soluções ao encontrar problemas
2. **Depois aprenda as melhores práticas**: otimize a configuração para maximizar a economia de Tokens
