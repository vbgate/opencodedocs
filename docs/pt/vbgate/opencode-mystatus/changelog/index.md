# Histórico de Versões

Este documento lista todas as versões e principais alterações do opencode-mystatus.

## Versão Mais Recente

### [v1.2.0 - v1.2.4](./v120-v124/)

- Suporte ao Google Cloud
- Gerenciamento de múltiplas contas
- Exibição de cota de 4 modelos (G3 Pro, G3 Image, G3 Flash, Claude)
- Correção de vários problemas de autenticação de plataformas

## Versões Anteriores

### [v1.0.0 - v1.0.1](./v100-v101/)

- Lançamento inicial
- Verificação de cota do OpenAI
- Verificação de cota do Zhipu AI / Z.ai
- Verificação de cota do GitHub Copilot
- Exibição básica de barra de progresso
- Suporte multi-idioma (Chinês/Inglês)
