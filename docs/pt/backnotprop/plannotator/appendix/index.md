# Apêndice

Esta seção fornece referências técnicas do Plannotator, incluindo documentação da API, definições de modelos de dados e informações de licenciamento. Este conteúdo é destinado principalmente a usuários que precisam compreender profundamente os mecanismos internos do sistema ou desenvolver aplicações secundárias.

## Conteúdo deste Capítulo

| Página | Descrição |
| --- | --- |
| [Referência da API](./api-reference/) | Consulte todos os endpoints de API fornecidos pelo Plannotator e os formatos de requisição/resposta |
| [Modelos de Dados](./data-models/) | Entenda os modelos de dados usados pelo Plannotator, incluindo Annotation, Block, CodeAnnotation, etc. |
| [Licenciamento](./license/) | Saiba mais sobre a licença do Plannotator (BSL-1.1) e restrições de uso |

## Recomendação de Percurso de Aprendizagem

O conteúdo do apêndice é relativamente independente, você pode ler seletivamente de acordo com suas necessidades:

1. **Quer entender interfaces de API** → Leia diretamente [Referência da API](./api-reference/)
2. **Quer entender estruturas de dados** → Leia [Modelos de Dados](./data-models/)
3. **Quer entender restrições de uso** → Leia [Licenciamento](./license/)

::: tip Dica
O conteúdo do apêndice é principalmente para consulta, não precisa ser aprendido em sequência. Recomenda-se consultar os capítulos relevantes quando encontrar problemas específicos.
:::
