# Início Rápido

Este capítulo ajuda você a instalar e configurar o plugin opencode-supermemory do zero, capacitando seu OpenCode Agent com capacidade de memória persistente. Após completar este capítulo, o Agent será capaz de lembrar a arquitetura do projeto e suas preferências.

## Pré-requisitos

Antes de começar este capítulo, certifique-se de:

- ✅ [OpenCode](https://opencode.ai) instalado, com o comando `opencode` disponível no terminal
- ✅ [Supermemory](https://console.supermemory.ai) conta registrada e API Key obtida
