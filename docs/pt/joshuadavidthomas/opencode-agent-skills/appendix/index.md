# Apêndice

Este capítulo fornece materiais de referência e especificações de desenvolvimento para o OpenCode Agent Skills, ajudando você a escrever habilidades de alta qualidade e consultar rapidamente as interfaces API.

## Conteúdo do Capítulo

| Página | Descrição |
|--- | ---|
| [Melhores Práticas de Desenvolvimento de Habilidades](./best-practices/) | Domine as técnicas e especificações para escrever habilidades de alta qualidade, incluindo convenções de nomenclatura, escrita de descrições, organização de diretórios, uso de scripts, etc. |
| [Referência de Ferramentas API](./api-reference/) | Veja os parâmetros detalhados e valores de retorno de todas as ferramentas disponíveis, incluindo `get_available_skills`, `read_skill_file`, `run_skill_script`, `use_skill` |

## Recomendações de Caminho de Aprendizado

::: tip Sequência Recomendada
1. **Primeiro, as Melhores Práticas** → Entenda as especificações e técnicas de desenvolvimento de habilidades, evitando erros comuns
2. **Depois, a Referência API** → Consulte parâmetros e valores de retorno específicos ao encontrar problemas durante o desenvolvimento real
:::

Estes dois conteúdos têm propósitos diferentes:

- **Melhores Práticas**: Adequado para ler completamente antes de começar a escrever habilidades, estabelecendo hábitos corretos de desenvolvimento
- **Referência API**: Adequado como livro de referência, consultando detalhes específicos quando necessário

## Pré-requisitos

::: warning Recomendado Completar Primeiro
Antes de ler este capítulo, recomenda-se que você já tenha:
- Concluído [Criando sua Primeira Habilidade](../start/creating-your-first-skill/), entendendo a estrutura básica de habilidades
- Entendido o [Mecanismo de Descoberta de Habilidades](../platforms/skill-discovery-mechanism/), compreendendo como o plugin encontra habilidades
:::
