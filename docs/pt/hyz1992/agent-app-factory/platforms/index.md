# Plataformas e Integrações

Este capítulo ajuda você a configurar o ambiente de desenvolvimento de IA e instalar os plugins necessários, garantindo que o pipeline possa funcionar normalmente.

## Caminho de Aprendizado

### 1. [Guia de Integração do Claude Code](./claude-code/)
Aprenda como usar o Claude Code para executar o pipeline, incluindo configuração de permissões e melhores práticas.
- Configurar permissões do Claude Code
- Compreender limitações de acesso a arquivos
- Dominar as etapas para executar o pipeline

### 2. [OpenCode e Outros Assistentes de IA](./other-ai-assistants/)
Descubra como usar OpenCode, Cursor e outros assistentes de IA para executar o pipeline.
- Configurar o CLI do OpenCode
- Configurar projetos no Cursor
- Notas de compatibilidade

### 3. [Instalação de Plugins Obrigatórios](./plugins/)
Instale os plugins superpowers e ui-ux-pro-max para garantir que o pipeline funcione normalmente.
- Instalar o plugin superpowers
- Instalar o plugin ui-ux-pro-max
- Verificar disponibilidade dos plugins
