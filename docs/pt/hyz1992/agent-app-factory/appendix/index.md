# Apêndice

Esta seção contém documentação de referência e materiais complementares para ajudá-lo a compreender os detalhes técnicos do AI App Factory.

## Navegação de Subpáginas

### [Referência de Comandos CLI](../cli-commands/)
Lista completa de comandos CLI, descrições de parâmetros e exemplos de uso, incluindo comandos como `init`, `run`, `continue`, `status`, `list` e `reset`.

### [Detalhes da Estrutura de Diretórios](../directory-structure/)
Conheça a estrutura completa de diretórios do projeto Factory e o propósito de cada arquivo, incluindo a função dos diretórios principais como `agents/`, `skills/` e `cli/`.

### [Padrões de Código](../code-standards/)
Conheça as especificações de codificação TypeScript que o código gerado deve seguir, estrutura de arquivos, requisitos de comentários e padrões de mensagens de commit.

### [Detalhes do Stack Tecnológico](../tech-stack/)
Conheça o stack tecnológico usado pelos aplicativos gerados: Node.js + Express + Prisma + React Native, bem como os motivos para a escolha de cada tecnologia.

## Recomendações de Rota de Aprendizado

A documentação desta seção é **material de referência** e não precisa ser lida em ordem. Recomenda-se consultar a seção correspondente conforme necessário ao estudar o tutorial principal:

- Ao estudar [Instalação e Configuração](../../start/installation/), você pode consultar a [Referência de Comandos CLI](../cli-commands/)
- Ao estudar [Inicialização do Projeto Factory](../../start/init-project/), você pode consultar os [Detalhes da Estrutura de Diretórios](../directory-structure/)
- Ao estudar as várias fases do pipeline, você pode consultar os [Detalhes do Stack Tecnológico](../tech-stack/)
- Ao precisar entender os padrões de qualidade do código gerado, você pode consultar os [Padrões de Código](../code-standards/)
