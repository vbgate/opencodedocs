# Primeiros Passos

Este capítulo ajuda você a começar rapidamente com o AI App Factory, entender o valor central, instalar e configurar as ferramentas, e compreender como funciona o pipeline de 7 estágios, preparando você para gerar seu primeiro aplicativo.

## Caminho de Aprendizagem

Recomenda-se seguir a ordem abaixo para estudar o conteúdo deste capítulo:

### 1. [Início Rápido: Da Ideia ao Aplicativo](../getting-started/)

Entenda o valor central do AI App Factory e os pré-requisitos, experimentando o fluxo de ponta a ponta da ideia ao aplicativo através de um caso completo.

- Compreenda o mecanismo de colaboração multi-Agent baseado em checkpoints
- Conheça o fluxo de trabalho completo do pipeline de 7 estágios
- Domine as técnicas de fornecer descrições claras de produtos

### 2. [Instalação e Configuração](../installation/)

Instale a ferramenta CLI globalmente, configure o assistente de IA (Claude Code/OpenCode) e o ambiente relacionado.

- Instale o agent-app-factory globalmente usando npm
- Configure o Claude Code ou OpenCode
- Verifique se a instalação foi bem-sucedida

### 3. [Inicialização do Projeto Factory](../init-project/)

Inicialize o diretório usando o comando `factory init`, entendendo a estrutura do projeto gerado e os arquivos de configuração.

- Execute o comando de inicialização do projeto
- Compreenda a estrutura do diretório do projeto e arquivos de configuração
- Prepare a entrada da ideia do produto

### 4. [Visão Geral do Pipeline de 7 Estágios](../pipeline-overview/)

Entenda profundamente o fluxo completo do pipeline, desde Bootstrap até Preview, as entradas e saídas de cada estágio e o mecanismo de checkpoint.

- Entenda os objetivos e produtos dos 7 estágios
- Domine o mecanismo de checkpoint e o processo de confirmação manual
- Aprenda como controlar a execução do pipeline

## Pré-requisitos

Antes de começar a estudar este capítulo, você precisa:

::: warning Requisitos Prévios

1. **Ambiente Node.js**: Node.js v18 ou superior instalado
2. **Assistente de Programação de IA**: Deve configurar uma das ferramentas abaixo
   - [Claude Code](https://claude.ai/code) (Recomendado) ⭐
   - [OpenCode](https://opencode.design/) ou outro assistente de IA que suporte modo Agent
3. **Operações Básicas de Linha de Comando**: Capacidade de usar o terminal para executar comandos

:::
