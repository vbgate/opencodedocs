# Registro de Atualizações

Este capítulo registra evolução de versão e notas de lançamento do Antigravity Tools. Antes de atualizar, primeiro ver Changelog, pode reduzir probabilidade de pisar em mudanças de compatibilidade ou problemas de regressão.

## Conteúdo do Capítulo

| Página | Descrição |
| --- | --- |
| [Evolução de Versão](./release-notes/) | Como ler Changelog embutido no README, julgar rapidamente impacto de atualização e completar verificação pós-atualização |

## Caminho de Aprendizado

1. **Antes da atualização**: primeiro em Settings → About confirmar versão atual, depois ir ao Changelog do README só ler parágrafos de versões que você cruzou
2. **Após atualização**: rode `/healthz` verificação mínima, confirme se proxy ainda funciona normalmente
